# bible/timeline_events/ — 时间线事件

## 关键时间节点

### 上古纪元
- **创世之初**：上古三神（存在神、虚无神、调和神）创造世界
- **远古**：龙族统治大地、天使族降临、上古种族繁荣
- **约3000年前**：龙族内战、暗影龙被封印、龙族衰落

### 零粒子时代
- **300年前**：零粒子陨石从天而降、魔法文明诞生
- **约200年前**：东和帝国建立、魔道教形成
- **约100年前**：虚空种首次出现、帝国开始培养觉醒者

### 现代
- **约20年前**：地球Phantasm计划启动
- **16年前**：零（Observer #7）被投放至星ノ宮诊所前
- **16年前**：白羽栞（Observer #6）同时降临
- **约13-15年前**：零投稿匿名论文至《帝国魔導学会誌》
- **约6年前**：朱音13岁觉醒零粒子能力
- **约3年前**：真夜12岁用身体封堵封印裂痕
- **现在**：星導学園入学 — 故事开始
