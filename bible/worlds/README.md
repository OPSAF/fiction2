# bible/worlds/ — 世界线变体

## 用途

本目录用于管理不同路线/结局下的世界状态差异。

---

## 世界线变体

| 世界线 | 对应路线 | 关键差异 |
|--------|---------|---------|
| World Line 0 | 主线（白羽） | 全真相揭示、可能返回地球 |
| World Line 1 | 帝国线（朱音） | 帝国改革/革命、政治格局改变 |
| World Line 2 | 宗教线（艾露娜） | 魔道教改革/崩坏、信仰重塑 |
| World Line 3 | 虚空线（真夜） | 虚空种和解/歼灭、龙族复兴 |
| World Line 4 | 古神线（小雪） | 弑神与否、世界起源重定义 |

---

## 世界线分歧点

- **共通线结束**（Ch6末）：第一次分歧——选择攻略对象
- **各线路Ch10**：第二次分歧——Normal/True分支
- **主线True End**：第三次分歧——是否返回地球
