# 星降りのコード・ウィザード — 虚空线：虚空种与卡巴拉

## 一、虚空种（Voidborn）基础定义

- **本质**：被高浓度魔力污染而发生变异的生物——**不是外星人·是这个世界的"本地产物"**
- **成因**：当某个区域的魔力浓度超过临界值时·当地生物会发生不可逆的变异
- 变异后的生物丧失理智·对所有正常生命体具有无差别攻击性
- 外形无固定形态：
  - 弱 = 黑雾状聚合体
  - 中 = 多足/多眼怪物
  - 强 = 可拟态任何物种·保留一定智力
- 涌出频率过去50年急剧上升——与魔矿大量开采·全球魔力浓度上升直接相关
- **人类在制造自己的敌人**——深层主题：对魔力资源的贪婪开发正在创造越来越多的虚空种
- 对抗虚空种是帝国召集精英(小雪被注意·零被招入特待科)和三国联盟的公开理由

## 二、虚空种的另一面（★重要设定）

- **虚空种死亡后的能量可以被回收**：杀死虚空种后·其体内浓缩的魔力可以提取
- 提取的魔力可以补充人类魔法使用者的魔力消耗——但**有侵蚀风险**
- 长期使用虚空种能量的人会逐渐"虚空化"——这是帝国军方的最高机密
- 部分激进军官默许甚至鼓励士兵使用虚空种能量——以换取短期战力提升

## 三、卡巴拉式设计框架（★核心设计）

> **设计参考**：犹太教神秘主义（カバラ/Kabbalah）+ 克苏鲁式宇宙恐怖（轻度）
> **不再使用**：卡巴拉生命之树对应魔法体系（已移至虚空种设计）

### 科学语言解读（隐藏信息）

```
正常世界:
  神粒子（秩序·形式·光明）
    + 零粒子（物质·能量·媒介）
    = 稳定的物理世界

虚空种诞生:
  零粒子异常凝聚（超过临界阈值）
    → 局部区域"过度实化"
    → 物质疯狂增殖但失去有序结构
    → 类似卡巴拉中"Qliphoth（逆性之树）"的概念
    → 产生的不是正常的生命·而是**生命的扭曲镜像**
```

## 五、出现方式

- 主要通过**空间裂缝（Breaches）**出现
- 裂缝位置不固定·但东和帝国境内频率最高（原因不明——可能与古代封印有关？）
- 裂缝大小决定了出现的虚空种数量和等级
- **传统魔法军队难以有效对抗虚空种**——因为：
  - 虚空种的物理性质不稳定（常规武器/魔法经常"穿过"它们）
  - 虚空种可以快速适应攻击模式
  - 只有**觉醒者的原生能力**能对虚空种造成有效伤害
  - **各种族的原生魔法对虚空种的克制效果不同**（见下文）

### 各种族原生魔法克制差异表

| 种族 | 原生魔法特点 | 对虚空种效果 |
|------|------------|-------------|
| **人类** | 多样但不极端 | 中等——依赖道具时效果一般 |
| **精灵** | 精·空间·光系 | **优秀**——空间操控可以切割虚空种结构 |
| **魔族（龙族后代）** | 暗·火·血系 | **最优**——天生克制+血脉加成 |
| **觉醒者（任何种族）** | 个人化独特能力 | **变化极大**——取决于具体能力类型 |

## 六、龙族·魔族·暗影龙的血脉

### 万年前"第一次虚空入侵"种族分工

| 种族 | 角色 | 后代 |
|------|------|------|
| **天使族**（天使神子民） | 远程支援·光属性净化 | → 精灵 |
| **龙族**（龙神子民） | **主力地面战·正面抗衡** | → **魔族**（含多个分支） |
| **海族**（海神子民） | 海上封锁·切断补给 | → 海洋种族（分散） |

**龙族的牺牲最大**——他们是抵抗虚空种的**主力军**，伤亡最惨重。

### 龙族内部分化图

```
万年前龙族:
├─ 正统龙族（主力战斗部队）
│   ├─ 光龙分支 → 后代不详（可能灭绝或进化为未知形态）
│   └─ 炎龙分支 → 现代魔族主体
│
├─ 特种作战部队:
│   ├─ 暗影龙（游击·情报·暗杀）→ **真夜的祖先**
│   └─ 冰龙/风龙等 → 其他魔族分支
│
└─ 后方支援:
    └─ 治愈龙/守护龙等 → 现代魔族中的温和派
```

### 暗影龙的悲剧（万年演化史）

- 暗影龙在战争中承担了**最危险的任务**——深入敌方腹地·破坏虚空种核心
- 战争结束后·龙族内部开始**勾心斗角**
- 暗影龙因长期在"阴暗面"作战·被其他龙族分支**污名化为"被污染者"**
- 遭受系统性迫害·排挤·最终被迫逃离
- 经过万年演化·成为现代的**暗杀家族**（真夜的家族）

### 天生克制虚空种的血脉能力（4种能力+科学猜测）

| 能力 | 效果 | 原因（科学猜测） |
|------|------|---------------|
| **虚空感知** | 能感知远处虚空种的存在 | MCCs对异常CaP场的高度敏感性 |
| **虚空抗性** | 对虚空种的精神/腐蚀攻击有较强抵抗力 | 龙族基因中的某种保护序列 |
| **虚空伤害加成** | 对虚空种造成的伤害高于其他种族 | 可能是针对零粒子异常结构的"反制酶" |
| **暗影龙特有** | **潜行/隐蔽能力**在对抗虚空种时特别有效 | 暗影龙原本就是为特种作战而进化的 |

## 七、真夜线的核心冲突

- 真夜作为**暗影龙后代**·她的觉醒能力很可能与暗影/潜行/暗杀相关
- 她**早已觉醒但未被发现的设定**——暗影龙的能力天然隐蔽
- 她的家族（暗杀家族）可能在**私下继续对抗虚空种**（作为雇佣兵/自由佣兵）
- 如果走真夜线：主角将通过暗杀任务认识真夜·发现她的觉醒身份和龙族血脉·卷入虚空种战争的深层秘密
