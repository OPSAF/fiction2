# 星降りのコード・ウィザード — 古神线：上古三神与世界起源

## 一、上古三神（Ancient Trinity）概览

> **设计参考**：克苏鲁神话（轻度·不过度恶心）+ 自身原创
> **关键原则**：三神**未必是传统意义的"善神"**——他们有自己的立场和考量·玩家需要自行判断

### 三神对照表

| | **天使神** (Angel God) | **龙神** (Dragon God) | **海神** (Sea God) |
|--|---------------------|---------------------|-------------------|
| **称号** | 光之主宰·启明者 | 力之支配·镇守者 | 深之渊主·循环者 |
| **对应种族** | 天使族 → 精灵 | 龙族 → 魔族 | 海族 → 海洋种族 |
| **司掌** | 光·知识·秩序 | 力·战争·守护 | 深·循环·记忆 |
| **神粒子使用者** | ✅ 是 | ✅ 是 | ✅ 是 |
| **当前状态** | 失踪/沉睡？ | 失踪/沉睡？ | 失踪/沉睡？ |
| **世人认知** | "传说中的光之天使" | "远古的巨龙守护者" | "深渊的不可视之神" |

## 二、三神的传说版 vs 真相版

| 事件 | 宗教/传说版本 | 可能的真相（剧情探索）|
|------|-------------|---------------------|
| **世界创造** | 三神携手创造世界 | 可能只是"改造"了已有世界·或发现了这个世界 |
| **零粒子创造** | "神赐予的礼物" | 三神预感到自身力量衰减·为后代创造的"替代能源" |
| **虚空种封印** | "三神合力封印了恶魔" | 确有其事·但封印正在减弱·三神已无力维护 |
| **三神离去** | "升入了更高的维度" | 可能是**耗尽了神粒子**·可能是在对抗中受损·可能是**死了** |
| **陨石撞击** | "神的惩罚/考验" | **绝非三神本意**——零粒子提前扩散的意外事故 |

## 三、克苏鲁式元素（轻度使用指南）

### 使用的元素（✅）

| 元素 | 表现 | 使用程度 |
|------|------|---------|
| **不可名状的存在感** | 三神的真实形态超出人类认知极限 | 轻——更多是"敬畏"而非"疯狂" |
| **古老的时间尺度** | 三神的存在以万年为单位·人类历史对他们如一瞬间 | 中——营造"渺小感" |
| **知识的代价** | 了解过多关于三神/世界真相可能导致精神负担 | 轻——表现为噩梦·偏执·而非发疯 |
| **ambiguity（模糊性）** | 三神的"善恶"立场故意不明确 | 重——这是核心设计·留给玩家判断 |
| **cosmic indifference（宇宙冷漠）** | 三神可能不在乎人类的死活·他们在乎的是更大的东西 | 中——推动剧情的关键动力 |

### 明确不使用的元素（❌）

- ❌ 过度的肉体恐怖/身体畸形描写
- ❌ "看到就疯"的设定
- ❌ 无意义的混沌/随机性
- ❌ 人类完全无足轻重的虚无主义

## 四、零粒子传播的真相时间线

```
时间线（推测）:

万年前: 三神带领种族击败虚空种·预感未来危机
        → 自身神粒子浓度开始降低（原因不明）
        → 决定创造"零粒子"作为替代能源
        → 零粒子在三神的控制下缓慢凝聚（预计需要数千年）

计划: 零粒子成熟后逐步释放 → 世界平稳过渡到"魔法文明"
      → 类似西幻世界观·有序发展

现实: ★ 意外发生了 ★
      → 大约数百年前·一颗富含高浓度凝聚态零粒子的陨石撞击了这个世界
      → 零粒子被**突然·无控制地**传播到全球
      → "迟来"的魔法导致世界走上了不同的演化路径
      → 不是有序的西幻魔法世界·而是"科技+魔法+宗教"混杂的独特形态

三神的反应: ???
      - 他们是否知道这次意外？
      - 他们是否还在干预？
      - 还是他们已经彻底消失/无力干预？
      → **这些是古神线要探索的问题**
```

## 五、小雪的特殊性

- 小雪是目前已知唯一能使用**神粒子類**的人（详见02_magic_physics.md §8）
- 这意味着她的身体可能保留了某种**与三神相关的古老遗传特征**
- 她的"后期觉醒"可能不只是普通的R级觉醒——而是**触发了某种更深层的、与古神相关的东西**
- **古神线 = 小雪的个人故事 + 世界起源真相 + 三神的真正面目**

## 六、"弑神"结局（古神线专属·道德判断型）

> 仅存在于古神线（小雪线）的特殊结局。
> 参考: 博德之门3の道德判断系统——没有标准答案·由玩家自行评判。

### 前提

走到古神线深层剧情·发现三神仍以某种形式存在·且他们的存在对人类构成某种威胁（或至少是限制）。

### 三个选择

| 选择 | 结果 | 道德定位 |
|------|------|---------|
| **弑神** | 击败/杀死/封印三神残余 | 人类获得完全自由·但也失去了保护·未来 uncertain |
| **不弑神** | 与三神达成某种妥协/共存 | 人类继续在三神的阴影下生存·但有庇护 |
| **第三条路** | 成为新的"神"的继承者 | 极端路线·代价巨大 |

**这个结局不做价值判断**——每个选择都有其合理性和代价。如同博德之门一样·让玩家自己决定什么是对的。

## 七、上古种族现状表

| 种族 | 特征 | 现存数 | 三国官方态度 |
|------|------|--------|-------------|
| 天使族 | 光翼·银发·极高魔力·寿命1000+年 | <50人 | 崇拜但不认现存个体为真 |
| 龙族 | 龙形/人形双形态·超强魔力·寿命2000+年 | <30人 | 崇拜但不认现存个体为真 |
| 海族 | 水生·鳞片·深海文明·寿命800+年 | <100人 | 崇拜但不认现存个体为真 |

**"进化论"**：天使→精灵 / 龙族→魔族 / 海族·人類。现存个体被视为"退化"或"混血"。

### 神话时代背景（星暦前）

- **"星之降临"**：天使族、龙族、海族从"星之彼方"来到这个世界。这是所有种族共同的神话记忆。
- 上古种族在大地上建立了第一个文明——"星之文明"。现存古代遗迹（北方山脉·星導の間·各地魔導基層論石板）均来自这个时代。
- 上古种族拥有直接操控"世界代码"的能力——即后来的"観測者"。
- **星之文明的终结**：原因不明。上古种族突然从大地上消失或大幅减少。天使/龙/海族残余个体散布世界，逐渐被后来兴起的种族视为"传说"。
