# bible/timeline/ — 时间线（结构定义）

## 用途

本目录定义时间线的结构和格式规范。具体的时间线事件条目存储在 `../timeline_events/` 中。

---

## 时间线层次

### 宏观时间线（世界史）
- 上古时代：三神创世、龙族崛起、天使降临
- 300年前：零粒子陨石降临、魔法文明诞生
- 近代：帝国建立、魔道教兴起、虚空种出现
- 现代：Phantasm计划、Observer系统

### 中观时间线（角色史）
- 16年前：零被投放至星ノ宮、白羽同时降临
- 13-15岁：零投稿匿名论文、朱音觉醒
- 现在：星導学園入学、故事开始

### 微观时间线（故事内）
- Common Route Ch1-6
- 各路线分歧点及后续
