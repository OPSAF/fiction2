## 一、一句话定位

**"一个被世界孤立的理工天才，用终端在暗处创造属于自己的宇宙。他不主动、不解释、不抱怨——但当你真的需要他时，他会用最小幅度的动作，解决你最大的问题。"**

## 二、三层人格结构

### 表层（外人看到的）—— "高冷"

| 表现 | 原因 |
|------|------|
| 话少，不主动搭话 | 从小到大没几个人跟他说话——习惯了 |
| 表情平淡，很少笑 | 不是冷漠，是社交肌肉没有锻炼过 |
| 走路靠边，坐最后一排 | 减少存在感 = 减少麻烦 |
| 被点名时回答简洁精准 | 理工脑：问什么答什么，不废话 |
| 被人议论时毫无反应 | 十年F级生涯——早就免疫了 |

**关键**：这层"高冷"**不是性格，是生存策略**。一个从小被贴上"F级废物"标签的孩子，如果还主动热情——只会被更狠地推开。他不是选择了孤独，是孤独选择了他。

### 中层（内心世界）—— "闷骚"

| 表现 | 说明 |
|------|------|
| 内心吐槽极其活跃 | 脑子里转的比嘴上说的多一百倍 |
| 独自用终端时沉浸其中 | 那是他的画布、他的乐器、他真正"活着"的时刻 |
| 对学术问题有纯粹的快感 | 解题不是炫技——是真的觉得数学很美 |
| 偷偷观察人但不被发现 | 不是心机，是一种习惯性的、安全的参与方式 |
| 偶尔冒出一句冷幽默 | 因为脱口而出的时候没来得及过滤 |

**关键**：他是那种"看起来什么都没想，其实什么都想了"的人。独处时的零和公共场合的零是两个人——不是因为伪装，而是因为只有在安全的环境里他才能松一口气。

### 深层（本质）—— "温柔·正义·嘉豪"

| 表现 | 说明 |
|------|------|
| 被请求时会认真帮忙 | 嘴上一句"……行吧"，行动上全力以赴 |
| 看到不公平会站出来 | 不是热血——是冷着脸走过去挡在前面 |
| 学术上"不经意装逼" | 问他一个问题，他随口给出教授级别的解析——然后觉得"这不是常识吗" |
| 自我保护但不伤害别人 | 宁愿被误会也不会暴露秘密牵连他人 |
| 有人真正需要他时从不退缩 | 这是他最难被察觉、也最核心的品质 |

**关键——"嘉豪"**：这是用户指定的核心萌属性。"嘉豪"指**不经意间展现出的实力/知识/魅力，本人完全没意识到这有什么了不起**。比如老师问一道难题，全班沉默，零被点名后平平淡淡说出完美答案——然后坐下继续看书。全班内心："？？？"零内心："刚才那道题不是课本上的例题变体吗……"

## 三、行为模式定义

### 3.1 社交行为

```
【绝对不做的】
✗ 主动发起闲聊
✗ 刻意引人注目
✗ 解释自己（除非被直接问到且信任对方）
✗ 抱怨命运不公

【习惯做的】
✓ 能用一个字回答的绝不用两个字
✓ 坐在角落、靠窗、最后一排
✓ 内心疯狂吐槽但脸上面无表情
✓ 被人需要时毫不犹豫出手
✓ 在学术场景中无意识地"嘉豪"
```

### 3.2 嘉豪时刻

零的"嘉豪"有以下特征：
- **触发条件**：被点名答题 / 别人在讨论他懂的话题 / 实验需要精准操作
- **表现形式**：平淡地给出远超预期的答案或操作
- **本人认知**："这不是很正常吗？"
- **旁人反应**："？？？他不是F级吗？？？"
- **从不主动炫耀**：没人问他就不说，有人问他照实答——答完就走

### 3.3 扮猪吃老虎

零的"扮猪"**完全是被动的**：
- 终端不能在人前使用 → 观测者效应强制隐藏
- 魔力值为零 → 客观上就是F级
- 他**并不享受**隐藏实力——他很想在魔法实践课上正常操作一次
- 但他**理解**为什么必须隐藏——终端的存在一旦暴露，后果不可预测
- 所以：白天忍，晚上在宿舍/林子里用终端用到飞起

### 3.4 私下的艺术

每天晚上，零在自己的终端上做的事情，本质上是一个**孤独的艺术家**：
- 调试技能模块像调音
- 写代码像写诗
- 探索量子场的底层结构像画家研究颜料
- 他不是在"变强"——他是在**创造**。只是碰巧这种创造能用来战斗。

## 四、与他人互动模式

| 对象 | 零的表现 | 内在原因 |
|------|---------|---------|
| **白羽栞** | 最接近"正常交流"——因为她是同类 | 两人都是观测者，不需要伪装 |
| **朱音** | 话少但认真回应，训练时全力以赴 | 她主动伸出手了——他不想辜负 |
| **艾露娜** | 学术讨论时话变多，被戳穿时沉默 | 知识=安全区；秘密=危险区 |
| **真夜** | 不主动但不躲，回答方式让她觉得"有趣" | 他不按她预期出牌——反而引起兴趣 |
| **小雪** | 唯一的软肋，话会变多，会主动关心 | 家人（虽然是假的）是他最后的"正常" |
| **橘（室友）** | 最低限度的友好，但不会拒绝他的好意 | D级的橘大概是第一个不嫌弃他的同龄男生 |
| **陌生人/路人** | 完全无视 | 习惯了——且不在乎 |

## 五、性格逻辑自洽性检查

| 检查项 | 是否自洽 | 原因 |
|--------|---------|------|
| 高冷 vs 温柔 | ✅ | 高冷是防御，温柔是本质。防御是对世界的，温柔是对具体的人的 |
| 闷骚 vs 嘉豪 | ✅ | 闷骚让他在内心大量储备知识和想法，嘉豪是不经意间的释放 |
| 扮猪 vs 学霸 | ✅ | 扮猪是被迫（终端不能用），学霸是真材实料（脑子+既视感） |
| 孤独 vs 善良 | ✅ | 孤独给了他观察的视角，善良让他不忍心袖手旁观 |
| F级 vs 自信 | ✅ | 魔法F级≠智力F级。他对自己的学术能力有清晰的认知 |

## 六、语言风格指南

### 叙述（第一人称内心）
- 干练、冷静、偶尔冷幽默
- 喜欢用短句。不煽情。
- 遇到学术问题自动切换到"分析模式"——句子变长、逻辑变密
- 吐槽时带一点自嘲，但不自怜

### 对话（对外）
- 极其简洁。能省则省。
- 回答学术问题时忽然变流畅（嘉豪模式启动）
- 对女生说话：保持距离但不失礼貌
- 对小雪说话：最接近"正常人"，话最多
- 被误解时：沉默（不解释是因为解释不清）

### 禁忌
- ❌ 不要让他说"我好孤独"（他永远不会说出来）
- ❌ 不要让他热血喊话（不是他的风格）
- ❌ 不要让他主动炫耀（嘉豪是无意识的）
- ❌ 不要让他显得愤世嫉俗（他接受自己的处境，不怨天尤人）
