## 设计文档使用指南

每个 DESIGN 文档包含：
1. 角色定位与参考来源
2. 属性标签与基本信息
3. 外貌描写（从头到脚）
4. 性格设计与萌点分析
5. 与其他角色的关系网
6. 剧情弧线（角色成长轨迹）
7. 关键场景设计
