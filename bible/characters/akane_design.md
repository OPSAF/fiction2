## 一、设计总览

### 1.1 用户的核心诉求（原文摘录）

> "朱音和白羽都是有点'傲娇'属性，不过区别是朱音比较热情，正义，阳光，更像'御坂美琴'那种"

> "就是在男主面前的表现比较傲娇，在其他人面前就比较'傲'"

> "就是热心，正义开朗什么的，而且因为卓越的能力也有能够傲的资本"

> "所以在很多人的面前都是'强人'的形象"

> "但是男主从初中的时候，设计类似远坂凛看士郎跳高和当麻美琴的一些剧情"

> "总之就是她私底打不过主角，然后又因为一些原因和主角比较熟，有时相处就会害羞表现出傲娇不坦率"

> "她的短发的形象，而且有点像'假小子'，但是还是很可爱的"

> "就是男女通吃，不管是男生还是女生那边她都很受欢迎，特别是女生之间，描述她像'王子'一般，和其他女生有种'百合'的感觉"

> "反正就类似御坂美琴"

> "不过她本人也不善和人交往，更多时候是自己一个人，没什么好的朋友"

> "刚入学的时候也只有和主角一个说不上关系特别好的朋友，但是确实唯一一个能够比较'正常'说上话的人"

### 1.2 从v8到v9的本质变化

| 维度 | v8（旧版） | v9（新版本） |
|------|------------|-------------|
| **核心类型** | 隐式傲娇·口嫌体正直 | **热血正义型·公强私弱·御坂美琴式** |
| **外在形象** | 未特别强调 | **短发假小子·王子属性·男女通吃** |
| **社交状态** | 后发型青梅竹马 | **不善交往·孤独·唯一能说话的人=零** |
| **"傲"的表现** | 全方位口嫌体正直 | **对零傲娇 / 对他人=强人形象（热心正义阳光）** |
| **私底弱点** | 身份束缚+完美主义 | **私底下打不过零+因熟悉而害羞不坦率** |
| **人气来源** | 未强调 | **女生中像"王子"般受欢迎·百合感** |
| **孤独感** | 被敬畏但没有朋友（较抽象） | **具体化：不善交往·一个人吃饭训练回家·帮了很多人但没变成朋友** |
| **参考重心** | 12个角色均衡参考 | **以御坂美琴为核心锚点**，其他为辅助 |

### 1.3 设计目标

1. **建立「公强私弱」双层人格系统**：公开场合是热血正义的"天宮の王子"，私下面对零时才会暴露害羞不坦率的一面
2. **具象化"王子属性"与"百合感"**：不是标签式的描述，而是通过具体的行为模式和互动场景来呈现
3. **深化"不善交往"的孤独感**：让读者/玩家真切感受到"被所有人喜欢但没有人真正靠近她"的矛盾
4. **设计初中关键事件**：类似"远坂凛看士郎跳高"式的场景——零在某方面超越/打败了朱音，成为她心中永远的"例外"
5. **保留v8优秀机制并适配新设定**：半个人线机制、崩坏告白场景、后发型框架、生理反应设计、"正常人心理"台词标准
6. **避免成为美琴的简单复刻**：提取美琴的核心特质但赋予完全不同的背景和表现形式

## 二、核心参考：御坂美琴式角色分析

### 2.1 御坂美琴的关键特质提取

#### 特质一：热血正义的性格内核

美琴不是一个"冷艳的高岭之花"，而是一个**会主动介入不公正事件的热血少女**：
- 看到 Level Lower 被欺负时会直接出手
- 对不合理的事情无法视而不见
- 行动力极强——想到就会去做，不会犹豫纠结
- 这种正义感不是"说教式的"，而是**身体力行的**

**本质**：美琴的"傲"不只是性格上的强势，更是一种**道德上的自信**——"我知道什么是对的，而且我有能力去执行"。这让她在面对大多数人时有天然的优越感和领导力。

#### 特质二：公众面前的"常盘台王牌"强人形象

在常盘台中学乃至整个学园都市，美琴是公认的"王牌"：
- Level 5 第三位，实力碾压绝大多数人
- 成绩优秀，举止得体
- 在女生中拥有极高人气（尤其是作为"帅气的前辈"）
- 被人敬畏、崇拜、仰慕——但很少有人敢真正靠近她

**关键细节**：
- 她吃早餐时习惯一个人
- 会刻意避开人群聚集的地方
- 很少有真正的朋友（故事初期）
- 即使是室友（白井黑子）和她之间也存在距离感

**本质**："强人形象"既是她的真实能力体现，也是一堵**无形的墙**——越强，越孤独。

#### 特质三：私下面对当麻时的傲娇表现

美琴对上条当麻的态度是她整个角色最核心的魅力来源：
- 从来不叫他的名字——用"那个笨蛋"/"那家伙"
- 明明很在意但故意表现得不在意
- 会因为当麻和其他女生的互动而产生明显的吃醋反应（虽然绝不承认）
- 在当麻遇到危险时会毫不犹豫地冲上去保护他
- **关键点**：她的"傲娇"不是表演性的，而是**真的控制不住自己的情绪**

**为什么只有当麻能看到这一面？**
1. 当麻是少数几个**不怕她**的人——他不因她是Level 5而改变态度
2. 当麻见过她**失败/狼狈的样子**（初遇时的电击误伤等）
3. 当麻对她没有**过度崇拜也没有无视**——这种"正常的态度"在美琴的人生中极其罕见
4. 当麻在某些方面**比她强**（幻想杀手的能力），这让美琴在他面前无法维持"完美强者"的姿态

#### 特质四：孤独感与不善交友

这是美琴角色中最容易被忽略但最深刻的特质：

- 她不是社恐——她在人群中可以表现得非常自然、自信、甚至 charismatic
- 但她**不知道如何深入地结交朋友**
- 她帮助了很多人，但这些人变成了"崇拜者"而不是"朋友"
- 她被人敬畏/喜爱/追随，但没人敢以平等的身份走近她
- 她习惯了一个人吃饭、一个人训练、一个人回家

**这种孤独的本质**：不是因为"没人喜欢她"，而是因为**"喜欢她的人太多了，但没有人喜欢『真实的她』**。大家喜欢的是"Level 5的御坂美琴"、"常盘台的王牌"、"那个帅气的学姐"——而不是那个会有迷茫、会失败、会害羞的普通少女。

#### 特质五：在女生中的人气（"王子般"的存在）

美琴在《某科学的超电磁炮》中的女性人际关系是一个非常重要的侧面：
- 白井黑子对她近乎痴迷（虽然是喜剧化的表现）
- 食蜂操祈与她有复杂的关系（对手/某种意义上的朋友）
- 很多低年级女生把她当作"理想的前辈"/"帅气的姐姐"
- 她的气质中有一种**中性魅力**——既不像传统的柔弱少女，也不像刻板的男性化角色，而是介于两者之间的"帅气"

**"王子感"的来源**：
- 果断的行动力（说做就做，不拖泥带水）
- 保护欲（会主动保护比自己弱的人）
- 不拘小节的性格（不在意细枝末节的形式）
- 偶尔流露的温柔（越是平时强硬的人，偶尔的温柔越动人）

#### 特质六：远坂凛看士郎跳高式的"被打败/被超越"的羞涩

这是一个来自《Fate》系列但与美琴高度共鸣的经典场景：

**场景还原**：远坂凛在第一次看到卫宫士郎跳高时的内心活动——
她一直以为自己是优秀的、全能的、站在高处俯视他人的存在。但当她看到士郎用那种**笨拙但纯粹的坚持**完成了一个她无法做到的动作时，她的世界观产生了一瞬间的动摇。

**为什么这个场景如此重要？**
1. 它打破了"强者的完美神话"——原来我也有做不到的事情
2. 它创造了一种**复杂的情感混合**：不甘心 + 认可 + 某种说不清的吸引力
3. 它成为了两人关系的**秘密锚点**——只有我们两个人知道这件事
4. 它解释了为什么凛会在士郎面前表现出不同于对他人的态度

**对美琴的对应**：当麻的"幻想杀手"能力在某种程度上扮演了同样的角色——美琴引以为傲的力量在当麻面前无效，这种"无力感"反而让她产生了前所未有的在意。

### 2.2 对朱音v9的转化

#### 转化一：热血正义 → 天宮の正义感

| 美琴的表现 | 朱音v9的转化 |
|-----------|-------------|
| 看到Level Lower被欺负直接出手 | 看到新生被欺负/不公正的事情会第一时间介入 |
| 对不合理的事无法视而不见 | 作为风纪委员，对校规内的不公严格处理；对校规外的"灰色地带"用自己的方式解决 |
| 行动力极强 | 说做就做的性格——不会花时间纠结"该不该管"，而是先行动再考虑后果 |
| 身体力行而非说教式 | 她很少讲大道理，而是用行动证明什么是正确的 |

**朱音的独特性**：美琴的正义感更多源于个人气质（天生如此）；朱音的正义感除了天性外，还有一层**天宮分家的教育烙印**——"有能力的人有义务保护没有能力的人"。这让她在做"好事"时有一种**理所当然的责任感**，而非自我感动式的英雄主义。

#### 转化二：强人形象 → 天宮の王子

| 美琴的表现 | 朱音v9的转化 |
|-----------|-------------|
| 常盘台王牌 | 星ノ宮的风纪委员 + 实践科顶尖实力者 |
| Level 5第三位 | B+级矢量学派·热力学分支·火属性，压制型速战风格 |
| 被人敬畏但少有真朋友 | 同样——甚至更甚，因为她还有"天宮家"的身份壁垒 |
| 一个人吃饭/训练 | 一个人吃饭/训练/回家/处理家族事务 |
| 女生中的"帅气前辈" | 女生中的"天宮王子"——详见第三章3.3节 |

**朱音的独特加强**：相比美琴，朱音多了一层**外在形象的加持**——短发、帅气、假小子感。这不是美琴刻意设计的，而是朱音天然的生理特征（短发）+ 性格特征（不拘小节）+ 能力特征（强大）共同塑造的"王子气场"。

#### 转化三：私下傲娇 → 面对零时的崩坏

| 美琴对当麻 | 朱音对零 |
|-----------|---------|
| 叫"那个笨蛋" | 称呼随情境变化（详见WIKI称呼图谱） |
| 在意但不承认 | 同样在意但不承认 |
| 吃醋但不承认 | 同样吃醋（对零与其他女生的互动） |
| 危险时毫不犹豫保护 | 同样——而且会更激烈（因为她觉得自己"应该"保护他） |
| **关键差异** | 朱音的表达更加内敛克制，且切换更加剧烈（从"王子模式"到"害羞少女"的反差更大） |

#### 转化四：孤独感 → 具象化的社交困境

| 美琴的孤独 | 朱音v9的孤独（深化版） |
|-----------|---------------------|
| 少有真朋友 | **刚入学时，零是唯一能"正常说话"的人** |
| 一个人吃饭 | **一个人吃饭，而且习惯了** |
| 帮了很多人但没变朋友 | **帮了很多同学解决问题，但他们只把她当"可靠的风纪委员"** |
| 被敬畏/崇拜 | **被男生暗恋/被女生崇拜/被老师信赖——但没人问她"你今天过得怎么样"** |

#### 转化五：王子属性 → 百合感的具体化

这是v9新增的重点内容，详见第三章3.3节。简言之：
- 美琴的"女性人气"在原作中是副线/隐性内容
- 朱音的"王子属性"和"百合感"是**显性设定**，需要有专门的章节来展开

#### 转化六：被打败/被超越 → 初中关键事件

| 凛看士郎跳高 / 美琴遇当麻 | 朱音v9的初中事件 |
|--------------------------|-----------------|
| 看到对方做到自己做不到的事 | **零在某领域（非武力）超越/打败了朱音** |
| 世界观产生动摇 | **"原来我不是最强的"的认知冲击** |
| 复杂的情感混合 | **不甘心 + 认可 + 说不清的在意** |
| 成为关系的秘密锚点 | **成为朱音对零特殊的根本原因** |
| 解释了特殊态度的来源 | **解释了为什么只有零能看到她的另一面** |

**具体事件设计**：详见第三章3.2节。

### 2.3 与美琴的关键差异（避免成为简单复刻）

| 差异维度 | 御坂美琴 | 天宮朱音v9 |
|---------|---------|-----------|
| **力量体系** | 超能力（Level 5，绝对优势） | 魔法（B+级，强大但非顶级） |
| **身份背景** | 一般家庭（父母在国外） | 天宮分家长女（政治家族+军事背景） |
| **"打不过"的含义** | 当麻的幻想克制她的能力 | 零在某**非战斗领域**超越她（详见3.2节） |
| **短发原因** | 个人喜好/方便行动 | 天然发质+性格使然+从小留短的惯性 |
| **孤独感的根源** | 太强了所以没人敢靠近 | 太强+身份壁垒+不善表达+完美主义=多重叠加 |
| **"傲"的本质** | 个人气质（天生就是这种人） | **后天习得的生存策略**（环境逼迫+教育结果） |
| **恋爱主动性** | 相对较高（会主动找当麻） | **更低**（更被动、更不坦率、更需要外部推动） |
| **友情线** | 有白井黑子/食蜂操祈等明确的朋友角色 | **几乎空白**——这才是她最大的悲剧 |
| **成长方向** | 接受自己的不完美/学会依赖他人 | **学会如何交朋友/接受"不需要完美也能被喜欢"** |

**核心差异总结**：美琴是一个"天生强大但孤独的天才"；朱音是一个"被迫强大且正在学习如何做一个普通人"的少女。美琴的孤独是**选择的结果**（她可以交朋友但她选择了保持距离）；朱音的孤独是**能力的缺失**（她想交朋友但她不知道怎么做）。

## 三、「公强私弱」双层人格系统

### 3.1 公开层：「天宮の王子」—— 热血·正义·阳光·强人

#### 外在形象：短发假小子的帅气美学

**基础外貌**：
- **短发**：长度及耳下颌间，利落的内扣/微翘造型，不扎不绑，自然垂落
- 发色偏深棕或黑色（根据最终设定），阳光下有健康的光泽
- 五官精致但不是传统意义上的"美少女"类型——更偏向**"好看的人"**而非**"漂亮的女孩子"**
- 身材匀称偏运动型（长期训练的结果），不是丰满型也不是瘦弱型
- 身高在同年级女生中偏高（约162-165cm），站姿挺拔

**"假小子"感的来源**：
- 不穿裙子时偏好裤装（校规允许范围内的最大程度利用）
- 动作幅度大、走路快、说话干脆
- 不太在意"女孩子应该怎样"的社会期待
- 表情丰富但偏向"帅气系"（自信的笑/不屑的撇嘴/认真的皱眉），较少出现"可爱系"表情
- **但依然很可爱**——这种反差才是关键。她以为自己很帅，但在别人眼里她"明明很可爱却装作很帅"的样子本身就很可爱

**视觉印象示例**：
> 午休时间的走廊上，她单手抱着几份文件靠在窗边，短发被风吹起一点弧度。几个路过的女生红着脸低头走过去，她完全没有注意到——或者说，她已经习惯了这种视线。
>
> "喂，那边的，校服领结歪了。"她随口指出，语气平淡得像在说今天天气不错。
>
> 被指出的女生慌乱地整理领结，脸涨得更红了。
>
> 朱音已经转身走了。这就是她的日常。

#### 行为模式：热血正义的行动派

**核心行为逻辑**：

```
感知到"不对劲"的事件
    ↓
【0.5秒内判断】这事我能不能管？→ 能
    ↓
【0秒犹豫】直接行动
    ↓
事后才考虑"这样做好不好/合不合适/会不会惹麻烦"
```

**具体表现分类**：

| 类别 | 行为描述 | 示例场景 |
|------|---------|---------|
| **正义干预** | 看到不公平的事立刻介入 | 新生被高年级欺负→直接挡在中间"有什么事冲我来" |
| **热心帮忙** | 别人有困难时主动伸出援手 | 同学的东西掉了→捡起来递过去，顺便说一句"走路看着点" |
| **责任感驱动** | 认为自己的事必须自己做完 | 风纪委员的工作从不假手于人，即使加班也要当天完成 |
| **保护欲** | 对比自己弱的人有天然的保护欲 | 训练场看到有人动作不规范→忍不住上前指导（然后发现指导得太严厉把人吓到了） |
| **好胜心** | 输不起尤其是在意的领域 | 体育祭输了项目→一个人默默加练到天黑 |

**关键设计原则**：她的"热心"和"正义"不是**圣母式的温柔**，而是**带着锋芒的温度**。她会帮你，但嘴上不会太好听；她会保护你，但方式可能让你觉得"这也太强硬了吧"。这正是"御坂美琴式"的精髓——**用强硬的方式表达温柔**。

#### 社交印象：男女通吃的"天宮王子"

**在男生眼中的形象**：
- "好强啊那个天宮……"
- "感觉完全追不上"
- "但她帮人的时候其实还挺——算了不想了"
- 大部分男生处于**"仰慕但不敢靠近"**的状态
- 少数胆大的尝试接近过，被她**天然的无意识距离感**劝退（不是故意冷淡，只是不太懂得如何回应别人的示好）

**在女生眼中的形象（重点）**：

这是朱音v9最独特的设计维度之一。

**"王子"定位的形成过程**：

1. **第一印象冲击**：开学第一天，短发、帅气、站姿笔直、说话干练——很多女生的第一反应不是"好漂亮的女生"而是"好帅的……人？"
2. **行为的持续强化**：每次她帮助别人、维护正义、展现果断行动力的时候，都在加深"王子"印象
3. **反差萌的意外发现**：偶然看到她某个不经意间流露出的可爱瞬间（比如被夸奖时耳根红了）——这种反差会让女生群体沸腾
4. **口碑传播**："你们知道吗天宮同学她其实——""真的假的！？""我亲眼看到的！"

**具体的"王子"行为模式**：

| 行为 | 描述 | 触发的"王子"反应 |
|------|------|-----------------|
| **公主抱式救援** | 体育课上女生摔倒，她单手把人扶起来（不是真的公主抱，但动作够帅） | "啊啊啊啊天宮王子！！！" |
| **壁咚式警告** | 把试图欺负人的男生堵在墙上（用手撑着墙，不是真的壁咚） | 女生旁观的内心："这是什么漫画情节吗" |
| **随手帮忙** | 经过时顺手帮别人拧开紧水瓶/拿高处的东西/搬重物 | "她注意到了我的困难……" |
| **认真指导** | 认真地教别人魔法控制（虽然语气有点凶） | "被天宮同学一对一教导了啊啊啊" |
| **无意间的温柔** | 训练完给别人递水/下雨时把伞借给别人自己淋雨跑回去 | "她明明自己也淋湿了……" |

**百合感的自然呈现**：

"百合感"在这里的定义**不是刻意卖腐**，而是一种**自然的、基于角色魅力的女性向吸引力**：

- 女生会对她产生**超越普通友谊的好感**（想被她注意、想被她帮助、想和她多说几句话）
- 她本人对此**完全不自知**（或者隐约感觉到但不知道怎么应对）
- 她对待女生的态度和对待男生的态度**几乎没有差别**——这也是"男女通吃"的原因之一
- 她不会有意识地利用这种好感，也不会因此感到困扰（大部分时候）

**具体的互动示例**：

> 场景一：更衣室外的偶遇
>
> 几个女生聚在一起讨论今天的训练内容，朱音经过时她们自动安静了一秒。
>

> 场景二：文化祭的准备期间

**为什么这种"百合感"不让人反感？**

因为它**不是刻意的**。朱音没有在"扮演王子"，她只是在做自己。那些让她显得"帅气的瞬间"都是她性格的自然流露，而非为了吸引谁而设计的表演。这种**无意间的魅力**才是最动人的。

#### 为什么有"傲"的资本

朱音的"傲"（在公开场合表现为自信/强势/不容置疑）建立在三个坚实的基础之上：

1. **卓越的实际能力**：B+级矢量学派，压制型速战风格，实战经验丰富。她确实很强，这不是自吹自擂。
2. **正义品格的一致性**：她不是"只说不做"的人。她说到做到，言行一致。这种**人格上的完整性**让人不得不服。
3. **天宮家的教育与资源**：从小接受精英教育，拥有比同龄人更广的视野和更强的执行力。这不是"天赋异禀"而是**后天努力的结果**（虽然起点确实比别人高）。

**关键区分**：她的"傲"不是"看不起别人"，而是**"我有标准，我会按这个标准要求自己和周围的人"**。她不会因为你弱就看不起你，但会因为你不努力/不正直/不负责任而对你失望。这是一种**带有道德优越感的强势**，而非单纯的能力炫耀。

### 3.2 私密层：面对零时的「崩坏」—— 害羞·不坦率·傲娇

#### 为什么只有零能看到这一面

零拥有四个**独一无二的特权条件**，这些条件的组合在整个星ノ宮（甚至在朱音的整个人生中）都是唯一的：

**条件一：同乡身份降低了初始距离感**
- 两人来自同一个地方（故乡/小镇）
- 在星ノ宮这种大城市/名校中，"同乡"是一个天然的破冰话题
- 这让朱音在第一次和零说话时，**不需要启动"天宮家女儿"的社交面具**

**条件二：零不因她"天宮家"身份而改变态度**
- 其他人：听到"天宮"这个名字→态度立刻改变（敬畏/谄媚/疏远）
- 零：听到"天宮"这个名字→"哦，那你家是不是——""嗯。""那挺好。"
- **这种"正常化"的反应对朱音来说是极其罕见的体验**

**条件三：零看到过她失败的样子（初中事件）**
- 这是**最关键的差异化因素**
- 其他所有人认识的朱音都是"完美的天宮家的女儿"
- 只有零认识的是"曾经输给过别人的天宮朱音"
- 这个共享的秘密成为了两人之间**无形的纽带**

**条件四：零不会过度赞美也不会无视她**
- 崇拜者：过度赞美→朱音感到虚假和压力
- 敬畏者：过度疏远→朱音感到孤独
- 敌对者：过度敌视→朱音感到疲惫
- **零：适度的关注+偶尔的吐槽+该夸就夸该损就损**→朱音感到**放松**

**总结**：零是朱音人生中第一个（也是很长一段时间里唯一一个）**同时满足以上四个条件的人**。这就是为什么她会在零面前展现出完全不同的一面——不是因为零有多特别，而是因为**零是唯一一个让朱音感到"安全"到可以卸下防备的人**。

#### 初中时期的"远坂凛看士郎跳高"式事件设计

**事件名称**：「废坑独练·意外的目击者」（暂定名）

**时间**：初中二年级，暑假前夕

**地点**：故乡的一个废弃采石场（朱音的秘密训练场）

**前情提要**：
- 朱音从小就有"必须完美"的压力（来自家族教育+自我要求）
- 她找到了这个废坑作为**独练场所**——在这里她可以试错、失败、而不被任何人看到
- 这是她为数不多的**完全属于自己的空间**

**事件经过**：

> 那是一个闷热的下午。朱音在废坑里练习一个新的火属性阵法——一个她已经在理论上推导完美但始终无法在实战中稳定释放的高级技术。
>
> 第十七次失败。
>
> 阵法的火焰在她面前炸开，灼热的气浪把她的短发吹得乱七八糟。她咬着牙站稳，手掌被自己的火焰烫出了一块红印。

>

> 是零。同镇的那个F级体质的男生。她听说过他——全镇都知道"天宮家的女儿"和"那个怎么都检测不出魔法适配度的怪胎"住在同一个镇上，虽然他们从来没有正式说过话。

**事件的后续影响**：

1. **朱音对零的特殊感情从这里开始萌芽**——不是爱情，而是一种**复杂的、无法归类的在意**
2. **"他在某方面比我强"的认知**——不是魔法能力，而是**洞察力/直觉/某种说不清的"看透事物的眼光"**。这种"非武力的超越"比武力上的胜负更具冲击力
3. **共享秘密**——"他知道我不完美"这个事实成为了两人之间无形的纽带
4. **朱音开始不由自主地关注零**——在学校里会不自觉地寻找他的身影，然后告诉自己"我只是确认他没有把今天的事说出去"
5. **零对此完全不知情**——对他来说这只是"路过看到一个同学在训练顺便说了句自己的想法"而已

**这个事件的设计意图**：
- 类似"远坂凛看士郎跳高"——看到对方在自己不擅长的领域展现出自己没有的特质
- 类似"美琴遇当麻"——自己引以为傲的领域被对方以一种意想不到的方式"破解"
- 为后续所有"朱音对零特殊"的行为提供**根本性的解释**——不是没有原因的傲娇，而是有深刻根源的不坦率
- 创造一个**只有两个人知道的秘密**——这是最牢固的情感粘合剂

#### 傲娇的具体表现形式（自然而非教科书式）

朱音面对零时的"傲娇"**不是她选择的表达方式**，而是**她控制不住的本能反应**。以下是具体的表现形式分类：

**类型A：防御性毒舌（最常见的模式）**

| 内心真实想法 | 嘴上说出来的 | 行动上做出来的 |
|------------|-------------|--------------|
| 我担心你受伤了 | "你这家伙……真是让人没办法。" | 专门绕路去确认他安全 |
| 我想和你一起吃饭 | "……食堂今天有红烧肉。你要来就快点决定。" | 在食堂门口等了他五分钟 |
| 你今天看起来不错 | "……领结。歪了。"（帮他整理好） | 手指碰到他领结的时间比必要的长了两秒 |
| 我很高兴你来帮我 | "算你识相。" | 嘴角压不住的上扬弧度 |
| 我喜欢你（未自觉） | （不会说，但会用更多的"麻烦你"来掩饰"我想和你在一起"） | 所有的空闲时间都用来找借口和他待在一起 |

**类型B：回避型转移（遇到无法用毒舌掩盖的情绪时）**

当情感强度超过了"毒舌"能覆盖的范围时，朱音会使用**话题转移**：

> 零："天宮同学，你今天帮我补习到这么晚，谢谢——"
>
> 朱音：（耳根开始泛红，快速打断）"别说废话。明天的魔法理论考试你复习了吗？第七章的矢量方程组如果再算错我就不管你了。"
>
> 零："呃、那个——"
>
> 朱音："还有，你的笔记字体太潦草了，我自己看都费劲。重新抄一遍。明天早上放到我桌上。"
>
> 零："……你是在惩罚我吗？"
>
> 朱音：（视线看向窗外）"这是为了让你长记性。……走吧，送你到校门口。反正我也要往那个方向。"

**类型C：语言卡壳（极端情况下的防线崩溃）**

当所有的防御机制都失效时，朱音会出现**语言卡壳**——平时能言善辩的她突然找不到词：

> 场景：零认真地夸奖了她某个她确实很在意的地方（不是客套式的夸奖，而是戳中了真实的点）
>
> 零："天宮同学，其实我一直觉得——你帮人的时候，虽然嘴上不好听，但真的很温柔。"
>
> 朱音：（张了张嘴，没有发出声音）
>
> 零："……天宮同学？"
>
> 朱音：（再次张嘴，发出一个模糊的单音节）"……啊。"
>
> （沉默了三秒钟）
>
> 朱音：（语速突然加快，音调升高了大约半个调）"你、你在说什么啊那种话太奇怪了吧我说你今天是不是发烧了我去叫保健室的老师——"
>
> 她转身就走。走了两步又停下来。
>
> （背对着零，声音变小了很多）
>
> "……谢谢。"
>
> 然后快步离开。耳根红得像要滴血。

**关键设计原则**：
- 她的傲娇**永远不会包含以下元素**：教科书式否认（"才不是"）、动漫腔感叹词（"无路赛"/"baka"）、夸张的身体反应（挥手/捂脸/大喊大叫）
- 她的傲娇**永远包含以下元素**：克制的语言（带刺但不过分）、细微的生理反应（耳根红/视线回避/手指绞在一起）、行动与语言的不一致（行动永远比语言诚实）

#### 私底下"打不过"零的含义

这里的"打不过"**绝不是武力层面的**（朱音的战斗力远高于零）。它指的是以下领域中的一个或多个：

**候选领域（按可能性排序）**：

1. **洞察力/直觉**：如初中事件所示，零能在朱音看不到的角度发现问题
2. **情感表达的坦诚度**：零虽然木讪但想说就说，朱音却永远在被"不坦率"折磨
3. **人际交往的轻松度**：零虽然朋友不多但他不在意；朱音在意但做不到
4. **"不在乎别人看法"的能力**：零天生具备；朱音一生都在学习但从未成功
5. **某种特定的游戏/竞技/技能**（待定，可以是象棋、某种体育项目、或者任何非魔法的领域）

## 四、社交困境与孤独感

### 4.1 「不善交往」的具体表现

朱音的"不善交往"**不是社恐**（social anxiety disorder）。她可以在人群中自如地发言、可以在公开场合自信地表现、可以在需要领导力的时候站出来。

她的问题是：**她不知道如何从"认识的人"变成"朋友"**。

#### 具体表现清单

**表现一：帮了很多人但没有变成"朋友"**

> 典型场景：

>
> 朱音这边呢？她也不知道接下来该说什么。在她的认知里，"帮了忙→对方感谢→结束"就是一个完整的社交流程了。她不知道流程后面还有"→闲聊→分享→一起做某事→成为朋友"这一大段。

**表现二：被人敬畏/崇拜但没人敢靠近**

>

>
> 她已经习惯了。但这不代表这不让她难受。

**表现三：不懂得如何深入结交朋友**

朱音的社交技能分布极度不均：

**原因二"正常态度"的稀缺性**：

**原因三共享秘密的亲密感**：

**原因四适度距离的安全感**：
- 零对她的态度是**"不远不近刚刚好"**——不会太近到让她感到侵犯，也不会太远到让她感到冷漠
- 更重要的是，零**不会过度赞美她**（这让她感到虚假和压力），也**不会无视她**（这让她感到被否定）
- 他就在那里，以一种**恰好的距离**存在着——这对朱音来说是前所未有的舒适体验

#### 具体的"王子"行为模式详解

**模式一：保护型王子**

触发条件：看到比自己弱的人（不分性别）遇到困难/受到不公待遇

行为模板：
```
感知到目标需要帮助
    ↓
迅速移动到目标身边（动作干净利落）
    ↓
用身体挡在目标和威胁源之间（如果有物理威胁）
    ↓
用简洁有力的语言处理局面（不说教，不啰嗦）
    ↓
确认目标安全后，用一种"这没什么大不了"的态度收尾
    ↓
（关键）转身离开时不回头看——不是冷漠，而是**不给对方制造"欠人情"的压力**
```

**示例场景**：

> 训练场上，一个一年级女生被高年级学生抢占了练习场地。
>
> 朱音走过来的速度比平时快了15%。她没有跑——跑步会显得"着急"，而"着急"意味着"这件事超出了她的掌控范围"，而没有什么能超出天宮朱音的掌控范围。
>
> "这块场地是先来后到的。"她的声音不大，但足以让周围的人都听到。
>
> 高年级学生转过身，看到是她，表情微妙地变化了一下。
>
> "天宮同学……我们只是——"
>
> "'只是'什么？"朱音的视线扫过那个被挤到一边的一年级女生，"这位同学在这里练习了多久？"
>
> "大概、半个小时……"
>
> "半小时。"朱音点点头，转向高年级，"你们来了多久？"
>
> 沉默。
>
> "三分钟。"旁边有人小声说。
>
> 朱音没有再多说什么。她只是站在那里，双手插在校服口袋里，用一种"我在等你自觉离开"的眼神看着他们。

**模式二：指导型王子**

触发条件：看到有人在努力但方法不对

行为模板：
```
观察到对方的努力和错误
    ↓
（内心评估：这个人值得帮助吗？→ 通常答案是肯定的）
    ↓
靠近，用直接的方式指出问题
    ↓
示范正确的做法（言传身教）
    ↓
让对方尝试，给出即时反馈（通常偏严格）
    ↓
对方做到了→简短肯定（"还行"/"就这样保持"）
    ↓
（关键）不会逗留太久——教完就走，不给对方"被特别关照"的错觉
```

**模式三：无意型王子（最致命的一种）**

触发条件：**无意识状态下做出的自然行为**

这是"百合感"的最主要来源。因为**无意识**，所以**最真实**；因为**最真实**，所以**最动人**。

常见行为列表：

| 行为 | 为什么"王子" | 为什么"致命" |
|------|------------|------------|
| 顺手帮别人拧开紧水瓶 | 力气大+行动自然 | 她甚至没意识到这是个"帮忙"的行为 |

|---------|---------|---------|------------|

| P3 | "天宮家女儿"身份的束缚 | 某个事件迫使她在"家族期望"和"自我意愿"之间做出选择 | 第一次为自己而活 |

**半个人线完成后朱音的状态变化**：

```
完成前：
    "我和零只是同乡/朋友/熟人"
    "我才不喜欢他"
    "我帮他是因为……各种合理的原因"
    "我是一个人也没关系"

完成后：
    "我……喜欢他"（终于承认了自己）
    "但我不能说"（因为说了就会失去现在的距离感）
    "就这样就好"（维持现状是最安全的选择）
    "……但其实不够"（内心深处知道不够但不敢迈出一步）
```

**关键设计**：半个人线的完成**不等于告白**。恰恰相反——半个人线完成后朱音**更难告白**了。因为她现在清楚地知道了自己的感情，而"知道"意味着"害怕失去"。这种**清醒的退缩**比无意识的迟钝更令人心疼。

> 他从口袋里掏出一瓶便利店的饮料（和两年前一模一样的那种），递给了她。
>
> "……还在哭吗。"
>
> 朱音看着那瓶饮料。眼泪又涌了出来。
>
> "……笨蛋。"
>
> 她接过了饮料。
>
> 两个人站在废坑里，看着最后一缕夕阳沉入地平线。
>
> 没有拥抱。没有亲吻。没有"我会永远等你"的承诺。
>
> 只有两个人。一瓶饮料。一个秘密基地。和一个迟到了两年的回答。
>
> ——这就够了。
>
> 因为这是朱音式的告别。
>
> **有尊严的、克制的、含蓄的、带着刺的温柔。**
>
> **正如她这个人一样。**

**v9版崩坏告白的优化要点**：

1. **加入了"王子铠甲"的破碎过程**：她的崩溃不只是因为失恋，更是因为她**维持了多年的"强人形象"在零面前彻底瓦解**了
2. **融入了初中废坑事件的回响**：同一地点、同样的夕阳、同样的"一瓶饮料"——形成了跨越时间的呼应
3. **强调了"不善交往"的悲剧内核**：她哭诉的内容不只是"我喜欢你"，更是**"我终于遇到了一个可以说话的人但我来不及了"**
4. **保留了v8的所有优秀元素**：四幕结构、反常的恭喜、自行崩溃、谎话结尾、尊严退场
5. **新增了"名字"的重量**：零第一次叫她"朱音"成为压垮她最后防线的最后一根稻草

## 七、台词设计原则（v9版）

### 7.1 核心原则不变

**所有台词都必须通过"正常人测试"**（继承自v8）：
> "如果一个正常的十六岁少女在这个情境下说出这句话，我会觉得奇怪吗？"
>
> 如果答案是"会"，那就重写。

### 7.2 双层台词系统

v9的最大变化在于：朱音的台词需要根据**当前处于哪一层人格模式**来设计完全不同的风格。

#### 公开场合（王子模式）的台词风格

**关键词**：干脆、有力、简短、带一点点居高临下的关怀

| 情境 | 台词 | 设计说明 |
|------|------|---------|
| 看到有人被欺负 | "有什么事冲我来。" | 七个字。不需要更多。 |
| 帮助别人后 | "举手之劳。" | 淡化自己的付出，不给对方心理负担 |
| 被人感谢 | "……嗯。" | 最小的回应 = 最大的从容 |
| 表达认可 | "还行。" / "比你想象的强。" | 低评价 = 高认可（了解她的人才知道） |
| 拒绝（温和版） | "不行。" | 直接拒绝不找借口 = 尊重对方的智商 |
| 鼓励（王子版） | "站起来。还没结束。" | 命令式的鼓励 = 最有效的激励 |
| 调侃/吐槽 | "你的领结歪了。" | 用事实代替情感表达 |
| 面对赞美 | "我知道。" | 自信但不自负（或者说，自信且稍微有点自负） |
| 面对挑衅 | "哦？" | 一个字就够了。兴趣+轻蔑的完美混合。 |
| 告别 | "走了。" | 不说再见。因为还会再见。 |

**公开场合台词的设计禁忌**：
- ❌ 不要太长（超过15字的公开场合台词要慎重考虑）
- ❌ 不要带有太多情感色彩（喜怒哀乐都要克制）
- ❌ 不要解释自己的动机（做了就是做了，不用说为什么）
- ❌ 不要使用疑问句（除非是真的在提问，否则用陈述句）

#### 私下场合（崩坏模式/面对零时）的台词风格

**关键词**：防御性、回避型、语言行动不一致、偶发性语言卡壳

| 内心真实想法 | 王子模式下会说 | 崩坏模式下会说 | 行动 |
|------------|---------------|---------------|------|
| 我担心你 | （不会主动提） | "你这家伙……真是让人没办法。" | 专门绕路去确认安全 |
| 我想和你一起吃午饭 | （等你开口） | "……食堂今天有红烧肉。你要来就快点决定。" | 在食堂门口等了五分钟 |
| 你今天看起来不错 | （不评价） | "……领结。歪了。"（动手帮他整理） | 手指碰领结的时间 > 必要时间 |
| 我专门为你做了这个 | （放在你桌上，不留名字） | "谁有空管你啊。只是做多了。" | 做的份量恰好是"多一个人的量" |
| 我很高兴你夸我 | "我知道。"（公开版） | "……每天都是一样的扎法。你看错了。" | 耳根红+卷发梢 |
| 我想和你多待一会儿 | （找各种工作-related的理由） | "哼，随便你了。"（=你出事我不会放过你） | 取消了所有其他安排 |
| 我喜欢你 | （绝对不会说） | （同样不会说，但会用行动表达） | 所有的空闲时间都用来找借口待在一起 |
| 我很难过/失落 | （一个人躲起来） | "我没事。你去忙你的吧。" | 说话时声音在抖 |

**私下场合台词的设计禁忌**：
- ❌ **绝对禁止**教科书式傲娇台词（"才不是因为你才XXX" / "无路赛" / "baka"）
- ❌ 不要出现夸张的否认（否认本身就已经是承认了）
- ❌ 不要在同一句话中出现超过两个借口/理由（多了就显得刻意）
- ✅ **允许/鼓励**：用第三方理由作为借口（红烧肉/顺路/风纪委员职责）
- ✅ **允许/鼓励**：用攻击性语言掩饰关心（"你这家伙" / "真是让人没办法"）
- ✅ **允许/鼓励**：用沉默/省略号/话题转移来替代无法说出口的话

### 7.3 台词三层检验法（继承自v8，v9增强版）

每一句朱音的台词都需要通过以下三层检验：

**第一层：合理性检验**（继承）
- 这句话在这个情境下合理吗？
- 一个十六岁"王子型"女高中生会这样说话吗？
- **v9新增**：这句话符合当前的"人格模式"吗？（王子模式 vs 崩坏模式）

**第二层：身份一致性检验**（继承）
- 这句话符合"天宮分家长女"的身份吗？
- 符合"风纪委员"的职务吗？
- 符合"热血正义"的性格吗？
- **v9新增**：符合"不善交往"的特征吗？（她不应该太擅长社交辞令）

**第三层：双层一致性检验**（v9新增）
- 如果是王子模式：是否足够**干脆/有力/简短**？
- 如果是崩坏模式：是否体现了**语言-行动的不一致**？
- 两层之间的**切换是否自然**？（不是突变，而是渐变）

**v9检验示例**：

> 台词："……食堂今天有红烧肉。你要来就快点决定。"
>
> 第一层 ✓：合理，用第三方理由（红烧肉）作为共同行动的借口
> 第二层 ✓：符合——略带命令的语气（王子模式的残留），但"快点决定"暴露了她在等他回答
> 第三层 ✓（崩坏模式检测）：表面="我去吃红烧肉你来不来随意" / 实际="我想和你一起吃饭但我不想直接说" / 语言行动不一致 ✓ / 切换自然（王子外壳下露出崩坏的内核）✓

## 九、总结：朱音v9的核心公式

### 9.1 一句话定义

> **一个用"热血正义"武装到牙齿的短发假王子，在全人类面前所向披靡，却在唯一的那个F级同乡面前溃不成军。**

### 9.2 核心公式

```
天宮朱音v9 =
    御坂美琴的灵魂（热血正义·孤独·不坦率）
    + 远坂凛的触发事件（被超越/被打败的羞涩起点）
    + 独创的"公强私弱"双层人格（王子vs少女的极致反差）
    + "不善交往"的具象化孤独（帮了所有人但没有朋友）
    + "王子属性"×"百合感"的独特魅力（男女通吃）
    + 短发假小子的视觉识别（帅气但可爱的悖论）
    + 废坑事件的秘密锚点（只有两个人知道的脆弱）
    + v8的全部优秀遗产（半个人线/崩坏告白/生理反应/正常人台词）
    × 零这个"唯一变量"（引爆一切的关键）
```

### 9.4 v9设计的最终自评

**做得好的地方**：
1. ✅ 成功将用户的碎片化诉求整合为一个**有机的角色系统**（而非简单的标签堆叠）
2. ✅ "公强私弱"双层人格提供了足够的**戏剧张力和创作空间**
3. ✅ "不善交往"的具象化让孤独感从抽象概念变成了**可写可感的具体困境**
4. ✅ 初中废坑事件为所有后续行为提供了**统一的根本解释**
5. ✅ "王子属性"和"百合感"通过大量场景示例实现了**自然呈现而非刻意卖腐**
6. ✅ 完整保留了v8的优秀遗产并实现了**无缝融合**

**需要注意的风险**：
1. ⚠️ "王子模式"过于完美可能导致距离感——需要在写作中适时展示她的**不完美瞬间**（如废坑事件的回响）
2. ⚠️ "百合感"的处理需要把握好尺度——过度则变成卖腐，不足则失去特色
3. ⚠️ "不善交往"不要写成"社恐"——她是**能社交但不擅长深度社交**，两者的区别很重要
4. ⚠️ 崩坏告白的时机需要精确把控——过早则情感积累不足，过晚则错过最佳冲击窗口

**一句话总结**：

> **v9不是对v8的否定，而是进化。v8建立了"隐式傲娇"的地基，v9在上面盖起了"热血正义の天宮王子"的大厦——然后亲手在墙上凿出了一个只有零能看到的洞，让阳光照进了那个从不示人的房间。**
