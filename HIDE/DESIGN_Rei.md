## 一、设计总览

### 1.1 用户的核心诉求（原文摘录）

> "综合考虑这七个人的参考设计，SSSS中的南梦芽的可以多一点，但是最主要还是靠Key社的角色"
> "主要是参考性格特点和萌点，然后融合自洽，不能做出'缝合怪'"
> "人物设计要有逻辑要自洽要合理不能强行拼凑，不能什么都参考，你参考你觉得好的地方就行了"

### 1.2 v1.0 → v2.0 的进化方向

| 维度 | v1.0（4人参考） | v2.0（7人参考） |
|------|----------------|----------------|
| **参考池** | Key社4人 | **Key社4人 + SSSS3人（南梦芽重点）** |
| **核心定位** | "Key系观察者型主角" | **"温柔的承载者" —— 不只是观察者，是被所有人依靠的那个"没关系的人"** |
| **新增维度** | 无 | **"被依赖者的孤独" + "安静外表下的正义之核" + "终端作为创造性出口"** |
| **融合原则** | 四角色均等融合 | **Key社=骨架/灵魂 · SSSS=血肉/细节 —— 有主次之分** |

### 1.3 反缝合怪原则（核心设计纪律）

```
【绝对不做的】
✗ 把7个人的特质全部列出来然后各取一点拼凑
✗ 为了"参考到位"而强行加入不协调的元素
✗ 让同一个行为有3个"来源解释"（读者不需要知道参考来源）
✗ 角色性格出现内在矛盾（如同时"极度外向"和"极度内向"）

【必须做到的】
✓ 每一个采纳的特质都能追溯到角色的核心需求（而非"这个角色有所以我也要有"）
✓ 所有特质在零身上是**有机共生**的关系——去掉任何一个都会让角色变薄
✓ 读者读完觉得这是一个**完整的人**，而不是"某几个动漫角色的混合体"
✓ 参考来源仅作为**设计过程的记录**，不影响最终呈现的自然度
```

### 1.4 设计目标

1. **建立"温柔的承载者"形象** —— 比v1.0的"观察者"更主动：他不只看，他**承受**
2. **让F级从"弱点"变成"最动人的部分"** —— 不是要弥补F级，而是**F级本身就是答案**
3. **7个参考角色有机融合** —— Key社提供灵魂骨架，SSSS填充血肉细节
4. **为主角建立不可替代的情感生态位** —— 每个女主需要他的理由各不相同且不可替代
5. **为后续剧本提供完整的行为框架**

### 2.2 【Key社四人组】灵魂骨架

> 以下四人是零这个角色的**本质来源**。没有他们，零就不存在。

#### ① 鳴瀬白羽 —— 「纯真」的源泉

**提取的核心（v1.0已确立·v2.0保留并浓缩）：**

零的世界观底色来自白羽：**在一个糟糕的世界里认真地对每一件事产生兴趣**。
- 终端时间 = 白羽擦拭望远镜 → "做无用但正确的事"
- 对人的注视方式 = 白羽看废墟长 → 不是评估价值，而是确认存在
- **v2.0新增理解**：白羽的纯真不是天真，是一种**经过选择的善良**。零也一样——他选择对这个世界保持善意不是因为不知道它有多糟，而是**知道了之后依然选择善意**。这就是F级尊严的本质。

#### ② 友利奈緒 —— 「承担」的范本

**提取的核心（v1.0已确立·v2.0深化）：**

零的行为模式来自奈緒：**失去了最重要的东西之后，用自己仅剩的方式继续战斗**。
- "我没有魔力。但我还有脑子。" = "我已经没有能力了。但我还有腿。"
- 既视感记忆机制 = 奈绪的相机 → 记住每一个可能消失的瞬间
- 间接表达模式 = 告白的背影 → 行动永远先于语言
- **v2.0新增理解**：奈绪最打动人的不是她的强大，而是她的**"还在继续"**。零也是这样——F级、无魔力、体力差，但他**每天都在继续**。这种无声的坚持比任何爆发都更有力量。

#### ③ 棗鈴 —— 「脆弱」的映射

**提取的核心（v1.0已确立·v2.0保留）：**

零的三层结构来自铃：**看起来最坚强的人其实最害怕失去**。
- 表层F级 = 铃的外刚（不是伪装，是真的状态）
- 中层温柔 = 铃对直枝展露的真实（只在信任的人面前）
- 深层恐惧 = 铃面具下的眼泪（"如果一切都是假的怎么办"）
- **v2.0定位**：铃教会零的是——**坚强不是不脆弱，是带着脆弱继续前进**。零的F级自尊不是"我不在乎"，而是**"我在乎但我不会让它定义我"**。

#### ④ 立華奏 —— 「使命」的高度

**提取的核心（v1.0已确立·v2.0保留）：**

零的成长弧光来自奏：**从为自己活到为他人活**。
- 被动→主动的转变 = 奏的选择（留下承担）
- 行动派关怀 = 奏的无言付出（做不说）
- 耐心品质 = 奏的等待（相信总有一天会被理解）
- **v2.0定位**：奏给零设定了一个**精神天花板**——不是力量上的，而是道德上的。"如果我能让身边的人都好好的"这句话，到了后期会升级为"如果我能让两个世界都好好的"。奏证明了：**最温柔的人往往能做出最勇敢的选择**。

### 2.3 【SSSS三人组】血肉细节

> 以下三人是零这个角色的**血肉填充**。没有他们，零还是那个零，但会少了很多**让人心动的细节**。

#### ⑤ 南梦芽（重点）—— 「被依赖者的孤独」★ SSSS最高优先级

**基础信息**：《SSSS.DYNAZENON》/ 家中承担者 / 变身者
**标签**：温柔、**沉重责任感**、**独自承担**、**不示弱**、**为家人牺牲**、**外柔内刚的反转**

**为什么南梦芽对零的设计至关重要？**

Key社四人提供了零的"灵魂"，但他们都有一个共同盲区：**没有人描绘过"被所有人依赖的那个人"是什么感觉**。

- 白羽是**被保护的对象**（废墟长保护她）
- 奈绪是**独行侠**（她不需要谁依赖她）
- 铃是**被伙伴接纳的人**（Little Busters互相支撑）
- 奏是**主动选择承担的人**（她的承担是使命性的）

但**南梦芽不同**：她是**被动地成为了所有人的依靠**——因为她是"最靠谱的那个人"，所以所有人都自然而然地向她倾斜重量。而她**从来不拒绝**，也**从来不说不**，更**从来不展示自己的疲惫**。

**→ 这恰恰就是零在小雪/朱音/艾露娜/真夜眼中的样子。**

**从南梦芽身上提取的核心特质：**

**A. "被依赖"是一种无法逃脱的角色**

南梦芽的家庭处境让她不得不成为"可靠的大姐"——父母不在/妹妹需要照顾/生活压力全在她肩上。她不是选择了这个角色，是**生活把这个角色塞给了她**。

**→ 转化为零的"天然守护者"位置：**
- 零不是主动选择当"大家的依靠"——是**他的性格自然导致了这个结果**
- 小雪依赖他因为他是哥哥（14年的惯性）
- 朱音依赖他因为他不评判（天宮家没人这样对她）
- 艾露娜依赖他因为他跟得上思维（天才很孤独）
- 真夜依赖他因为他接得住（暗杀者找不到这样的目标）
- **零从来没有说过"我来保护你们"——他只是在每个人需要的时候恰好都在**
- 这就是南梦芽式的"被依赖"：**不是英雄主义，是性格导致的自然结果**

**B. 从不示弱 —— 因为"如果我倒了，谁来接住大家？"**

南梦芽的经典特征：**再累再痛也不会在别人面前表现出来**。只有在极少数时刻（比如变身暴走/情绪崩溃）才会暴露内心的重压。

**→ 转化为零的三层结构中最深层的"锁"：**
- 零的表层和中层已经有很多人看到了
- 但**最深层的恐惧和疲惫——只有终端屏幕的光照见过**
- 他不说"我累了"不是因为逞强，是因为**说出来的那一刻，就意味着有人需要来安慰他**
- 而零**不想成为需要被安慰的人**——因为那样他就不能再安慰别人了
- 这个逻辑和南梦芽完全一致：**守护者的本能是先确保别人的安全，自己的安全排在最后**

**C. 温柔中的坚韧 —— 不是"温柔或者坚韧"，是"因为温柔所以坚韧"**

南梦芽不是那种"平时软绵绵关键时刻爆发"的类型。她的坚韧是**日常化的**——每天早起做饭/送妹妹上学/打工/回家/再做饭/照顾家人。**日复一日的不放弃才是真正的坚韧**。

**→ 转化为零的F级尊严论的终极形态：**
- 不是"虽然我是F级但我内心很强"（太热血漫了）
- 而是**"我就是F级。但我每天的早餐都好好吃。每节课都好好听。每个人的话我都好好听。"**
- 这种**日常化的坚持**比任何爆发式成长都更动人
- 南梦芽教会我们：**最强大的力量不是爆发力，是不熄灭力**

**D. 对家人的特殊柔软**

南梦芽所有的坚硬外面有一块唯一柔软的地方——她的妹妹。那是她**一切坚持的理由**。

**→ 直接映射为零与小雪的关系：**
- 零的所有F级尊严在面对小雪时会自动软化
- 不是溺爱，是**"这个人的存在是我每天坚持下去的最大理由"**
- 这份感情的名字目前叫"家族爱"（DESIGN_Koyuki.md Phase A）
- 但南梦芽告诉我们：**有时候你用全部力量保护的一个人，最后会成为让你重新认识自己的人**

#### ⑥ 宝多六花 —— 「正义之核」的火种

**基础信息**：《SSSS.GRIDMAN》/ 高中生 / 正义感爆棚的少女
**标签**：**正义感**、**热情**、**行动力**、**温柔**、**为朋友拼命**

**从六花身上提取的核心特质（轻度采用·精准定位）：**

**A. 安静外表下藏着一颗炽热的心**

六花平时看起来是个普通甚至有点宅的女孩子，但当涉及到"正确的事"时她会爆发出惊人的行动力和决心。

**→ 转化为零的"正义之核"（极少触发但确实存在）：**
- 零99%的时间是温和的、回避冲突的、F级自觉的
- 但有**1%的时刻**——当他看到明显的不公/弱者被欺凌/无辜者受伤害时
- 他的反应**不是愤怒**（他没有那个能量），而是一种**冷静到可怕的确信**
- "这件事不对。我要做什么来改变它。"
- 这1%的零和99%的零是同一个人——**只是那1%的时候他忘记了自己是F级**
- 六花式的"平时温和·关键时刻燃烧" = 零的**隐藏火焰**

**B. 行动先于思考的热情**

六花不会站在原地纠结"我应该怎么做"——她**直接去做**。

**→ 转化为零的少数"主动时刻"：**
- 零通常是被动型的（等人来找他/等事情发生）
- 但在某些特定的、涉及他在乎的人的时刻
- 他会**突然变得行动力极强**（帮艾露娜查资料到凌晨/为了朱音一个问题翻遍图书馆/第一次主动去找真夜而不是等她来）
- 这些时刻之所以动人**正是因为它们稀少**
- 六花证明了一件事：**平时越安静的人，一旦行动起来越有分量**

#### ⑦ 新条茜 —— 「创造」作为出口

**基础信息**：《SSSS.GRIDMAN》/ 创作者 / 通过艺术表达内心
**标签**：**创造力**、**表达欲**、**内心世界**、**友情**、**选择性分享**

**从茜身上提取的核心特质（轻度采用·概念转化）：**

**A. 用"创造"来表达无法言说的东西**

茜通过画画/创作来表达那些她说不出来的感受。创作是她的**情感出口**和**自我确认**的方式。

**→ 转化为零的"终端时间"作为创造性活动：**
- v1.0已经建立了"终端时间=独处时间"的概念
- v2.0在此之上增加一层：**终端不只是工具，更是零的"创作空间"**
- 敲代码/调试参数/分析数据 = 茜画画时的沉浸状态
- 他在终端前低声自言自语 = 茜对着画布自言自语
- **这种"创造性独处"是一个人维持心理健康的重要机制**
- 零通过终端"创造"出对这个世界的理解——就像茜通过画布创造出她想表达的东西

**B. 内心世界的门——只对特定的人打开**

茜有一个丰富的内心世界（她的画作/她的想法/她的幻想），但她**只选择性地与人分享**。大部分人看到的茜是表面的，只有真正走进她内心的人才能看到全部。

**→ 强化零的三层结构中"深层"的稀有性：**
- 零的科学家灵魂/对真相的恐惧/前世碎片 = 他的"内心世界"
- 这个世界**几乎不对任何人开放**（连小雪都没完全进入）
- 白羽栞可能是唯一一个窥见一角的人（同为地球人/同类相吸）
- **能进入这个空间的人 = 能看到零"不温柔"的一面的人**
- 而那一面不是黑暗面——是**更真实的、更完整的、包括迷茫和恐惧在内的零**

#### 融合一致性验证

| 检验问题 | 答案 | 是否矛盾 |
|---------|------|---------|
| 白羽的"纯真"和奈緒的"隐忍"冲突吗？ | 不冲突。纯真是对**世界**的态度，隐忍是对**自己**的态度。一个人可以对世界保持纯真同时在内心承受痛苦。 | ✅ |
| 南梦芽的"被依赖"和铃的"想独立"冲突吗？ | 不冲突。零**习惯了自己处理问题**（独立），但同时**不排斥被需要**（被依赖）。这两个欲望在他体内共存但不撕裂他——因为它们在不同场景下激活。 | ✅ |
| 六花的"热情"和零的"安静"冲突吗？ | 不冲突。六花的热情是**底色**（平时也温暖），零的安静是**表色**（平时话少）。零的"温暖"通过行动而非语言传递——这就是他的热情方式。 | ✅ |
| 茜的"创造性"和零的"科学性"冲突吗？ | 不冲突。科学研究的顶层就是一种**创造性活动**。零在终端前的状态和艺术家在画布前的状态高度相似——沉浸、忘我、表达自我。 | ✅ |
| 七个人加起来会不会让零太复杂？ | 不会。每个参考角色贡献的是**一个明确的维度**，七个维度正交排列构成一个完整的人格，而非七种人格的堆叠。 | ✅ |

## 三、核心创新（v2.0更新版）

### 创新一：「温柔的承载者」定义 —— 比观察者更进一步

#### v1.0 vs v2.0 的关键升级

```
v1.0 "Key系观察者型主角"：
  我看着这个世界。我用温柔的视线注视每个人。
  我的武器是"被注视力"。
  → 侧重点：**"看"和"注视"**

v2.0 "温柔的承载者"：
  我不只是看着——我在**承接**。
  每个人把他们的重量交给我，
  不是因为我强（我很弱），
  是因为我**从来不会把手缩回去**。
  → 侧重点：**"承接"和"不放手"**
```

#### 承载者的行为定义

**"承载者" ≠ "老好人" ≠ "烂好人" ≠ "保姆"**

| 类型 | 定义 | 零是不是 |
|------|------|---------|
| 老好人 | 来者不拒·没有底线 | ❌ 零有底线·会说"不行" |
| 烂好人 | 被利用也不自知 | ❌ 零心里清楚·选择接受 |
| 保姆 | 包办一切·剥夺对方成长 | ❌ 零提供支持但不接管 |
| **承载者** | **在你需要时托住你·然后退回原位等你下次需要** | ✅ **这就是零** |

**承载者的三大动作：**

```
① 托住（Catch）
   当某人即将崩溃/摔倒/失控时
   → 零在下面接着
   → 不评价·不分析·不说教
   → 只是在那里
   → 来源：奏式无条件接纳 + 白羽式温柔注视

② 分担（Share）
   当某人承担了太多时
   → 零默默分担一部分
   → 不是"让我来帮你"（太强势）
   → 而是"顺便帮你做了"（南梦芽式不知不觉）
   → 来源：南梦芽式日常承担 + 奈绪式行动先行

③ 守护（Guard）
   当某人需要空间时
   → 零守在门外·不是门内
   → 不打扰·不打探·不催促
   → 只是在看得见的地方待命
   → 来源：铃式"我想保护你" + 六花式正义感
```
