## 一、设计总览

### 1.1 用户的核心诉求（原文摘录）

> "根据R3的参考设计黑铁真夜任务形象，重点参考高攻低防，枣伊吕波，一色伊吕波，橘万里花这三个人，其他轻微参考"

### 1.2 从现有设定到v1.0的本质变化

| 维度 | 现有设定（基础版） | v1.0（当前版本） |
|------|-------------------|------------------|
| **核心定位** | "第四女主·暗杀家族·隐藏觉醒者" | **"高攻低防小恶魔系学妹"** —— 以性格驱动而非背景驱动 |
| **性格深度** | 有高攻低防+小恶魔标签但未展开 | **完整的攻防切换机制 + 三层人格结构** |
| **与零的关系起点** | 暗杀任务→感化→暗中保护 | **保留但深化：从"猎物"到"最有趣的玩具"再到"唯一能接住我的人"** |
| **参考来源** | 无明确参考 | **枣伊吕波/一色伊吕波/橘万里花三主参 + 其他轻参** |
| **萌点系统** | 角/异色瞳/高攻低防（散点式） | **系统化：角感情开关 + 异色瞳语言 + 攻防切换表 + 夜行性反差** |
| **弱点设计** | 有但不完整 | **完整的"防御崩溃谱系"——不同类型攻击对应不同崩溃方式** |

### 1.3 设计目标

1. **建立"高攻低防"的系统化模型** —— 不是一句标签，而是一套完整的行为机制
2. **让小恶魔系讨喜而不惹人厌** —— 参考三个成功样本的共同公式
3. **将暗杀家族背景融入性格而非仅作设定** —— 她的"攻"和"防"都源于她的过去
4. **创造独特的年下学妹动态** —— 不是传统的"可爱学妹"，而是"危险的可爱"
5. **为后续剧本提供足够的行为框架** —— 台词模式、场景索引、成长弧光

### 2.2 枣伊吕波 — 「高攻低防·小恶魔系」的主要样本

#### 基础信息
**出处**: 《灰色的果实》系列
**身份**: 美滨学园学生 / 前偶像 / 家族企业继承人候补
**标签**: 小恶魔、適応力怪物、軽薄、隠された家族の闇、積極的、**被认真对待时反而不知所措**

#### 从枣伊身上提取的核心特质

**1. "高攻低防"的行为原型**

枣伊吕波的互动模式是**所有"高攻低防"角色的教科书级范例**：

```
【攻撃モード】
✓ 自分から距離を詰める（物理的·心理的両方）
✓ 含み笑いで相手を翻弄する
✓ 大胆な発言·ボディタッチを恐れない
✓ 相手の反応を楽しんでいるのがバレバレ
✓ 「ふふ～ん」「へぇ～」といった手前味噌なリアクション

【防御崩壊モード】（★ここが重要）
✗ 相手が真面目に返してくる → 動揺
✗ 相手が彼女を「一人の人間」として扱う → 対応できない
✗ 純粋な好意や優しさを受け取る → パニック
✗ 褒め言葉をストレートに言われる → 照れ隠しに余計なことを言って自爆
```

**→ 直接转化为真夜的"高攻低防"核心机制：**
- 真夜的"攻"同样来自**主动缩短距离、含み笑い、挑発言動**
- 真夜的"防"同样在面对**真诚回应时崩溃**——但原因不同（详见§三）
- 关键差异：枣的"防崩"来自"不习惯被认真对待"；真夜的"防崩"来自**"从未有人接住过我的全部"**

**2. 適応力怪物 × 軽薄の仮面**

枣的最大特质是她惊人的社会适应能力——能在任何环境中迅速找到自己的位置并如鱼得水。但这种适应力的代价是：
- **没有人知道真正的她是谁**（连她自己可能都不确定）
- "軽薄"是一种**生存策略**而非本性
- 她用"谁都能接受的样子"来换取"不被拒绝的安全感"

**→ 转化为真夜的"小恶魔面具"：**
- 真夜的"小恶魔"不是天性——是**暗杀家族生存训练的结果**
- "先に攻撃すれば傷つかない"（先下手为强就不会受伤）——这是从小被灌输的逻辑
- 她的含み笑い背后是**"如果我不逗你，你就可能来伤害我"的防御本能**
- 但与枣不同：真夜的"仮面"更**原始、更本能**——不是社交技巧而是生存技能

**3. 隐藏的黑暗面（家族之闇）**

枣的角色深度来自于她光鲜外表下的家族黑暗——这不是简单的"悲惨过去"，而是**持续影响她当前行为的结构性问题**。

**→ 转化为真夜的"革命孤儿"设定：**
- 枣的闇 = 家族企业的利益牺牲品
- 真夜的闇 = **革命失败的幸存者**（父母牺牲、幼年逃亡、以暗杀维生）
- 两者共享："看起来正常生活着的人，其实每天都在生存边缘"
- 真夜比枣多一层：**她不只是受害者，她也是加害者**（暗杀者身份）
- 这种**加害者×受害者的二重性**是真夜独有的深度来源

**4. 積極的アプローチ — "自分から行く"美学**

枣从不等待。她想接近谁就去接近，想要什么就去争取。这种行动力源于：
- **不想再被动地等待命运安排**（过去的创伤反应）
- **与其担心结果不如先行动**（实用主义哲学）
- **即使被拒绝也比什么都不做强**（存在确认需求）

**→ 完整继承给真夜：**
- 真夜同样是"自分から行く"型——但她的方式更具侵略性（物理距离压缩、突然出现、耳语）
- 动机类似：**与其担心被零讨厌不如先让他无法忽视我**
- 但真夜的行动力带有**暗杀者的职业习惯**（无声接近、从背后出现、掌握对方节奏）

### 2.3 一色伊吕波 — 「年下小恶魔·元気系」的主要样本

#### 基础信息
**出处**: 《五等分的新娘》（及其外传《∬》）
**身份**: 编辑部打工高中生 / 漫画家助手 / 年下女主角
**标签**: 元気、小恶魔、**からかい好き**、年上にタメ口、**実は真面目**、行動力、**被认真回应时慌张**

#### 从一色身上提取的核心特质

**1. 年下キャラの"攻撃的親密さ"**

一色最大的特色是她对年上角色的态度——**完全不分辈分地靠近**：

- 对风太郎（前辈）：直接叫名字、开黄色玩笑、物理距离极近、毫无顾忌地撒娇
- 对年上同事：同样不客气，甚至更加放肆
- 核心：**"年下だからこそ許される"を最大限活用する**

**→ 转化给14岁真夜对16-17岁零的态度：**
- 真夜叫零"先輩"但语气完全没有敬意的意思——更像一个**昵称/标签**
- 物理距离自动拉近到30cm以内——**不是因为喜欢而是因为"学妹特权"**
- 用"先輩"这个称呼来做任何事情：撒娇、命令、挑衅、求助
- "先輩ってば～"这句话在她嘴里可以表达至少8种完全不同的情绪

**2. からかい好き = 最高的娱乐形式**

一色的人生乐趣之一就是**看别人吃瘪**：

- 故意说错话看对方反应
- 在关键时刻捣乱然后装无辜
- 把对方的窘迫当作最好的笑料
- **但她从来不是恶意的**——这是一种亲昵的表达方式

**→ 转化为真夜的核心娱乐来源：**
- 真夜"吓零一跳"的习惯直接继承自此
- 从背后拍肩膀、突然出现在转角、在他耳边说"先輩～"
- **零的心跳加速反应是她重要的快乐指标**（虽然嘴上说"只是实验"）
- 区别：一色的からかい是"友好的玩笑"；真夜的からかい带有一层**"我在测试你的反应速度"的职业惯性**

**3. "実は真面目"的反差**

一色在漫画/工作相关的事情上**极其认真**：

- 作为助手时的专业度远超同龄人
- 对自己负责的稿件质量有极高要求
- 工作模式和日常模式判若两人

**→ 转化为真夜的"战斗模式 vs 日常模式"反差：**
- 日常：吊儿郎当、上课睡觉、数学及格线徘徊
- 战斗/暗杀：**瞬间切换为极致专业**——冷静、精准、高效
- 这种反差的来源：**战斗是她唯一被系统训练过的领域，也是她唯一确信"我不会搞砸"的领域**
- 类似于一色"只有在工作时才是认真的大人"——真夜"只有在战斗时才是完整的自己"

**4. 被认真回应时的慌张**

一色的经典模式：**把别人逗得团团转，但当对方认真起来时自己先慌了**。

典型场景：一色开玩笑说"前辈你喜欢我对吧"，风太郎认真思考了一下说"确实……"，然后一色**当场石化**。

**→ 这是"高攻低防"的最精确表达，直接构成真夜的核心萌点：**
- 真夜每天对零发动10次"攻撃"（挑発/からかい/近距離攻撃）
- 零只要认真回应**1次**，真夜全天防御归零
- 具体表现：耳尖红（角的根部）、声音变小一个八度、视线开始做布朗运动、"……な、なに言ってんだ急に"

### 2.4 橘万里花 — 「一途·行動力·二面性」的主要样本

#### 基础信息
**出处**: 《伪恋（Nisekoi）》
**身份**: 武士世家大小姐 / 医院院长千金 / 前未婚妻
**标签:** **一途**、**行動力MAX**、**二面性**（表=お嬢様/裏=ヤンキー気質）、**妄想癖**、**強固意志**、可愛い

#### 从万里花身上提取的核心特质

**1. 一途（Yi-chu）— 不动摇的感情方向**

万里花最核心的特质：**一旦决定了就不会改变**。

- 对一条乐的感情：从第一话到最后一话，从未动摇
- 行动上：想做什么就去做，不在乎别人的眼光
- 表达上：直球进攻，不会绕弯子

**→ 转化为真夜的一旦认定后的执著：**
- 真夜不是一见钟情型（初期关系=暗杀目标），但**一旦好感度跨过某个阈值**就会变成万里花式的一途
- 这个阈值 = **"这个人没有因为我的身份（魔族/暗杀者）而远离我"**
- 跨越之后：真夜会变得异常坦率（相对她自己的标准）、主动、不容许其他人抢走零的关注
- 区别：万里花的一途是**从一开始就有**；真夜的一途是**经过考验后才激活的**（更有戏剧张力）

**2. 行動力MAX — "想到就做"**

万里花的行动力在所有galgame女主中属于顶级：
- 想要接近一条乐？转学到他的学校
- 想要他注意到自己？制造各种"偶然"相遇
- 想要表白？直接策划大型告白作战

**→ 转化为真夜的"夜袭"行为模式：**
- 真夜的行動力体现在**非正常时间段**：深夜爬到零窗边、凌晨在门口等、半夜发消息
- 不是万里花式的"光明正大的追求"而是**暗杀者风格的"渗透"**
- 共享逻辑：**"我想见他 → 所以我现在就去见他了"** —— 中间没有任何犹豫环节
- 真夜的特殊限制：白天要上学/躲人，所以**行動力集中在夜间释放**

**3. 二面性 — 表与里的极端反差**

万里花的二面性是角色设计的核心：
- **表**：优雅的大小姐、成绩优秀、举止得体、说话温柔
- **裏**：不良少女气质、暴力倾向、说脏话、战斗力爆表
- 切换条件：**是否涉及一条乐**（涉及→裏出没 / 不涉及→表维持）

**→ 转化为真夜的"白天vs黑夜"二面性：**

| 维度 | 白天的真夜 | 深夜的真夜 |
|------|-----------|-----------|
| 性格 | 吊儿郎当的小恶魔 | 安静的、真实的自己 |
| 表情 | 含み笑い（常驻） | 无表情或淡淡的微笑 |
| 语言 | 挑発的·からかい満載 | 少言的·正常的对话 |
| 行动 | 制造骚动·引人注意 | 独自待在屋顶·看月亮 |
| 战斗意愿 | 低（能不打就不打） | 高（如果有需要的话） |
| 对零的态度 | "逗他玩" | "想和他在一起但不知道怎么表达" |

**关键差异**：万里花的二面性是**人格切换**；真夜的二面性是**环境依赖型的自然状态变化**——不是她在"扮演"两个角色，而是她在两种环境下呈现不同的真实面貌。

**4. 妄想癖 — 脑内剧场丰富**

万里花的想象力极其丰富：
- 经常在脑内上演各种和一条乐的场景
- 有时候会不小心把脑内台词说出来
- 妄想内容从纯爱到不可描述应有尽有

**→ 转化为真夜的"独白/自言自语"习惯：**
- 真夜不会大声说出妄想（暗杀者的职业素养=控制输出）
- 但她会在**无人的地方**（屋顶/深夜的教室）自言自语
- 内容通常是对今天和零互动的"复盘"+"如果当时我说了XX会怎样"的模拟
- **偶尔被零撞见她在自言自语 = 最大级别的防御崩溃事件之一**

### 2.6 轻微参考角色（补充细节）

#### 七宫智音（干物妹！小埋）— 外表vs日常反差
- **提取**：外面是完美大小姐，回家后变成不修边幅的干物妹
- **转化**：真夜的"校服内的另一面"——脱掉外套后可能是完全不同的状态（比如里面穿着运动服/有暗杀用的装备）

#### 中野三玖（五等分的新娘）— 内敛感情的爆发
- **提取**：平时冷静克制，但感情积累到临界点后会爆发
- **转化**：真夜的小恶魔面具**不是永远不会破**——在某些特定触发条件下（如提到父母/背上的伤疤/ひより），她会展现出从未示人的脆弱

#### 五更琉璃（俺妹）— 夜行性×双重身份
- **提取**：白天是普通高中生，晚上是网络作家"黑猫"；夜行性；表里差距极大
- **转化**：真夜的**昼夜二面性**的直接参考——白天的小恶魔和晚上的安静少女几乎是两个人

## 三、三大核心创新

### 创新一：「攻防切换矩阵」—— 高攻低防的系统化模型

#### 为什么需要系统化？

现有的"高攻低防"只是一个标签。为了让作者在任何场景中都能准确写出真夜的反应，我们需要把它变成一套**可查询的行为矩阵**。

#### 真夜的攻防切换矩阵

```
┌─────────────────┬──────────────────────────────────────┐
│   【攻撃モード】  │         発動条件：真夜が主導権を持つ時      │
├─────────────────┼──────────────────────────────────────┤
│ レベル1·軽攻撃    │ 含み笑い + "ふぅん" + 観察眼             │
│                 │ （默认状态·随时待机）                     │
├─────────────────┼──────────────────────────────────────┤
│ レベル2·中攻撃    │ 物理距離30cm + 上目遣い + 挑発的発言        │
│                 │ （零が反応した時·燃料補給）               │
├─────────────────┼──────────────────────────────────────┤
│ レベル3·強攻撃    │ 耳元囁き + ボディタッチ + 危険な賭け        │
│                 │ （余裕がある時·または焦っている時）          │
├─────────────────┼──────────────────────────────────────┤
│ レベル4·必殺      │ 直球の好意表現（稀·本人も予期せず）          │
│                 │ （防御が完全に外れた時のみ）                │
└─────────────────┴──────────────────────────────────────┘
                          ↕ 状態遷移トリガー
┌─────────────────┬──────────────────────────────────────┐
│  【防御崩壊モード】 │     発動条件：零が真剣に返してきた時        │
├─────────────────┼──────────────────────────────────────┤
│ レベル1·動揺      │ 含み笑いが一瞬止まる + "え？"              │
│                 │ （轻微困惑·可快速恢复）                   │
├─────────────────┼──────────────────────────────────────┤
│ レベル2·照れ      │ 耳（角の根元）赤くなる + 視線逃避           │
│                 │ + "……なに言ってんだ急に"                  │
│                 │ （标准崩坏·最常见的萌点场景）               │
├─────────────────┼──────────────────────────────────────┤
│ レベル3·パニック  │ 言葉に詰まる + 身体が後ずさり + 角が激しく   │
│                 │ 揺れる + "なんでもない！"と強制終了          │
│                 │ （重度崩坏·需要时间恢复）                  │
├─────────────────┼──────────────────────────────────────┤
│ レベル4·素が出る  │ 小恶魔面具完全脱落 + 無防備の表情            │
│                 │ + 本音が漏れる（非常に稀）                  │
│                 │ （终极崩坏·只在极度特殊的触发条件下）         │
└─────────────────┴──────────────────────────────────────┘
```

#### 触发条件的精确定义

什么算"零が真剣に返してきた"？以下是**真夜判定基准**（不是客观基准，是真夜的主观感受）：

| 零的行为 | 真夜的主观判定 | 防御等级 |
|---------|--------------|---------|
| 无视她的挑発 | "つまんない先輩" | 不触发（继续攻撃） |
| 配合她的挑発反击 | "ふふ、いい反応ね" | 不触发（升级攻撃） |
| 叹息/无奈 | …… | Lv1 动揺（一瞬） |
| 认真看她一眼 | ？？ | Lv1~2 |
| 说"你的眼睛很漂亮" | **！！！** | **Lv2~3 照れ/パニック** |
| 问"你为什么对我这么好？" | **脳内フリーズ** | **Lv3~4** |
| 主动缩短物理距离 | **？？？何これ？？** | **Lv2~3** |
| 在她脆弱时认真关心 | **完全無防備** | **Lv4 素が出る** |

**关键设计**：真夜的"防"不是"厚"而是**"根本不存在"**。她的高攻本质上是一种** preemptive defense**（先制防衛）——只要一直在攻击，就没有人有机会看到她不设防的样子。

### 创新二：「角感情开关」+「异色瞳语言」—— 非语言的萌点系统

#### 角感情开关（Horn Emotion Indicator）

真夜的角（約2cm·刘海中若隐若现）是她**唯一无法控制的生理指标**——比心跳更诚实。

| 情绪状态 | 角的状态 | 本人自觉 |
|---------|---------|---------|
| 平静/无聊 | 完全隐藏在刘海中 | 无 |
| 轻微兴趣/愉悦 | 角尖微微露出 | 通常无 |
| 兴奋/得意 | 角完全露出·微微颤动 | 偶尔察觉 |
| 困惑/思考 | 角左右不对称（一高一低） | 无 |
| **害羞/动揺** | **角剧烈震颤·颜色微变（深→浅）** | **完全无自觉** |
| **愤怒/警戒** | **角竖直·发出微弱光芒** | **有自觉（但不能控制）** |
| 悲伤/难过 | 角无力下垂·缩回刘海深处 | 无（也不想知道） |
| **对零的好感累积** | **角在零面前露出的频率逐渐增加** | **死活不承认** |

**叙事功能**：
- **读者/零能看到但真夜自己不知道** = 信息不对称萌点
- 角成为真夜情感的**"测谎仪"** —— 她嘴上说"才没有开心"但角在疯狂抖动
- 后期零学会"读角"之后可以精准打击真夜的防御盲区

#### 异色瞳语言（Heterochromia Language）

左深红·右金的异色瞳不只是外观特征——它是一套**微妙的情绪表达系统**：

| 情境 | 左眼（深紅） | 右眼（金） | 综合含义 |
|------|------------|-----------|---------|
| 小恶魔模式 | 半眯（戏谑） | 明亮（聚焦） | "我在观察你的反应" |
| 认真时刻 | 全开（凝视） | 柔和（放松） | "我在听你说" |
| 战斗状态 | 锐利（锁定） | 冷淡（分析） | "目标确认" |
| **动揺状態** | **闪烁不定** | **回避接触** | **"别看我"** |
| **深夜（屋顶）** | **黯淡** | **遥望远方** | **"在想事情"** |
| 提到父母/伤疤 | **失去光泽** | **紧闭** | **话题终结** |

### 创新三：「夜行性反差」—— 同一个人的两种生存模式

#### 设计概念

真夜不是"白天装样子晚上做自己"——她是**真的在两种状态下呈现不同的真实面貌**。

这不同于万里花的"表/裏人格切换"（那是表演性的），也不同于智音的"家外/家中"反差（那是生活习惯）。真夜的昼夜差异来源于**生物性+文化性+心理性的三重叠加**：

```
【生物性】魔族/龙族后裔的夜行性基因
  → 白天代谢慢·反应迟钝·嗜睡
  → 夜间感官敏锐·战斗力最大化

【文化性】暗杀家族的职业训练
  → 白天 = "混入人类社会"模式（保持低调·不引人注目）
  → 夜间 = "活动时间"（执行任务·自由行动·做真正的自己）

【心理性】创伤后的安全区认知
  → 白天 = 危险的（人群·注视·可能的敌意）
  → 夜间 = 安全的（黑暗·安静·没人看见脆弱的自己）
```

#### 昼夜对照详表

| 维度 | 白天（学園モード） | 深夜（真夜モード） |
|------|-------------------|-------------------|
| **起床时间** | 7:30（极限）·经常迟到 | 21:00（真正的一天开始） |
| **精神状态** | 半永久瞌睡·靠窗位置固定 | 异常清醒·眼睛发亮 |
| **说话方式** | 挑発的·からかい満载·含み笑い | 正常的·少言的·偶尔自言自语 |
| **社交距离** | 30cm以内（侵入式） | 3m以外（回避式） |
| **对零的态度** | "逗他玩"·"先輩ってば～" | 想靠近但不知道怎么开口 |
| **战斗意愿** | 能躲就躲（省能量） | 如果需要的话·全力以赴 |
| **食欲** | 随便塞点什么·便利店饭团 | 烧烤摊·大量肉食·超辛カレー |
| **弱点暴露度** | 低（小恶魔装甲全开） | **高（装甲卸下·角自由露出）** |
| **最自然的笑容** | 含み笑い（表演性的） | 看月亮时无意识的淡淡微笑（真实的） |
| **典型场所** | 教室窗边·走廊·零的旁边 | 学園屋顶·時計塔·深夜的教室 |

**关键洞察**：零认识的真夜是**白天的真夜**（小恶魔·高攻低防）。但真正的真夜存在于**深夜的真夜**（安静·孤独·渴望连接却不敢伸手）。故事的本质张力之一就是**这两个真夜之间的鸿沟**——以及零何时/如何跨越它。

## 四、「高攻低防」的心理学解构

### 4.1 为什么"高攻"？

真夜的"攻撃性"不是天生的性格——它是**后天习得的生存策略**：

```
起源链条：
幼年逃亡（6-8岁）
  → 目睹父母牺牲
  → 被暗杀家族收养训练
  → 学到的第一条规则：「弱者が生き残れない」
  → 学到的第二条规则：「先に攻撃すれば傷つかない」
  → 内化为一套自动化行为模式：
     任何不确定的人际关系 → 先发起攻势试探
     任何人靠近 → 先评估威胁等级再用挑発建立优势位
     任何可能的脆弱暴露 → 立即用攻击性语言掩盖
```

所以真夜的"攻"本质上是** preemptive defense mechanism**（先制防卫機構）。她不是"喜欢欺负人"，而是**"如果不先出手，就会被伤害"的条件反射**。

### 4.2 为什么"低防"?

同样的经历导致了她的"低防"：

```
防御缺失的原因：
1. 从未被认真地·无条件地接纳过
   → 不知道如何应对纯粹的善意
   → 善意对她来说是"未知事物" → 未知 = 恐惧

2. 所有的"亲近"都带有目的性（暗杀任务的伪装/生存需要的交易）
   → 当零不带任何目的地对她好时
   → 她的认知系统中没有对应的处理程序
   → 结果：CPU过载 → 防御崩溃

3. 她的"攻"消耗了大量心理能量
   → 维持小恶魔形象是累的
   → 当零的真诚回应绕过了她的攻击层
   → 已经没有剩余能量来启动第二道防线
   → 结果：裸奔状态
```

### 4.3 枣/一色/万里的"低防" vs 真夜的"低防"

| | 枣伊吕波 | 一色伊吕波 | 橘万里花 | 黒鉄真夜 |
|--|---------|-----------|---------|---------|
| **低防原因** | 不习惯被认真对待 | 没想到会被认真回应 | 被戳中痛点时失控 | **善意是未知领域** |
| **崩坏速度** | 中（几秒内恢复） | 快（当场石化） | 快（秒怂） | **快（CPU过载）** |
| **恢复方式** | 用更多玩笑掩饰 | 装作什么都没发生 | 恼羞成怒 | **"なんでもない"強制終了** |
| **深层原因** | 家族の闇导致的自我价值感低 | 年下补偿心理 | 怕被拒绝 | **从未被无条件接纳过** |
| **独特性** | 適応力带来的疏离感 | 元气掩盖的不安 | 一途带来的焦虑 | **生存本能与渴望爱的矛盾** |

## 五、与零的关系发展路线图

### Phase 0 · 猎物（Ch01 Part 3 之前）

```
真夜视角：「ただの標的。金のためなら誰でもいい」
零视角：「存在しない人物（まだ出会っていない）」
互动：无
距离：∞
```

### Phase 1 · 无法杀死的猎物（Ch01 Part 3）

```
事件链：商会雇佣暗杀 → 零假死逃脱 → 二次暗杀 → 零设陷阱 → 真夜被困
真夜视角：「こいつマジで殺せない。……面白い。初めてだ」
零视角：「この子は……危ないけど、子供だよね？」
关键转折：嘴炮感化（"誰かのために戦ってるんじゃないのか"）
结果：放生 → 真夜不放弃暗杀工作但开始**暗中保护零**
距离：从"猎手-猎物"变为"有趣的玩具-持有者"
```

### Phase 2 · 最有趣的玩具（Ch02~Ch04 日常篇）

```
真夜的行为模式：
- 每天找理由出现在零面前（"偶然先輩"×n）
- 用各种方式挑発零（参考攻撃モード Lv1-3）
- 收集零的反应数据（暗杀者职业病：目标分析）
- 开始在深夜独自"复盘"今天的互动（创新三·夜行性反差）
- 嘴上说"只是确认目标状态"但实际上……

零的行为模式：
- 习惯了真夜的突然出现（从吓一跳到"哦你又来了"）
- 偶尔认真反击 → 享受真夜的防御崩溃
- 开始注意到"白天的真夜"和"偶尔看到的安静真夜"的差异
- 还不知道真夜在暗中保护自己

关键场景示例：
- 场景A：真夜从背后拍零肩膀 → 零头也不回"おはようまや" → 真夜Lv2崩坏
- 场景B：零认真问"你为什么总是帮我？" → 真夜Lv3パニック → "なんでもない！"
- 场景C：零在深夜偶然发现真夜一个人在屋顶 → 看到了"另一个真夜"
```

### Phase 3 · 从"玩具"到"特别"（半個人線）

```
触发条件：某个事件让真夜意识到"我不只是觉得他有趣"
可能是：
- 零在没有真夜保护的情况下遇到了危险（真夜的缺席焦虑）
- 零看到了真夜的角/伤疤/脆弱面但没有嘲笑或追问
- 零主动说"以后我来保护你吧"（对14岁的她说的话）

真夜的变化：
- 小恶魔模式的频率降低（不是消失·而是不再需要那么厚的装甲）
- 开始尝试正常的对话（虽然经常失败·因为不知道"正常"是什么样）
- 夜间"复盘"的内容从"今天逗了他几次"变成"他今天是不是有点……"
- **一途模式激活**（参考万里花·但有门槛）

关键设计：真夜的半個人線不是"从不喜欢到喜欢"——
**她从一开始就觉得零"有趣/特别"**，只是不知道这种"特别"的名字叫什么。
半個人線的作用是让她**给这种感觉命名**。
```

### Phase 4 · 承认之后的混乱（半個人線后 ~ 个人线前）

```
一旦承认（对自己）"我喜欢零"之后：
- 真夜变得更加矛盾：想靠近又怕被发现自己的感情
- 攻撃モード变得不稳定：有时候过度攻击（掩饰），有时候完全无法攻击（太在意了）
- 防御崩溃的触发阈值大幅降低（以前需要"认真夸"才会崩，现在"看了她一眼"就崩）
- 夜间行为变化：从"独自在屋顶"变成"在零可能出现的地方徘徊"
- 开始出现万里花式的行動力：想做某件事就直接做了（然后事后后悔）

与枣/一色/万里的对比：
- 枣：承认后变得更坦诚（因为她终于可以不做假了）
- 一色：承认后变得更害羞（因为不知道该怎么处理）
- 万里花：承认后变得更积极（因为终于可以光明正大了）
- 真夜：**承认后变得更加混乱**（因为她的整个行为系统都建立在"不要让人靠近"的前提上）
  → 这就是真夜独有的戏剧张力来源
```

## 六、台词设计原则

### 6.1 真夜的台词分类

| 类型 | 使用频率 | 示例 | 场景 |
|------|---------|------|------|
| **挑発型** | ★★★★★ | "先輩って、本当にダメだねぇ" | 日常互动·起手式 |
| **からかい型** | ★★★★☆ | "ふふ～ん、先輩その反応面白かったよ？" | 零中招后追击 |
| **撒娇型** | ★★★☆☆ | "先輩～、これ貸て～" | 想要某物时（用"先輩"当武器） |
| **防御崩壊型** | ★★★☆☆ | "……な、なに言ってんだ急に" | 被认真回应时 |
| **本音漏出型** | ★☆☆☆☆ | "……別に、君のこととか……" | 极稀有·Lv4崩坏 |
| **暗杀者模式** | ★★☆☆☆ | "ターゲット確認。行動開始" | 任务中/战斗中 |
| **深夜独白型** | ★★☆☆☆ | （自言自语·内容为当天复盘） | 独处时 |

### 6.2 台词对比：同一情境下的三种反应

#### 情境：零问她"你怎么又来了？"

| 反应模式 | 台词 | 条件 |
|---------|------|------|
| **攻撃Lv1** | "ふぅん、先輩がいるからじゃない？偶然だよ" | 日常·默认 |
| **攻撃Lv2** | （凑到30cm距离）"先輩、私のこと嫌いなの？"（上目遣い） | 余裕ある時 |
| **防御Lv2** | "……ただの通り道だよ。誤解しないでよね" | 被問到心坎时 |
| **本音（極稀）** | "……ここにいると、なんか落ち着くんだ" | 深夜·只有两人·守备全无 |

#### 情境：零夸她"今天很可爱"

| 反应模式 | 台词 | 条件 |
|---------|------|------|
| **攻撃Lv2** | "へぇ～、先輩意外と目が高いじゃん。残念だった？" | 即刻反击 |
| **防御Lv2~3** | "……は？ 何それ。変なこと言わないでよ" （耳尖红·视线飘走） | 没有预备 |
| **防御Lv3** | "あ、もう！ 知らないよ！" （角剧烈摇晃·转身就走） | 连续攻击 |

### 6.3 禁止使用的句式

❌ 传统傲娇句式（"才不是"、"无路赛"）—— 真夜不是傲娇
❌ 过于成熟的发言 —— 她14岁
❌ 过于可爱的发言 —— 她是小恶魔不是吉祥物
❌ 暗杀术语过多 —— 只有任务中使用
❌ 完全正常的发言 —— 她的正常是"不正常"的一部分

**✅ 真夜的语言特征**：
- 短句为主（不耐烦听长篇大论）
- "先輩"一词的多义性滥用
- 以"ふぅん"起始的句子占40%+
- 防御崩溃时语句完整性骤降（"……なに、これ……"）
- 深夜独白时反而会用完整的长句（因为不需要伪装）

### 7.2 真夜的独特生态位

**"唯一从'敌人'变成'同伴'的女主"**
- 朱音/艾露娜/小雪从一开始就是"自己人"阵营

**"唯一14岁的女主"**
- 年下 positioning = 独特的互动模式
- 可以做其他女主"不适合做的事"（更肆无忌惮的からかい·更直接的撒娇·更不计后果的行動力）
- 但同时也有"还是个孩子"的限制（某些话题她处理不了·某些决定她不够成熟）

**"唯一隐藏觉醒者"**
- 公开 = C~B级非觉醒者（"普通的中等部学生"）
- 真实 = R级觉醒者（早已觉醒·未登记）
- 这个秘密是**个人线的核心悬念之一**
- 也是她"不能被发现"的另一层压力来源

## 九、废弃的方向记录

### 9.1 已否定的设计方向

| 方向 | 否定原因 | 替代方案 |

| R级觉醒者早期暴露 | 失去隐藏觉醒者的戏剧张力 + 降低后续个人线冲击力 | 长期隐藏·仅在特定条件下暗示·个人线正式揭露 |

### 9.2 尝试但调整的方向

**方向A：让真夜的"攻"带有恶意**
- 调整：改为"攻带有目的性但不带恶意"——她的挑発是测试/娱乐/接近的手段，不是伤害
- 最终方案：小恶魔系的"攻"本质上是**亲昵的逆向表达**

**方向B：让真夜完全不擅长学习**
- 调整：改为"战斗直觉极强但理论课完全不行"——保留一个她擅长的领域（战斗）平衡"笨蛋"印象
- 最终方案：理论笨蛋 + 战斗天才 = 反差萌的一种

## 十、总结：真夜v1.0的核心公式

```
【表象】14歳学妹·小恶魔·高攻低防·異色瞳·角·夜行性
    ↓（穿透一层）
【中层】枣式適応力×一色式元気×万里花式一途
         → 融合为「攻撃的親密さ」行動模式
    ↓（穿透两层）
【内核】暗殺者の生存本能（先制防衛＝小恶魔面具）
         × 革命孤児の孤独（善意＝未知領域）
         × 14歳少女的純粋（一旦認めたら絶対に曲げない）
    ↓（穿透三层）
【底色】一个用"逗你玩"來說"靠近我一点好不好"的孩子
       因为她不知道還有別的方式可以表達
       而当她終於學會了的時候
       那大概就是這條線最好的結局
```

### 与其他三女主的设计公式对比

```
朱音：  身份の鎧 × 好勝心 × 口嫌体正直 → "不坦率是环境所迫"
艾露娜：天才の頭脳 × 温柔の心 × 論理不能 → "聪明人面对感情时的逃避"
小雪：  謎のベール × 古神の血 × 無邪気 → "最接近'真相'的普通人"
真夜：  攻撃の仮面 × 孤独の核 × 一途な core → "用推开的方式拉近距离"
```

**四人四种"无法坦率"的理由**—— 各不相同，各自成立。
