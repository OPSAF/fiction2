## 一、简介

艾露娜·温德尔是来自精灵王国的留学生，天使族后裔。初中起就以交换生身份进入星導学園，连续三年霸占初中部学年首席。

她是本作开局**综合实力最强的角色**——已达到宗師階（A级）魔法水平，在纯理论和实战层面都碾压同辈。但她主动选择转入理論科而非実践科，因此没有担任学术代表（那个位置归了白羽）。

性格上，她是典型的**天才+温柔治愈系**：对所有人都很温柔、很纯真、很会照顾人。但正因为太强、太聪明，她在故事初期带有一种**无意识的"傲"**——不是看不起人的傲慢，而是**对"常识"的不理解**。这种"傲"随着她融入集体、建立真正的朋友关系后逐渐消失。

她对主角零的态度经历了独特的转变轨迹：因一篇关于「波粒二象性」的学术论文而崇拜作者 → 发现作者疑似零后开始调查（学术兴趣驱动）→ 通过试探和观察确认零就是作者 → 开始在意零（仍是学术憧憬为主）→ 完成半个人线后产生了无法定义的「奇怪感觉」→ **开始躲着零**。

## 三、外貌特征

**淡金色长发**（天使族后裔特征）：月光下微弱发光（光属性残留的微弱表现）。平时扎成松散侧马尾，发丝间偶尔夹着做笔记时随手插的羽毛笔。

**翡翠色瞳孔**：精灵族天赋零粒子视觉 —— 能看见零粒子流动的轨迹与密度分布（天使族特殊感知能力的退化表达）。看人时有种**被X光扫描的感觉**（零的原话）。

**长耳约8cm**：情绪摆动的实时显示器 —— 前倾=集中/竖起=兴奋/微垂=困惑/平贴=害羞或不安/轻颤=紧张。本人以为控制得很好，实际上**全员可见**。

**高挑纤长**（168cm）：精灵族特有优雅身形。站姿笔直得不像高中生（天使族后裔/精灵族的优雅体态训练从五岁开始）。

**校服细节**：白色或黑色过膝袜（根据天气切换）。脚偏大（精灵族特征），常年穿定制鞋。

## 四、性格

### 四层结构：天才的外壳 × 温柔的核心 × 纯真的底层 × 迷茫的表层

```
第一层：「无意识的傲」（仅初期）
│  不是看不起人，而是对"常识"的不理解
│  "诶？这个不是大家都知道的基础吗？"
│  "原来人类在这个年纪才学这个啊"
│  ↓ 随着她理解"普通人"的存在后迅速消退
│
第二层：天才的思维模式
│  纯粹的知识欲驱动力
│  对"不懂的东西"有近乎本能的追求
│  逻辑至上，但不会用逻辑去压迫别人
│  ↓
第三层：温柔的治愈系内核
│  对所有人都很好，发自内心地关心
│  会注意到别人注意不到的小细节
│  默默付出，不求回报
│  ↓
第四层：纯真 + 恋爱概念天然
│  不知道"喜欢"是什么感觉
│  可以一本正经地问出让全场石化的问题
│  对感情的反应延迟3-5秒（天然发言→才意识到尴尬）
```

### 关于「初期微傲」的设计说明

艾露娜的"傲"**只存在于故事初期**，而且有以下严格限定：

| 特征 | 有这种"傲"的角色 | 艾露娜的版本 |
|------|------------------|-------------|
| 来源 | 天生高贵/优越感 | **纯粹因为不知道**——她真的以为那些知识是"大家都应该知道的" |
| 表现 | 看不起人/嘲讽 | 困惑/真诚地提问/"诶？" |
| 持续时间 | 整个故事 | **约1-2章**（到她第一次被普通人的生活智慧"打败"为止） |
| 消失方式 | 被主角打脸/剧情 forced | **自然消退**——她通过观察和学习逐渐理解了"常人"的世界观 |
| 消失后的状态 | 变成温柔系 | **变成纯粹的温柔治愈系**——"傲"的痕迹几乎不复存在 |

**经典初期台词 vs 后期对比**：

| 场景 | 初期（带微傲） | 后期（纯温柔） |
|------|---------------|--------------|
| 别人解不出题 | "这个……不是基础公式直接就能推导出来的吗？"（真诚困惑） | "这里可能需要换个思路，要不要我帮你梳理一下？"（温和引导） |
| 别人不知道某概念 | "啊，原来还没学到这个部分啊"（微微点头） | "这个概念确实有点抽象，我用个例子解释一下吧"（耐心教学） |
| 自己犯错时 | "……这不可能。我的计算不可能出错。"（短暂僵硬后自我怀疑） | "啊，果然是我哪里弄错了呢。谢谢你指出来。"（坦然接受） |

## 五、能力设定

### 魔法能力：A级（宗師階）

- **学派**：空间学派（主）+ 风属性表现（辅）
- **战斗风格**：精确、高效、最小化魔力消耗。能用初級魔法解决的问题绝不用中級。
- **限制/代价**（参考洛琪希的"雷光只能用一次"）：
  1. **宗師階术式消耗极大**：全力输出一次后需要长时间恢复（半天~一天），所以日常基本不用
  2. **实战经验不足**：转理論科后缺乏实战训练，真正打起来可能不如白羽流畅
  3. **过度依赖视觉**：零粒子视觉在某些干扰环境下（如强光/浓雾/特定结界）会失效或降低精度
  4. **身体强度一般**：精灵族体质偏向敏捷而非耐打，物理防御是短板

### 知识能力：全角色最高

- 初中三年学年首席（三连霸）
- 对魔法的理论理解远超同龄人（甚至超过大部分教师）
- 能阅读精灵古文、帝国古语、多种古代文献
- 记忆力和分析能力极强 —— 但**生活常识严重不足**

### 特殊能力：零粒子视觉（精灵族天赋）

- 能看到零粒子流动的轨迹、密度分布、性质变化
- 能察觉到零身上某些"理论上说不通"的现象（虽然零的F级伪装完美）
- 她看到的不是"零粒子亲和性异常"，而是**"异常的理论理解"**——一个F级学生不应该有的深度洞察力
- 这个能力是她对零产生最初兴趣的关键触发器

## 六、与零的关系

### 关系发展时间轴

```
Phase 0 · 远距离崇拜（初中～高中入学前）
│  在学术期刊上读到零（匿名）发表的「波粒二象性」论文
│  "终于有人用数学语言描述了我一直感觉到的东西！"
│  将作者视为精神导师级别的存在
│  但不知道作者是谁（匿名发表）
│
Phase 1 · 同班同学（高中入学～第1-2章）
│  发现零就在自己班上（同为理論科）
│  注意到零的某些言论和论文风格惊人相似
│  "这个人……有点奇怪。他的思维方式……好像在哪里见过。"
│  开始不动声色地"调查"（观察、试探对话、留意写作习惯）
│  此阶段：学术兴趣 > 个人好感。零对她来说是一个"有趣的研究对象"
│
Phase 2 · 接近与确认（第3-5章）
│  通过多次"试探"（假装随意地聊魔法理论话题）确认零就是论文作者
│  典型试探方式：
│   "你对波粒二象性怎么看？最近有篇论文写得很有意思呢"
│   "如果让你用数学描述零粒子的波动性，你会怎么建模？"
│  零的回答每次都精准命中论文核心观点 → 确信度逐次累积
│  最终确信：**零就是作者**
│  **但没有揭穿** —— 她想看看零还会写出什么
│  此阶段开始从"研究对象"变成"在意的人"
│  表现：开始找借口和零说话、会在意零和其他女生的互动（但自己没意识到这是在意的表现）
│
Phase 3 · 自然相处（第6章～半個人線前）
│  已经是关系很好的同班同学/研究伙伴
│  经常一起在图书馆/讨论室待到很晚
│  对零的好感持续升温但仍停留在"憧憬+欣赏"层面
│  本人完全没意识到自己对零的感情已经超越了学术兴趣
│  典型的天然迟钝："为什么我看到他和别的女生说话时耳朵会不舒服？大概是零粒子波动紊乱了吧"
│
Phase 4 · 半個人線完成 → 异常感知（半個人線後）
│  完成自己的半個人線事件后（内容待定）
│  某一天突然发现自己在零面前有种"奇怪的感觉"
│  心跳加速、注意力无法集中、不想直视零的眼睛
│  **"这是什么？书上没有写过这种感觉。……我不喜欢这种失控感。"**
│  → **开始躲着零**
│  找各种借口不单独见面、对话时保持距离、避免眼神接触
│  但越躲越容易暴露 —— 因为她平时的行为模式是"见到零就自然靠近"的
│
Phase 5 · 个人线（攻略後）
│  （待详细展开）
```

### 关键设计原则

**这不是传统的"一见钟情→追求→在一起"路线。**

艾露娜的情感发展是**反向的**：
- 大多数角色：好感度从低→高（逐渐喜欢）
- 艾露娜：**好感度从一开始就很高（崇拜/欣赏），但性质从"非恋爱"向"恋爱"转化时她产生了恐惧反应（躲避）**

这种**"因为太在意所以逃避"**的模式，配合她的天然/纯真属性，构成了独特的萌点。

## 七、与白羽的关系 — 「星導双璧」

### 初中时代的"暗中对手"

| 维度 | 艾露娜 | 白羽栞 |
|------|--------|--------|
| 初中成绩 | **学年首席（三连霸）** | 永远的第二名 |
| 魔法水平 | **A级（早已达到宗師階）** | B+级（一直在追赶） |
| 学生中的印象 | "那个精灵族的天才" | "那个努力型的全能优等生" |
| 对"竞争"的认知 | **甚至没意识到这是竞争** | 一直把艾露娜当成目标 |

**关键点：这是艾露娜单方面的"碾压"，而且她根本没当回事。**

对艾露娜来说，白羽只是一个"还不错的同学"。但对白羽来说，艾露娜是一座**永远追不上的高山**。

然而两人的关系**非常好**：
- 经常一起在图书馆学习（艾露娜看书，白羽刷题）
- 白羽会帮艾露娜处理各种行政手续（艾露娜在这方面是小白）
- 艾露娜会帮白羽解答她卡住的难题（但解答方式经常是"你这样想……"然后白羽听不懂）
- 午休时经常凑一起吃饭

### 「星導双璧」

其他学生给她们起的组合名。含义：星導学園初中部的两根支柱，一个代表**绝对天赋**（艾露娜），一个代表**绝对努力**（白羽）。

每当考试成绩发布时的固定节目：
1. 艾露娜的名字出现在榜首
2. 白羽的名字紧随其后
3. 全班齐声：「星導双璧又来了——！」
4. 艾露娜：（困惑脸）"他们在说什么？"
5. 白羽：（苦笑）"别理会……"

**到了高中后这个组合名逐渐没人提了** —— 因为艾露娜转去了理論科，不再参加実践科的排名。但白羽偶尔还是会用这个名字调侃她。

## 八、人际关系

### 与朱音

- 朱音对艾露娜有一种**复杂的微妙感**：一方面承认艾露娜的实力（A级vs自己的B级），另一方面觉得艾露娜"太过完美"让人不太舒服
- 艾露娜对朱音的评价："天宮家的思维方式很有趣。她明明可以直接说出来却总是绕弯子。——也许是某种社交防御机制？"
- 两人偶尔会因为零的话题产生微妙的对立（但都不是主动的）

### 与真夜

- 真夜觉得艾露娜**太难懂了**（理论vs实战的根本冲突）
- 艾露娜觉得真夜的战斗方式"效率很低但结果意外地好"
- 互补关系：艾露娜负责分析和策略，真夜负责执行

### 与小雪

- 小雪是少数能让艾露娜**展现出完全放松状态**的人
- 小雪给的甜点艾露娜会礼貌吃完但表情管理得很辛苦
- 艾露娜是小雪的"精灵文化咨询窗口"

## 九、喜好与生活习惯

### 喜欢的事物

- **古籍和古代文献**：纸张泛黄的手感、古体字的解密过程。在地下図書館B5层度过了最多的时间
- **新知识和未知现象**：发现"不懂的东西"时的兴奋感。零身上那些"理论上说不通但确实存在"的现象是目前最大的研究课题
- **月光下的研究时间**：天使族后裔对月光的天然亲近感。夜间图书馆的静谧
- **风拂过长耳的感觉**：微风拂过时会有一种"世界在对我说什么"的安心感
- **精确的数据和逻辑链**：模糊的描述让她不舒服。一切必须可量化、可验证、可重复
- **精灵传统弦乐器**（类似低音竖琴）：独奏时耳朵会随着旋律轻轻摆动。不在他人面前演奏

- **有建设性的学术争论**：不是为了赢而是为了逼近真理的辩论
- **咸味煎饼**：在帝都发现的意外喜爱（口感干脆·不甜·适合边吃边看书）
- **高智商的人**：**这是她的特殊开关** —— 对聪明人有天然的好感和亲近欲。零就是因为通过了这个开关才引起她注意的

### 讨厌的事物

- **不合逻辑的事**："直觉""感觉""大概"这类词汇在她的对话中出现频率为零
- **低效率的方法**：每一秒都在浪费生命

- **人类复杂暧昧的社交规则**：言外之意、面子工程、客套话让她头痛。零是她遇到的第一个"说话直来直去"的人类
- **鞋子不合脚**：走路久了小趾会痛（隐藏的不适）

## 十一、经典场景

### 场景1：初遇的"微傲"（第1章）

零在理論科教室第一次见到艾露娜。她正在用精灵文字以三倍速度记笔记，周围同学在窃窃私语"那个传说中的初中首席"。

零不小心碰掉了她的笔。她低头捡起来，用那种**不带任何感情的平静语气**说：

「——落ちたわよ。重力加速度は9.8m/s²で、この角度からの落下時間は約0.4秒。計算上は当たり前のことだけど。」

（——掉了哦。重力加速度9.8m/s²，从这个角度落下来大约0.4秒。计算上是理所当然的事。）

零：「……你在跟我讲物理吗？」

艾露娜：「え？ごめんなさい。習慣で。（停顿）……あ、拾ってくれてありがとう。」

**这是她"微傲"的典型表现 —— 不是故意炫耀，而是真的觉得这些是"理所当然该知道的事"。**

### 场景2：论文作者的试探（第3-4章）

图书馆。艾露娜假装随意的语气：

「ねえ、零。波動と粒子の二重性についてどう思う？ 最近読んだ論文がとても面白かったの。」

（呐，零。关于波粒二象性你怎么看？最近读到的论文非常有意思。）

零（内心）：？？？（那是我写的吧？？？）

零：「呃……我觉得……波和粒子可能都是……某种更深层的……同一个东西的不同表现？」

艾露娜：（耳朵微微竖起——捕捉到关键词了）「同じものの違う表現……。ふむ。面白い視点ね。もう少し詳しく聞かせて。」

（同样的东西的不同视角……嗯。有意思的观点。再详细说说。）

**——这次对话后，艾露娜的確信度从30%提升到了65%。**

### 场景3：确信时刻但不揭穿（第5章）

某次讨论中，零无意中使用了一个**只有论文作者才会用的独特表述方式**（比如某个自创的数学符号用法）。

艾露娜的动作序列：
1. 笔尖停在纸面上（0.5秒）
2. 翡翠色瞳孔微微放大
3. 左耳竖起到极限然后强行压回去
4. 恢复正常表情（全程不超过3秒）

内心独白：「——やっぱり。彼だ。間違いなく。でも……まだ言わない。もっと見てみたい。彼がどこまでいくのか。」

（——果然是他。没错。但是……现在不说。我想再看看。他能走到哪一步。）

### 场景4：半个人线后的"逃亡"（半個人線後）

完成某个关键事件后的第二天。艾露娜在走廊看到零正朝她走来。

正常人反应：打招呼/微笑/点头。

艾露娜的反应：
1. 视线接触的瞬间 —— 耳朵**猛地平贴**
2. 「あ、そうだ。図書館の予約資料が来てるか確認しなきゃ。」
3. 以**竞走速度**转向离开
4. 在转角处差点撞上白羽

白羽：「艾露娜？你跑什么？」

艾露娜：（耳朵还在抖）「没、没什么。只是突然想起一件很重要的事。非常重要。关乎……关乎学术研究的进展。」

白羽：「……你和零出什么事了吗？」

艾露娜：「！！！」（耳朵彻底竖起来了）「什、什么都没有！我和他什么关系都没有！」

白羽：「我没说什么啊？」

艾露娜：「…………」（意识到自己反应过度了，耳朵缓缓垂下，脸颊以肉眼可见的速度染上淡粉色）

**——这是她第一次对自己"无法用逻辑解释的反应"感到恐惧。**

## 十三、个人线概要

> 注：具体章节内容待后续填充。以下为框架性大纲。

### 半个人线主题：「天才的界限 —— 第一次遇到'无法理解的东西'」

**核心事件**（待定）：某个让艾露娜**第一次承认"有些东西不是靠聪明就能理解的"**的事件。可能是：
- 目睹零为了保护某人而做出的"理性上来说完全不合理的决定"
- 或者她自己产生了无法用逻辑解释的情感反应的前兆

**完成后状态**：对零产生模糊的好感 → 开始出现"奇怪的感觉" → **进入回避模式**

### 个人线主题：「从遥望星空到脚踏实地 —— 学者学会爱人」

**核心冲突**：艾露娜一生都在追求"理解一切"，但**感情是她唯一无法用逻辑解析的东西**。个人线的主题是她学会**接受"不需要理解也可以珍视"**这件事。

**可能的走向**：
- 从"躲着零"到"不得不面对"
- 从"用研究的态度对待感情"到"承认这就是感情"
- 从"天才的孤独"到"有人陪伴的温暖"
- 最终：她可能不会成为"最懂爱的人"，但会成为**"最认真地去学爱的人"**
