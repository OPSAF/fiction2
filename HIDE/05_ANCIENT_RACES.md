# 第一篇：上古三族起源（约1万年前）

## 一、起源概述

### 1.1 "星之降临"——三族的首次出现

约**一万年前**（星暦前约9700年），这个世界迎来了三个"非自然诞生"的种族：

- **天使族（Angels）**：从天而降·光翼银发·如神明般降临大地
- **龙族（Dragons）**：自地底深处苏醒·龙形威压震慑万物
- **海族（Sea Folk）**：从深海中浮出·鳞片闪烁·携带着未知文明

这三个种族并非由世界原生演化而来——他们的出现方式超出了当时所有生命形式的认知范围。后世将这一事件称为**"星之降临"**（Starfall），并以此作为纪元起点（星暦元年≈约5000年前·以传说中三族建立第一个文明为基准）。

### 1.2 神话时期的定义

**神话时代** = 星暦前约9700年 ～ 星暦前约4700年（约5000年的漫长时期）

这个时期的特点：
- 上古三族是世界的绝对主导者
- 他们拥有的力量远超后世任何种族所能理解
- "神粒子"充盈于世——这是神话时代的根本标志
- 普通生物（后来的兽族祖先等）在三族面前如同蝼蚁

# 第二篇：神粒子（God Particle / 神粒子）

## 一、基本定义

### 1.1 什么是神粒子？

**神粒子**是神话时期充斥于整个世界的一种**超高能粒子**。它不是"魔法"——它比后世的魔法更原始、更本质、更强大。

| 属性 | 描述 |
|------|------|
| 本质 | 世界原生的/或来自更高维度的能量载体 |
| 表现形式 | 高密度能量场·可被直接感知和操控 |
| 与零粒子的关系 | 可能是零粒子的"源"形式（未衰变前的完整态） |
| 后世称呼 | 已失传·仅存于最古老的石板残片中 |

### 1.2 神粒子的威力层级

神粒子的力量级别**远超**现代的零粒子（即魔力粒子）：

```
神粒子威力（神话级）
    │  ├── 单体攻击可摧毁山岳
    │  ├── 可改写局部区域物理法则
    │  ├── 可创造/毁灭小型生态系统
    │  ├── 寿命延长至千年以上
    │  └── 几乎不消耗使用者自身体力
    │
零粒子威力（现代级）
    │  ├── 单体攻击摧毁建筑级别
    │  ├── 仅能在物理法则内操作
    │  ├── 无法创造生命或永久改变环境
    │  ├── 对寿命影响有限（血统相关）
    │  └── 大量使用会导致魔力枯竭
    │
普通人类体能（无魔法）
       └── 日常生活级别
```

### 1.3 为什么被称为"神粒子"

上古三族能够**直接操控**这种粒子——其表现出的能力在普通生物眼中无异于"神迹"：
- 天使族用光治愈不可能治愈的伤
- 龙族一息之间焚城灭野
- 海族召唤滔天巨浪重塑海岸线

这就是为什么后世将他们称为"天使""龙""海神"——并最终演变为宗教崇拜的对象。

## 二、神粒子 vs 零粒子 完整对比表

| 维度 | **神粒子（God Particle）** | **零粒子 / 魔力粒子（Mana Particle）** |
|------|---------------------------|-------------------------------------|
| **来源** | 世界原生 / 更高维度渗透 | 外太空陨石携带（300年前坠落） |
| **威力等级** | 极强（神话级·可改写局部法则） | 较弱（需长期积累·在物理法则内运作） |
| **使用者** | 上古三族（纯血个体） | 所有现代种族（亲和性因种族/血统差异极大） |
| **操控方式** | 直接意志驱动·无需媒介 | 需通过体内激发+与周围魔力场共鸣 |
| **副作用** | 几乎无副作用 | 过度使用导致魔力枯竭·身体负担 |
| **现存状态** | 几乎完全消失·无法被现代人使用 | 扩散全球·工业化开采利用（魔矿/魔晶） |
| **理论本质** | 可能是零粒子的"源"形式（完整态） | 神粒子的"衰变/稀释"产物？或完全不同的另一种粒子？ |
| **浓度变化** | 神话期：全球饱和 → 逐渐消退 → 约星暦前2000年几乎归零 | 300年前随陨石降临 → 浓度逐年上升（工业开采加速） |
| **与观测者的关系** | #1（Observer #1）可能在上古时期接触过神粒子体系 | 现代观测者（#5~#7）面对的是零粒子体系 |

### 关键推论（作者参考）

**神粒子是否就是零粒子的"前身"？**

两种可能性供选择：

**假说A——同源论**：神粒子和零粒子本质上是同一种东西。神粒子是世界原生的高纯度形态；300年前的陨石携带的是同一类粒子的"外太空稀释版"。神话末期神粒子消退不是因为消失而是因为某种原因"转化"了形态。

**假说B——异源论**：神粒子和零粒子是完全不同的两种能量体系。它们恰好都表现为"可被智慧生命操控的粒子"，但底层机制不同。神粒子消退后世界进入了长达数千年的"魔力真空期"，直到300年前零粒子随陨石到来才重新激活了魔法的可能性。

> **建议采用假说A（同源论）**，因为它更好地解释了为什么现代种族的血统仍然对零粒子有亲和性差异——那是上古种族基因中残留的神粒子适配性的"退化表达"。

# 第三篇：上古三族详述

## 一、天使族（Angels）

### 1.1 外貌特征

| 特征 | 详细描述 |
|------|----------|
| 光翼 | 背部生有发光的羽翼·颜色从纯白到淡金不等·收拢时几乎不可见·展开时翼展可达2-3米 |
| 发色 | 银白色为主·少数为淡金或珍珠色·发丝在暗处会发出微弱荧光 |
| 瞳色 | 天蓝色·淡金色·或罕见的纯白色·瞳孔形状略异于常人（更接近竖瞳但更柔和） |
| 体型 | 身高普遍高于人类平均·体型修长匀称·骨骼轻盈但强度极高 |
| 皮肤 | 苍白透明感·阳光下近乎半透明·触摸温度偏低 |

### 1.2 能力特征

- **光/神圣类神粒子能力**：这是天使族的标志性行为
  - 光之治愈：修复肉体损伤·甚至可以再生断肢（需大量神粒子消耗）
  - 圣域结界：展开绝对防御领域·内部不受外界攻击影响
  - 光之审判：聚焦光束进行精确打击·穿透力极强
  - 羽翼飞行：不需要魔法阵辅助的自然飞行能力
- **极高魔力容量**：天使族的体内神粒子容纳量是三族中最高的
- **寿命**：1000年以上（神话期纯血个体可能达到1500-2000年）
- **特殊感知**：对"恶意"和"谎言"有天然的直觉感应（类似读心术但不完全相同）

### 1.3 性格与社会

- 天使族天性**冷静理性**·情感波动较小
- 倾向于**观察者**角色而非统治者——这也是为什么他们在三族争战中相对被动
- 内部有严格的行为准则·类似于"不干涉原则"
- 对其他两族的态度：**中立偏善意**——不主动攻击但也绝不软弱

### 1.4 现状

- **现存数量：<50人**
- 散布世界各地·多数隐姓埋名以人类或其他种族身份生活
- 极少数仍保持接近原始形态（如静·化形隐藏已16年+）
- 帝国官方态度：**崇拜但不认现存个体为真**——教科书中的天使族=完美神话形象·现实中的天使族混血=被当作"退化的冒牌货"

### 1.5 与精灵族的血缘关系

> **核心设定：天使族 ↔ 精灵族 血缘相近**

- 天使族与其他种族（尤其是早期人类部落）通婚后·后代逐渐失去光翼等标志性特征
- 但保留了部分天使族基因表达：
  - 银色/淡金色头发（精灵常见发色）
  - 长耳（天使族原始耳朵的残留进化形态·对高频魔力波段的接收器）
  - 长寿（300-500年·远超人类但不及纯血天使）
  - 风/光属性高适性（天使族光属性能力的退化表达）
  - 天生魔力视觉（天使族特殊感知能力的残留）
- 精灵族自称"天使后代"——**这在基因层面是真实的**
- 但纯血天使族对此通常**不予置评**（既不承认也不否认）

## 二、龙族（Dragons）

### 2.1 外貌特征

| 特征 | 详细描述 |
|------|----------|
| 双形态 | **龙形**：巨型飞龙·身长10-30米不等·鳞甲颜色因个体而异（红/黑/金/蓝/白）·人形：保持部分龙特征（角·鳞片·竖瞳·异常身高） |
| 鳞片 | 人形状态下鳞片分布于颈部·手臂·背部·触感如打磨过的金属 |
| 角 | 头顶生有犄角·形状因亚种而异（分叉角·螺旋角·冠状角等） |
| 瞳色 | 竖瞳·颜色与鳞片对应（红龙=金瞳·黑龙=紫瞳·金龙=琥珀瞳等） |
| 体型 | 人形状态下身高普遍2米以上·肌肉密度极高 |

### 2.2 能力特征

- **超强魔力 + 物理破坏力**：龙族是三族中**战斗力最强**的
  - 龙息（Dragon Breath）：从口中喷出高能粒子流·属性因个体而异（火/冰/毒/光等）
  - 龙鳞防御：鳞甲可抵挡绝大多数物理和魔法攻击
  - 巨型化身：变回龙形后战力呈指数级上升
  - 地脉感知：能感知地下魔矿分布和地质活动
- **寿命**：2000年以上（神话期纯血个体可能达到3000年）
- **肉身强度**：即使不使用神粒子·纯物理层面也远超其他种族

### 2.3 性格与社会

- 龙族天性**骄傲好战**·重视荣誉和实力
- 内部有严格的等级制度·以战斗力和年龄为评判标准
- 三族争战中龙族是最积极的一方——他们享受战争
- 对其他两族的态度：**轻视但尊重强者**——看不起弱者但认可有实力的对手

### 2.4 现状

- **现存数量：<30人**
- 比天使族更加隐秘·多数沉睡于偏远山脉或深海海沟
- 少数以人类身份活跃于世间（但极少暴露真实力量）
- 帝国官方态度：同天使族——**崇拜神话·否认现实**

### 2.5 与魔族的血缘关系

> **核心设定：龙族 ↔ 魔族 血缘相近**

- 龙族与其他种族通婚后的后代：
  - 保留了角（龙角的缩小版·魔族的标志性特征）
  - 保留了竖瞳和异色瞳倾向
  - 保留了对暗/火属性的高适性（龙息能力的退化表达）
  - 寿命200-350年（大幅缩短但仍长于人类）
  - 战斗本能和竞争意识强烈
- 魔族自称"龙的后代"——**基因层面真实**
- 纯血龙族对此的反应：**嗤之以鼻**——"那些长着小角的家伙也配称龙？"
- 这种傲慢态度反而让魔族更加坚定地宣称自己的龙族血统（逆反心理）

## 三、海族（Sea Folk）

### 3.1 外貌特征

| 特征 | 详细描述 |
|------|----------|
| 双形态 | **水生形态**：下半身为鱼尾·全身覆盖鳞片·鳃裂位于颈部两侧·可在深海水压下生存·两栖形态：双腿 humanoid ·保留部分鳞片和鳃裂痕迹·可在陆地长时间活动 |
| 鳞片 | 分布于肩部·上臂·腰部·小腿·颜色多样（蓝绿·银灰·珍珠色·彩虹色） |
| 发色 | 多为蓝绿色系·湿润时有金属光泽·水中会飘动如水草 |
| 瞳色 | 海蓝色·翠绿色·或罕见的琥珀色·瞳孔可自适应光线强弱 |
| 皮肤 | 触感微凉滑腻·在水中会产生微量生物荧光 |

### 3.2 能力特征

- **水系神粒子能力**：
  - 深海召唤：召唤和控制巨量水体·可制造海啸级水灾
  - 水域感知：感知水中一切生命体的位置和状态
  - 压力适应：在极端水压下生存·免疫大部分物理挤压伤害
  - 治愈之泉：用水加速伤口愈合（不同于天使的光治愈·更偏向"促进再生"而非"奇迹修复"）
- **寿命**：800年以上（神话期纯血个体可能达到1200年）
- **文明程度**：三族中海族的文明发展最为独特——他们在深海建立了城市和网络·其科技树与陆生文明完全不同

### 3.3 性格与社会

- 海族天性**温和神秘**·情绪内敛
- 重视群体和家庭·内部凝聚力极强
- 三族争战中海族是最**被动**的一方——他们倾向于回避冲突
- 对海洋以外的事物天然好奇但又谨慎
- 创造了独特的深海文字和记录系统（至今未被完全破译）

### 3.4 现状

- **现存数量：约100人**（三族中最多）
- 主要分布在深海区域·少数沿海居住
- 与人类的接触比其他两族更多（地理邻近性）
- 帝国官方态度：同上——**崇拜神话·否认现实**

### 3.5 与人类的血缘关系

> **核心设定：海族 ↔ 人类 血缘相近**

- 海族与其他种族通婚后的后代：
  - 失去了鱼尾和鳃·变成完全陆生形态
  - 鳞片退化成皮肤上的微小痕迹（部分人类在特定光线下可见淡淡光泽）
  - 寿命缩短至70-80年（最大幅度的退化）
  - 保留了水的适应性（人类对水域的亲和力高于兽族）
  - 保留了学习和适应能力（海族的好奇心和学习能力的残留）
  - 零粒子适性范围最广（S~F均有·不像其他种族那样集中在特定属性）
- 人类自称"海族后代"——**基因层面真实**
- 但纯血海族对此反应复杂：**既欣慰又悲伤**——欣慰的是血脉延续·悲伤的是后代已经完全失去了与大海的联系

# 第四篇：种族演化历史

## 一、神话时代的六大历史分期

### 1. 创世期（星暦前约9700年～9500年）

**事件**：三族首次出现在世界上

- 天使族从天空降临·龙族从地底苏醒·海族从深海浮出
- 三族初次相遇·彼此试探·确认对方的存在
- 各自探索新世界·划分初步的活动范围：
  - 天使族→高山和天空
  - 龙族→大陆腹地和火山地带
  - 海族→海洋和沿海区域
- 此时世界上的其他生物（普通动物·早期智慧生物的祖先）在三族面前如同蝼蚁
- **神粒子全球饱和**——三族的力量处于巅峰状态

### 2. 神战期（星暦前约9500年～8000年）

**事件**：三族互相争战·死伤惨重

**起因**：资源和领土争端——虽然世界很大·但三族都是扩张性极强的种族

**主要冲突**：
- **天使 vs 龙**：理念冲突·天使主张"秩序与不干涉"·龙主张"力量即正义"。这是持续时间最长·规模最大的冲突。
- **龙 vs 海**：领地冲突·龙族向海岸扩张·侵犯了海族的活动范围
- **天使 vs 海**：最少正面冲突·多为间接对抗（争夺中立地带的影响力）

**结果**：
- 三族均损失惨重·人口锐减（估计各损失30%-50%的人口）
- 大地满目疮痍·多处地形被永久改变（某些现在的"天然奇观"实际上是神战的遗迹）
- 神粒子因过度消耗开始出现**区域性枯竭**的迹象

### 3. 分治期（星暦前约8000年～7000年）

**事件**：三族决定停战·划分领土和平共处

**关键转折点**：某次大规模战役后的"三方会谈"——具体细节已不可考·但结果是划定了明确的势力范围：

| 区域 | 主导种族 | 说明 |
|------|----------|------|
| 天空和高山 | 天使族 | 绝对控制区 |
| 大陆腹地和火山 | 龙族 | 绝对控制区 |
| 海洋和沿海 | 海族 | 绝对控制区 |
| 平原和丘陵 | 中立地带 | 三族共同管理·后来成为各族混居区 |

**意义**：这是世界历史上第一次"国际条约"——标志着从野蛮走向文明的转折点。

### 4. 混血期（星暦前约7000年～5000年）

**事件**：和平后三族开始杂交繁衍→现代各族诞生

**背景**：
- 长期和平导致三族之间的隔阂逐渐消除
- 边境地区和中立地带的三族成员开始通婚
- 第一代混血儿诞生——他们的外观和能力介于父母双方之间

**演化路径**：

```
天使族 ──┬── ×人类部落祖先 ──→ 半天使混血 ──(多代演化)──→ 精灵族
          │                                              (银发·长耳·长寿·风/光属性)
          │
          └── ×龙族混血 ──→ 稀有种群（极少量·多早夭或不育）

龙族   ──┬── ×人类部落祖先 ──→ 半龙混血 ──(多代演化)──→ 魔族
          │                                              (角·竖瞳·暗/火属性)
          │
          └── ×海族混血 ──→ 稀有种群（极少量）

海族   ──┬── ×人类部落祖先 ──→ 半海混血 ──(多代演化)──→ 人类
          │                                              (广泛适性·短寿·学习能力)
          │
          └── ×其他混血 ──→ 离散诸族的部分祖先？
```

**关键规律**：
- 混血后代的能力总是**弱于**纯血亲代·但**强于**无血统的普通生物
- 经过数代演化后·混血后代逐渐形成了稳定的遗传特征→成为新的"种族"
- 每一代混血都意味着神粒子亲和性的进一步**衰减**
- 这解释了为什么现代种族的魔法能力远不如上古种族

### 5. 神隐期（星暦前约5000年～星暦元年）

**事件**：神粒子消退→纯血上古种族消失/隐退

**现象**：
- 全球神粒子浓度持续下降·从"饱和"降至"稀薄"再到"几乎为零"
- 纯血上古种族的力量随之大幅减弱
- 无法再使用曾经司空见惯的神粒子能力
- 混血后代（现代各族祖先）受影响较小——因为他们从一开始就没有完整的神粒子操控能力

**纯血上古种族的选择**：
- **一部分选择了"沉睡"**：进入深度休眠·等待神粒子可能的重返（龙族居多）
- **一部分选择了"隐匿"**：化形为其他种族的外貌·混入人群中生活（天使族和海族居多）
- **极少数选择了"离去"**：离开了这个世界·去向不明（各族均有少量）
- **没有纯血上古种族选择了"抗争"**——他们都明白神粒子的消退是不可逆的

**时间节点**：约星暦前4700年·神粒子浓度降至临界值以下·上古时代正式终结。此后约200年为过渡期·直到星暦元年（约5000年前）·现代各族建立的早期文明标志着一个全新时代的开始。

### 6. 零粒子降临（星暦前约300年·即约320年前）

**事件**：外太空陨石撞击→零粒子（魔力粒子）扩散全球→新时代开始

- 一颗携带大量零粒子的大型陨石坠落在北方大陆（具体位置不详·可能与サラ出身的北方山脉有关）
- 陨石撞击造成巨大的生态破坏·但同时释放的零粒子重新激活了世界的"魔力潜能"
- 原本无法使用魔法的混血后代们突然发现自己可以使用一种**新的力量**
- 这种力量比神粒子弱得多·但在神粒子消失数千年后·已经是"神迹"级别的存在
- **工业革命的基础**：约150年后·人类发现零粒子可以富集在矿物中（魔矿）→能源革命开始

# 第五篇：现代种族的零粒子亲和性

## 一、亲和性原理

### 1.1 为什么不同种族的魔法能力不同？

**核心答案：基因记忆**

上古种族的基因中编码了对神粒子的**特异性亲和模式**。当这些基因通过杂交传递给后代时·亲和模式也被继承了下来——但经过了多次"稀释"和"变异"。

当300年前零粒子降临时·不同种族体内的**残余亲和模式**决定了他们对零粒子的响应方式：

| 种族 | 上古祖先 | 残留亲和模式 | 零粒子表现 |
|------|----------|-------------|-----------|
| 精灵族 | 天使族 | 光/高频波段敏感 | 风/光属性极高适性·天生魔力视觉 |
| 魔族 | 龙族 | 暗/火/强力输出敏感 | 暗/火属性极高适性·战斗本能 |
| 人类 | 海族 | 广谱低敏（海族原本就不是专精型） | 全属性S-F分布·学习能力和适应性最强 |
| 兽族 | **无** | 无上古种族基因残留 | 魔法适性普遍低下·靠体能弥补 |
| 离散诸族 | 部分可能有微量混血 | 不确定/极微弱 | 各异·但整体低于三大正统种族 |

### 1.2 血统性与学派分化

**同一血统或种族的人使用的魔法比较相近**——这不是巧合·而是基因决定的：

- 天宮家（人类·但可能有远古微量龙族混血？）：火属性世代相传
- 水無瀬家（人类·海族血统表达较强）：水属性世代相传
- 精灵王室家族：光/风属性高度集中
- 魔族古老家族：暗属性几乎垄断

这解释了：
- 为什么会有**属性家族**（某个家族世世代代都是同一属性）
- 为什么**魔法学院**会有**学派分化**（学生按属性分组·形成不同的研究传统）
- 为什么**跨属性婚姻**有时会被反对（"会稀释血统"的担忧——虽然科学上不一定准确·但社会观念如此）

### 1.3 小雪的特殊案例

**小雪的天使血统 = 远古神粒子亲和性残留**

小雪的情况是一个完美的例证：

| 特征 | 来源解释 |
|------|----------|
| 冰属性（C级纸面/A级实际） | 天使族光属性的"折射"表达——光→冰是频率降低的结果 |
| 银白发 | 天使族银发基因的直接表达（半血·不完全隐性） |
| 冰蓝瞳 | 天使族天蓝/白色瞳色的稀释版 |
| 左手腕冰晶痣 | 天使族印记（光翼根部能量节点的残留） |
| 月光下头发泛荧光 | 天使族光的微弱残留 |
| 魔力A级实际力量 | 天使族天然高魔力量容量的遗传（即使只有半血） |

> **深层含义**：小雪不只是"有天赋"——她体内沉睡着一段**神话时代的记忆**（以基因编码的形式）。如果神粒子有一天重返世界（或某种等价物出现）·小雪可能是唯一一个能够重新连接那段力量的现代人类。

## 二、避免的信息泄露

- ❌ 不要在前中期明确说明"神粒子就是零粒子的前身"——留悬念
- ❌ 不要让角色轻易相信"某某是纯血上古种族"——世界观中主流观点是不信的
- ❌ 不要把上古种族描写得过于强大以至于现代角色毫无意义——强调神粒子消退后他们也变得脆弱
- ✅ 通过侧面描写暗示（古籍残片·老人的醉话·无法解释的现象）
- ✅ 让读者自己拼凑真相——比直接告诉更有满足感
