## 1. 双轨制等级总览

这个世界存在**两套完全独立的等级/评级系统**：

```
┌─────────────────────────────────────────────────────┐
│                  魔法能力者分类                       │
├──────────────────────┬──────────────────────────────┤
│    非觉醒者（普通）   │       觉醒者（稀有）          │
│                      │                              │
│  F → E → D → C → B → A │ R → SR → UR → PR → HR → EX  │
│  (NPC)  (普通角色)     │ (限定/SSR角色)               │
│                      │                              │
│  标准化魔法·道具依赖   │ 个人化独特能力·原生发动        │
│  可教授·可批量培养     │ 不可复制·高度个性化           │
│  占人口~99.5%         │ 占人口~0.5%                   │
└──────────────────────┴──────────────────────────────┘
```

**二游类比**：

| 二游概念 | 这个世界的对应 |
|---------|--------------|
| **F级** | NPC / 无法编入队伍的背景角色 |
| **E ~ A级** | 普通卡牌（N/R）——可以通过游戏进程获得和养成的常规角色 |
| **R级（初始觉醒）** | SSR/限定卡牌——稀有、独特、每个都有专属机制 |
| **R → SR → UR ... → EX** | 卡牌"升阶/突破"系统——觉醒者的成长路径 |
| **觉醒认证** | 卡牌"解锁/觉醒"仪式 |
| **觉醒者计划/特殊培训** | 卡牌"专武/专属装备/技能升级" |
| **非觉醒者 = 觉醒者辅助** | 辅助位/后勤位 —— 主C是觉醒者，其他人打配合 |

## 2. 第一轨道：非觉醒者等级（F ~ A）

> 这套等级代表的是**非觉醒者的魔法实力水平**。类似二游中"普通角色"的品质分级。

| 等级 | 名称 | 人口占比 | 二游类比 | 能力描述 | 社会定位 |
|------|------|---------|---------|---------|---------|
| **F** | 「無階」（无阶） | ~60% | NPC / 不可编入 | 无任何可检测的魔法能力；MCCs表达为零 | 底层劳动者；社会边缘人 |
| **E** | 「初識」（初识） | ~18% | N卡（普通） | 能勉强感知CaP场；集中状态下可产生极微弱效应（如让烛火晃动） | 刚展现魔法天赋的普通人 |
| **D** | 「基礎」（基础） | ~12% | R卡（常见） | 稳定的基础道具操作能力；能熟练使用标准魔法道具（魔杖/药水等） | 最常见的"魔法使用者"——靠道具干活的人 |
| **C** | 「熟練」（熟练） | ~5% | HR卡（优良） | 精准的道具操控+一定的理论素养；可在特定领域独立工作 | 技术工人/基层军官/专业匠人 |
| **B** | 「專業」（专业） | ~3% | SR卡（精锐） | 高效的多道具协同操作+战术级应用能力；可带队执行任务 | 中层管理者/精英战士/资深研究者 |
| **A** | 「宗師」（宗师） | ~1.5% | **非觉醒者天花板** | 道具使用达到艺术境界；对魔法物质的理解远超常人；一人可抵一支小队 | 传奇工匠/学术权威/战略顾问 |

**关键规则**：**A级是非觉醒者的绝对顶点**。无论一个非觉醒者多么努力、经验多么丰富，他的等级**不可能超过A级**。要突破到觉醒者轨道，必须经历"觉醒"——这是一个质的飞跃，不是量的积累。

**非觉醒者的战斗方式**：
- **完全依赖魔法道具和装备**
- 自身不产生原生魔法效应
- 战斗力 = 道具品质 × 操作熟练度 × 战术素养
- A级非觉醒者用顶级道具可以战胜低阶觉醒者（但面对同阶觉醒者仍处劣势）
- **这就是为什么零（F级）在实战中差距不大**——大家都是靠道具打，无非零少了一个"紧急情况下用原生魔法应对"的手段而已

## 3. 第二轨道：觉醒者等级（R ~ EX）

> 这套等级代表的是**觉醒者的成长阶段**。类似二游中"SSR限定角色"从初始状态到完全体的升阶过程。

### 3.1 觉醒的本质

**觉醒 = 体内MCCs（魔导细胞）发生质变**。

非觉醒者的MCCs只能进行基础的魔法物质代谢（正常生化过程）。但觉醒者的MCCs发生了某种**结构性变异/进化**：

```
非觉醒者MCCs:                    觉醒者MCCs:
┌──────────────┐                ┌──────────────┐
│ 正常代谢通路  │                │ ★突变通路★    │
│ (标准酶系统)  │    ──觉醒─→    │ (新型酶系统)  │
│              │                │              │
│ 输入: CaP    │                │ 输入: CaP    │
│ 输出: 标准效应│                │ 输出: **独特  │
│ (喷火/造冰等)│                │      个人化   │
│              │                │      效应**   │
│ 可教·可复制  │                │ 不可教·不可复 │
└──────────────┘                └──────────────┘
```

**觉醒的发生条件**（世界尚未完全理解）：
- 基因层面：必须拥有特定的罕见基因组合（"觉醒基因序列"）
- 触发事件：通常是**极端情绪体验**或**生死临界状态**——日常训练无法触发
- 年龄窗口：大多数觉醒发生在12~20岁之间（青春期前后），25岁后几乎不再发生
- 不可强制：外部无法强迫某人觉醒（虽然有些组织在研究"人工诱导觉醒"）

### 3.2 觉醒者六阶体系

| 等级 | 名称 | 二游类比 | 描述 | 典型表现 |
|------|------|---------|------|---------|
| **R** | 「覚醒」（觉醒） | 初始SSR / 刚抽到的限定卡 | **刚完成觉醒**；能力原始、粗糙、不稳定 | 能勉强使用个人化能力但控制力差；容易"过载"反噬 |
| **SR** | 「超覚」（超觉醒） | 突破1次 / +1阶 | 能力初步稳定；开始形成个人风格 | 可以可靠地在实战中使用能力；有了初步的"必杀技"雏形 |
| **UR** | 「真覚」（真觉醒） | 突破2次 / 满破预备 | 能力高度成熟；机制明确 | 个人能力的"完整形态"；可以在不借助道具的情况下改变局部战局 |
| **PR** | 「極覚」（极觉醒） | 突破3次 / 完全体 | 能力的艺术级运用；开始触及"规则"边缘 | 部分效应已经超出常规物理/魔法解释范畴；一人可敌一支军队 |
| **HR** | 「聖覚」（圣觉醒） | 超越满破 / 传说级 | 存在本身即是一种"现象"；其能力成为传说 | 各国争相招揽的战略威慑力量；名字被写入教科书 |
| **EX** | 「神覚」（神觉醒） | ?? / ？？？ | **超越当前认知体系的终极形态** | 能力本身即是自然法则级别的存在；历史上屈指可数 |

**升阶机制**（觉醒者如何从R升到SR、UR...）：

| 升阶方式 | 说明 | 类比二游 |
|---------|------|---------|
| **实战磨炼** | 在真实战斗中使用能力→逐渐理解和掌控→自然升阶 | "熟练度满了自动突破" |
| **特殊培训** | 觉醒者计划的系统性训练→加速升阶 | "使用专属素材突破" |
| **关键事件** | 某些重大情感/生存体验可能触发跳跃式升阶 | "剧情触发的觉醒任务" |
| **未知因素** | 有些觉醒者的升阶原因连他们自己都不清楚 | "随机掉落？" |

**重要约束**：
- **不是所有觉醒者都能升到EX**——大部分人终其一生停留在R~UR
- 升阶没有固定时间表——有人3年从R到UR，有人20年还在SR
- **降阶可能**：重伤/精神创伤/某些特殊攻击可能导致觉醒者降阶（甚至退化为非觉醒者——极罕见但有可能）

### 3.3 星導学園的觉醒者

**零入学的这一届（高等部一年级），已确认的觉醒者只有数人：**

| 角色 | 入学时等级 | 备注 |
|------|-----------|------|
| **艾露娜** | **R**（实际潜力远超此评估） | 初中时已完成觉醒（作为"预备役"记录）；入学后正式登记为觉醒者 |
| **朱音** | **R** | 13岁早熟觉醒（极少数的早例）；因家族关系提前完成觉醒认证 |
| **真夜** | **R**（**隐藏**） | 实际早已觉醒但**未被发现**（本人不知/学校未检测到）；入学时以普通非觉醒者身份就读 |

> **⚠ v4.0 统一修正**：以上三名角色的等级均已统一为 **R级觉醒者**。不再使用旧版写法（如"B+級專業階"、"A级宗師階"、"B+級"等）。

**全校觉醒者总数**：约50余人（含各年级）。在整个东和帝国范围内，登记在册的觉醒者大约数百人。

**预备役觉醒者**：
- 定义：已经觉醒但**年龄不足**或**尚未完成正式认证**的年轻觉醒者
- 通常为初中部学生（12~15岁）
- 不需要上战场；接受基础保护和观察
- **学妹**属于此类——初中部学生，已是觉醒者（预备役），但年龄不够无法正式编入
- 朱音和艾露娜在初中时也经历过"预备役"阶段

### 3.4 觉醒者 vs 非觉醒者的关系

**在这个学校的定位**：

| 身份 | 定位 | 培养目标 |
|------|------|---------|
| **觉醒者** | **主战力（DPS/Carry）** | 上战场对抗虚空种；接受特殊培训 |
| **非觉醒者（D~A级）** | **辅助/后勤/支援** | 为觉醒者提供道具支援、情报分析、后勤保障、战术协调 |
| **非觉醒者（F级）** | **待观察/后备** | 大多数会被淘汰或转岗；少数有特殊价值者（如零的学术能力）被保留 |

**教学方式**：
- 觉醒者和非觉醒者**在同一个班级上课**（混合编班）
- 必修课完全相同（魔法基础理论等）
- **选修课**可能不同（觉醒者可选"能力精控"类课程；非觉醒者选"道具制作/战术分析"等）
- 觉醒者额外有**特殊培训**（通常在课后或周末，内容保密程度较高）
- **一视同仁**——学校不会公开歧视非觉醒者（毕竟大多数人都是非觉醒者）

**零的特殊性**：
- F级（無階）= 连最基本的魔法道具操作都做不好
- 在一个"大家都能用道具打仗"的环境里，F级确实**很显眼地弱**
- 但由于实战主要靠道具品质而非个人魔法水平，所以**差距没大到离谱**
- 零真正的问题是：**没有魔法抗性** + **不能使用原生魔法应急** + **被当成废物/猪**

## 4. 觉醒认证与发现机制

### 4.1 正式流程（已知觉醒者）

```
疑似觉醒迹象被发现
  ↓ （异常CaP波动/能力自发显现/体检异常）
地方上报 → 帝国魔法研究学会备案
  ↓
派遣调查员确认（检测MCCs是否发生质变）
  ↓ 确认
觉醒认证测试（压力测试/实战模拟/?）
  ↓ 通过
正式登记为觉醒者 + 分配初始等级（通常为R）
  ↓
按年龄分流：
  ├─ 未满15岁 → 编入"预备役"（初中部观察保护）
  └─ 已满15岁 → 编入觉醒者名册 + 分配至最近的高等部
      ↓
  可选：加入觉醒者计划（星導学園等）
```

### 4.2 未发现/隐藏的觉醒者

**关键设定：不是所有觉醒都会被发现。**

| 隐藏原因 | 案例 | 发现契机 |
|---------|------|---------|
| **觉醒者自己不知道** | 真夜？ | 某次极端情况下的无意触发 |
| **觉醒症状太微弱** | 某些低强度觉醒者 | 精密仪器偶然检测到异常 |
| **刻意隐瞒** | 有人不想被征召 | 无法永久隐瞒（迟早会暴露） |
| **检测技术盲区** | 某些特殊类型的觉醒 | 新型检测方法开发 |
| **上报链条断裂** | 边远地区觉醒者 | 巡查/偶然发现 |

**真夜的情况**：
- 她的觉醒类型可能与"暗影/潜行"相关——这种觉醒的症状极其隐蔽
- 她自己可能隐约感觉到"有些不对劲"，但不确定那是什么
- 家族（暗杀家族）可能有**故意不上报**的传统——保持隐蔽是他们的生存策略
- 学校的常规入学检测**没有发现**她是觉醒者
- **直到某个关键时刻**（可能是暗杀任务中的极限状态？），她的觉醒能力才会真正显露

## 5. 学历资格体系（旧「階」位）

> **重要区分**：之前使用的「階」位名称（啓蒙階/基礎階/熟練階等）**并未废弃**——它们现在代表的是**学历/学术资格**，而非个人实力等级。
>
> **⚠ 关键原则：学历资格 ≠ 战斗力。两者相互独立。**

| 旧名称 | 新定位 | 对应现实 | 获得方式 | ≈战斗力参考（仅供参考，不等价） |
|--------|-------|---------|---------|-------------------------------|
| 啓蒙階 | 初等部毕业资格 | 小学毕业(12岁) | 自动获得（义务教育完成） | F ~ E |
| 基礎階 | 中等部毕业资格 | 初中毕业(15岁) | 自动获得（义务教育完成）= **社会就业基准线** | D |
| 熟練階 | 高等部前期課程修了（3年） | 高中/大专(18岁) | 修满学分+GPA达标 | C ~ C+ |
| 專業階 | 高等部後期課程修了（5年） | 学士/本科(20岁) | 修满学分+毕业论文 | B- ~ B+ |
| 宗師階 | 研究生課程修了 | 硕士(22-24岁) | 研究成果+论文答辩 | A- ~ A+ |
| 伝説階 | 最高学术荣誉 | 博士/院士 | 开创性贡献+学会认定 | S ~ S+ |

**简单来说**：
- **F~A / R~EX** = 你的**战斗力/能力值**（二游卡牌的"星级/稀有度"）
- **啓蒙階~伝説階** = 你的**学历/学位**（二游卡牌的"等级上限/突破等级"）
- 两者**相互独立**——一个A級戦闘力的人可能只有基礎階学历；一个宗師階学历的人可能只有D級戦闘力

## 6. 教育体系详解

### 6.1 教育制度总览

```
初等部（6年·6-12岁）          ← 义务教育
    ↓
中等部（3年·12-15岁）         ← 义务教育 · D级=毕业基准 · 16岁可合法工作
    ↓
高等部（5年制·15-20岁）       ← 进阶/精英教育 · 选科制 · 无高考
    ├─ 前3年（15-18岁）        ← 修了 = "高等部前期課程修了証"（≈高中文凭）
    └─ 后2年（18-20岁）        ← 修了 = "高等部後期課程修了証"（≈本科/学士）
         ↓
研究生课程（可选·20-24岁）     ← ≈硕士/博士路径
```

**核心特征**：
- **无高考（统一入学选拔考试）**：几乎所有想继续深造的学生都可以直接从高等部前3年升入后2年
- **高中+大学融合制**：5年一贯制，前3年≈高中+本科低年级，后2年≈本科高年级+初步研究
- **选科制（Elective System）**：学生根据兴趣和能力选择不同的选修方向，不再按"理論科/実践科"分班
- **魔法基础理论=必修课**：无论选什么方向，所有人都必须学习魔法基础理论——纯学术研究不被歧视
- **D级(基礎階)=中等部毕业基准 = 社会就业基准线**
- **16岁合法工作年龄**

### 6.2 初等部（6年·6-12岁）— 义务教育

全国の子供が通う。読み書き·算数·**魔法基础理论（必修）**·基础安全规范·体育·道徳。

- 魔法の才能がある子は放課後の特別クラスで実技練習（道具使用为主·非个人魔法）
- 入学時に魔力適性筛查（零はここでF級判定）
- 初等部毕业时获得**啓蒙階**资格（自动授予）

**通用魔法教育阶段（义务教育6-15岁）的内容**：

所有孩子从初等部开始学习**通用魔法基础理论**（必修课）。魔法基础理论是这个世界的**核心通识教育**——就像现实世界的数学和语文一样，是每个人必须掌握的基础知识：

- 魔法物质的基本性质（什么是魔矿、魔晶、魔油）
- 基础安全规范（如何安全接触魔法物质）
- 七大魔法属性/类型的基础概念
- 简单的魔法道具使用方法（不是个人魔法，而是工具使用）
- **纯理论研究不再被视为"异端"或受到歧视**——因为魔法基础理论本身就是必修课的一部分

### 6.3 中等部（3年·12-15岁）— 义务教育

- 基礎課程深化 + **選択科目導入の予備段階**
- 魔法基础理论继续必修 + 开始接触简单的属性分类知识
- **D級（基礎階）= 中等部毕业认证标准** = **社会就业基准线**
- 毕业后可选择：
  - 直接就业（**16岁合法工作年龄**）——初中毕业后直接进入社会的年轻人占相当比例
  - 升入高等部（需申请·但门槛不高·大部分申请者都能通过）

### 6.4 高等部（5年制·星導学園等）— ★核心

#### 结构概览

| 阶段 | 年级 | 年龄 | 学位/证书 | 说明 |
|------|------|------|----------|------|
| **前期課程** | 1~3年 | 15-18岁 | 高等部前期課程修了証（≈高中毕业） | 通识教育+选修课+基础研究方法 |
| **後期課程** | 4~5年 | 18-20岁 | 高等部後期課程修了証（≈学士/本科） | 专业深化+研究方向确定+论文 |
| **研究課程**（可选） | 追加2-4年 | 20-24岁 | 研究修了証（≈硕士/博士） | 独立研究+原创贡献+高等级考核 |

#### 选科制（Elective System）

**不再区分「理論科」和「実践科」。所有学生混合编班，通过选修课来差异化发展。**

**必修课（所有人必学）**：
- 魔法基础理论Ⅰ·Ⅱ·Ⅲ（逐年深化）
- 数学（基础→高等）
- 通用语（读写+修辞）
- 世界历史与地理
- 体育（含基础体能训练）
- 安全规范与伦理

**选修课方向示例**：

| 方向类型 | 代表性选修课 | 适合人群 |
|---------|------------|---------|
| **学术/理论研究** | 高等物理·数学分析·魔法物质化学·古代文献解读·量子场理论基础（这个世界还没有这个概念·但可能有类似的前驱理论） | 理论倾向强的人·**零的主攻方向** |
| **实践/战斗应用** | 战术基础·魔杖精控训练·团队协同·野外生存·对抗演练 | 觉醒者·战斗型角色 |
| **道具制作/炼金** | 魔法物质合成·道具结构设计·材料科学·品质鉴定 | 工匠型·技术型 |
| **医疗/治愈** | 生物组织修复·药剂调制·急救技术·长期护理 | ε型（愈者）能力者 |
| **情报/侦查** | 隐蔽行动·信息分析·环境感知·反侦察 | ζ型（感者）能力者 |
| **社会/管理** | 行政实务·外交礼仪·经济基础·法律概论 | 未来从政/经商的人 |

> **主角零的选修倾向**：偏学术/理论研究方向（高等物理·数学分析·魔法物质化学），同时在情报/侦查方向有少量涉猎（因为终端的分析能力天然适合这类工作）。但他**不能选实践/战斗方向的进阶课程**（需要展示实际魔法能力作为先修条件）。

#### 升学机制

- **从前3年到后2年：几乎自动晋升**
  - 不需要类似"高考"的统一选拔考试
  - 前期課程修了时累计GPA达到一定标准即可申请升入後期課程
  - 极少数因成绩太差或违纪无法晋升的情况存在
  - **这意味着：同一个班级的同学会一起待满5年（除非主动退学或转学）**

### 6.5 特待生制度

- 特待科：学費·寮費全免·最難関·エリートコース·国家が宗師階～伝説階人材を囲い込む仕組み
- 特待生的选拔不看家庭背景·只看潜力（包括学术潜力和觉醒潜力两方面）
- **零以学术类特长生身份入学**——虽然魔力=F级/0·但在笔试中展现出的学术能力让招生方认为值得投资

### 6.6 社团活动（Club Activities）

星導学園拥有完整的社团体系，与其他普通学校无异：

| 类型 | 示例 | 说明 |
|------|------|------|
| 学术类 | 物理学研究会·魔法理论研讨社·文学部·棋艺社 | 偏理论研究的学生聚集地 |
| 战斗类 | 武术部·战术研究会·魔杖竞技社 | 觉醒者和实战派学生的主场 |
| 制作类 | 炼金工房·道具改良社·料理部 | 技术型和工匠型学生 |
| 艺术类 | 音乐部·美术部·演剧部 | 文化活动 |
| 服务类 | 学生会·図書委員会·風紀委員会 | 学校运营相关 |
| 其他 | 园艺部·探索部·天文部 | 兴趣爱好 |

### 6.7 少年班（帝都特别英才学院）

- 小雪が通う学校·星導学園の隣
- 飛び級·超優秀児童向け·全国から選抜
- 実質的には帝国の"天才囲い込み"機関
