## 一、线路总览

| 线路 | 类型 | 对应角色 | 核心主题 | 最终BOSS | 特殊结局 |
|------|------|---------|---------|---------|---------|
| **主线** | 身世/真相 | **白羽栞** | 主角身世·地球Phantasm计划·零粒子真相 | ??? | **仅此线揭露全部真相** |
| **支线1：帝国线** | 政治/革命 | **朱音** | 帝国独裁·人权·民主·阶级斗争 | **皇帝** | 朱音线专属结局 |
| **支线2：宗教线** | 信仰/权力 | **艾露娜** | 魔道教·教皇厅·信仰与真理 | **教皇** | 艾露娜线专属结局 |
| **支线3：虚空线** | 生存/战争 | **真夜** | 虚空种入侵·龙族血脉·暗影龙遗产 | **虚空领袖** | 真夜线专属结局 |
| **支线4：古神线** | 起源/哲学 | **小雪** | 上古三神·世界起源·零粒子创造 | **三神（可选战）** | "弑神"结局（道德判断型） |

### 1.1 攻略机制

- 男主**只能攻略一名角色**
- **攻略白羽 = 进入主线** → 揭露地球/Phantasm/穿越的全部真相 → 可能回到原世界
- **攻略其他角色 = 进入对应支线** → 主角留在该世界与该角色在一起 → 不会触发主线真相
- **未攻略的角色**：根据玩家选择可能有"好结局"或"一般结局"，但不会有专属深入剧情
- **路线之间看似孤立，实则互相关联**——不同路线会揭示同一事件的不同侧面

## 二、支线1：帝国线（朱音）

### 2.1 东和帝国的本质

**东和帝国是一个独裁君主制国家。**

| 维度 | 描述 |
|------|------|
| **政体** | 绝对君主制——皇帝拥有立法、司法、军事的最终决定权 |
| **人权状况** | 极度受限。没有言论自由、集会自由、出版自由 |
| **民主力量** | 被系统性压制。进步人士被监视、逮捕、或"意外死亡" |
| **对外目标** | 称霸世界·巩固统治·消灭一切潜在威胁 |

### 2.2 帝国的黑暗面

**这个国家远非表面光鲜：**

| 黑暗现象 | 具体表现 |
|---------|---------|
| **资本主义剥削** | 九条财阀控制经济命脉·工人超时低薪·童工存在 |
| **贵族歧视** | 五大家族（尤其是天宮家）垄断资源和机会·平民上升通道极窄 |
| **兽族歧视** | 亚人种（兽族）被法律性和制度性歧视·居住限制·职业限制 |
| **奴隶交易** | 表面上"劳动斡旋"实质是人口买卖·亚人种为主要受害者 |
| **贫富差距** | 顶层0.01%掌握80%财富·底层60%人口生活在贫困线以下 |
| **言论控制** | 所有出版物需审查·异见声音被消音·秘密警察无处不在 |
| **天宮家的角色** | **旧贵族/军国主义代表**·主张扩军·对外强硬·内部镇压·是帝国黑暗面的核心受益者和维护者 |

### 2.3 觉醒者的军事用途

- 帝国培养觉醒者的**明面目的**：对抗虚空种——这是对外宣传的正当理由
- **实际目的**：
  - 制造超级士兵·巩固军事优势
  - 觉醒者是**不可复制的战略资源**——各国都在争夺
  - 星導学園本质上是帝国的**觉醒者兵工厂**
  - 天宮家通过军方渠道控制大部分觉醒者的去向

### 2.4 帝国线的核心冲突

- **朱音出身天宮分家**——她既是既得利益者，又能看到系统的黑暗面
- 她的内心冲突：**家族忠诚 vs 正义感** · **帝国公民 vs 世界公民**
- 如果走朱音线：主角将深度卷入帝国的政治斗争·可能面临革命或改革的选择

## 三、支线2：宗教/魔法线（艾露娜）

### 3.1 魔道教（Magi-Theism）

> 这个世界的国教/主流宗教。融合了基督教的组织架构和神道教的泛灵论元素。

#### 起源

`
上古"古神教"（东方原始多神教）
    ↓ （约2000年前·魔法大发现时代）
宗教改革运动——从古神教中分裂出来
    ↓
魔道教成立——以"魔法"为核心崇拜对象
    ↓
与皇权结合→皇帝=最高宗教领袖
    ↓ （约500年前）
再次改革——教权与皇权分离
    ↓
教皇厅独立→教皇成为实际上的宗教+魔法双料领袖
`

**类比现实**：类似犹太教→基督教的演化关系。古神教≈犹太教（原始一神/多神），魔道教≈基督教（从母体中分裂、发展出独立的组织体系和神学）。

#### 核心教义（公开版本）

| 教义 | 内容 |
|------|------|
| 魔法起源 | 魔法是"神"赐予人类的礼物（实际上来自零粒子传播） |
| 魔素（零粒子類） | 神之微粒·世界的基石 |
| 觉醒 |神的选中·被赋予"神力"的人 |
| 教皇 | 神在地上的代行者·最强觉醒者 |
| 来世 | 善用魔法者升入"魔天·滥用者堕入"虚渊" |

#### 实际运作（隐藏信息）

| 方面 | 公开宣称 | 实际情况 |
|------|---------|---------|
| 教皇权力来源 | 神圣任命 | **自身是最强觉醒者（HR级?）**·用实力维持权威 |
| 财富来源 | 信徒奉献 | **巨额税收+魔法产业垄断**·教皇厅是帝国最富有的组织之一 |
| 学校目的 | 培养虔诚的魔法人才 | **培养私人军队**·教皇厅暗中建立直属武装 |
| 最终意图 | 服务皇帝·荣耀神名 | **政变推翻皇帝**·建立神权政治 |
| 对古神的态度 | 尊崇为"上古圣人" | **知道部分真相但歪曲利用**·将古神传说作为控制工具 |

### 3.2 教皇厅 vs 皇帝

| | 皇帝（皇室） | 教皇（教皇厅） |
|--|------------|--------------|
| **权力基础** | 传统君权·军队·法律 | 宗教权威·觉醒者·财富 |
| **控制手段** | 行政命令·法律·秘密警察 | 信仰洗脑·信息控制·觉醒者忠诚 |
| **最终目标** | 维持并强化独裁统治 | 推翻皇帝·建立神权国家 |
| **当前关系** | 表面合作·实际暗斗 | 表面效忠·实际准备叛变 |
| **谁更危险？** | 暴力压迫·直接可见 | 精神控制·更难反抗 |

### 3.3 艾露娜线的核心冲突

- 艾露娜是**学术型/理性型**人物——她会自然地接触到魔道教的历史文献和理论矛盾
- 她的波粒二象性论文可能**无意中触及了某些教皇厅不想让人知道的真相**
- 如果走艾露娜线：主角将卷入教皇厅内部的权力斗争·发现宗教背后的谎言·面临"揭发还是沉默"的选择

## 四、支线3：虚空线（真夜）

### 4.1 虚空种（Voidborn）——重新设计

> **设计参考**：犹太教神秘主义（卡巴拉/Kabbalah）+ 克苏鲁式宇宙恐怖（轻度）
> **不再使用**：卡巴拉生命之树对应魔法体系（已移至虚空种设计）

#### 本质（科学语言·隐藏信息）

虚空种是被**高浓度零粒子類污染后发生变异**的生命体——但更深层的真相涉及**卡巴拉式的"逆向创造"**：

`
正常世界:
  神粒子（秩序·形式·光明）
    + 零粒子（物质·能量·媒介）
    = 稳定的物理世界

虚空种诞生:
  零粒子异常凝聚（超过临界阈值）
    → 局部区域"过度实化"
    → 物质疯狂增殖但失去有序结构
    → 类似卡巴拉中"Qliphoth（逆性之树）"的概念
    → 产生的不是正常的生命·而是**生命的扭曲镜像**
`

**卡巴拉式解读（世界语言的模糊认知）**：

这个世界的一些神秘学者（极少数学派）用类似卡巴拉的框架来理解虚空种：

| 卡巴拉概念 | 对应虚空种特征 | 说明 |
|-----------|--------------|------|
| **Sefirot（圣树）** | 正常的世界秩序 | 光明·有序·有层次的结构 |
| **Qliphoth（逆树/壳）** | 虚空种的来源领域 | 混沌·无序·倒转的结构 |
| **Thaumiel（双子神）** | 高阶虚空种的"双核" | 某些强大虚空种有两个"意识核" |
| **Satariel（隐藏之神）** | 隐形/潜行型虚空种 | 无法被常规手段侦测的虚空种 |
| **Gha'agsheblah（撕裂者）** | 破坏型虚空种 | 以撕裂空间裂缝方式出现的先锋种 |
| **A'arab Zaraq（ ravens of death ）** | 群体型虚空种 | 如乌鸦般成群出现的小型个体 |

**虚空种干部体系**（参考卡巴拉的10个逆性球）：

| 干部代号 | 卡巴拉对应 | 特征 | 危险度 |
|---------|-----------|------|--------|
| **Thaumiel** | 第1逆位（双子） | 双意识核·极高智能·可指挥其他虚空种 | ★★★★★ |
| **Ghagiel** | 第2逆位（扰动） | 制造混乱和恐慌·群体失控诱导 | ★★★★☆ |
| **Sathariel** | 第3逆位（隐藏） | 隐形·潜伏·渗透专家 | ★★★★☆ |
| **Gamchicoth** | 第4逆位（吞噬） | 吸收一切物质/能量·不断膨胀 | ★★★★★ |
| **Golachab** | 第5逆位（燃烧） | 高温·放热·引发大规模火灾 | ★★★☆☆ |
| **Tagerion** | 第6逆位（腐烂） | 生物腐蚀·疾病传播·缓慢致死 | ★★★☆☆ |
| **A'arab** | 第7逆位（群鸦） | 数量型·蜂群战术·单个弱但集群致命 | ★★☆☆☆ |
| **Atem** | 第8逆位（空虚） | 创造真空/虚无区域·抹除存在 | ★★★★☆ |
| **Malkuth** | 第9逆位（根基） | 基础型·最常见的虚空种·杂兵 | ★★☆☆☆ |
| **Nehemoth** | 第10逆位（诱惑） | 精神影响·幻觉·精神污染 | ★★★☆☆ |

#### 出现方式

- 主要通过**空间裂缝（Breaches）**出现
- 裂缝位置不固定·但东和帝国境内频率最高（原因不明——可能与古代封印有关？）
- 裂缝大小决定了出现的虚空种数量和等级
- **传统魔法军队难以有效对抗虚空种**——因为：
  - 虚空种的物理性质不稳定（常规武器/魔法经常"穿过"它们）
  - 虚空种可以快速适应攻击模式
  - 只有**觉醒者的原生能力**能对虚空种造成有效伤害
  - **各种族的原生魔法对虚空种的克制效果不同**（见下文）

### 4.2 龙族·魔族·暗影龙的血脉

#### 上古历史（隐藏真相）

**万年前·"第一次虚空入侵"**：

| 种族 | 角色 | 后代 |
|------|------|------|
| **天使族**（天使神子民） | 远程支援·光属性净化 | → 精灵 |
| **龙族**（龙神子民）| **主力地面战·正面抗衡** | → **魔族**（含多个分支） |
| **海族**（海神子民） | 海上封锁·切断补给 | → 海洋种族（分散） |

**龙族的牺牲最大**——他们是抵抗虚空种的**主力军**，伤亡最惨重。

**龙族内部分化**：

`
万年前龙族:
├─ 正统龙族（主力战斗部队）
│   ├─ 光龙分支 → 后代不详（可能灭绝或进化为未知形态）
│   └─ 炎龙分支 → 现代魔族主体
│
├─ 特种作战部队:
│   ├─ 暗影龙（游击·情报·暗杀）→ **真夜的祖先**
│   └─ 冰龙/风龙等 → 其他魔族分支
│
└─ 后方支援:
    └─ 治愈龙/守护龙等 → 现代魔族中的温和派
`

**暗影龙的悲剧**：
- 暗影龙在战争中承担了**最危险的任务**——深入敌方腹地·破坏虚空种核心
- 战争结束后·龙族内部开始**勾心斗角**
- 暗影龙因长期在"阴暗面"作战·被其他龙族分支**污名化为"被污染者"**
- 遭受系统性迫害·排挤·最终被迫逃离
- 经过万年演化·成为现代的**暗杀家族**（真夜的家族）

#### 天生克制虚空种的血脉

**所有龙族后代（包括现代魔族）都保留了一种特殊血脉能力：**

| 能力 | 效果 | 原因（科学猜测） |
|------|------|---------------|
| **虚空感知** | 能感知远处虚空种的存在 | MCCs对异常CaP场的高度敏感性 |
| **虚空抗性** | 对虚空种的精神/腐蚀攻击有较强抵抗力 | 龙族基因中的某种保护序列 |
| **虚空伤害加成** | 对虚空种造成的伤害高于其他种族 | 可能是针对零粒子异常结构的"反制酶" |
| **暗影龙特有** | **潜行/隐蔽能力**在对抗虚空种时特别有效 | 暗影龙原本就是为特种作战而进化的 |

**各种族原生魔法差异**（对抗虚空种时）：

| 种族 | 原生魔法特点 | 对虚空种效果 |
|------|------------|-------------|
| **人类** | 多样但不极端 | 中等——依赖道具时效果一般 |
| **精灵** | 精·空间·光系 | **优秀**——空间操控可以切割虚空种结构 |
| **魔族（龙族后代）** | 暗·火·血系 | **最优**——天生克制+血脉加成 |
| **觉醒者（任何种族）** | 个人化独特能力 | **变化极大**——取决于具体能力类型 |

### 4.3 真夜线的核心冲突

- 真夜作为**暗影龙后代**·她的觉醒能力很可能与暗影/潜行/暗杀相关
- 她**早已觉醒但未被发现的设定**——暗影龙的能力天然隐蔽
- 她的家族（暗杀家族）可能在**私下继续对抗虚空种**（作为雇佣兵/自由佣兵）
- 如果走真夜线：主角将通过暗杀任务认识真夜·发现她的觉醒身份和龙族血脉·卷入虚空种战争的深层秘密

## 五、支线4：古神线（小雪）

### 5.1 上古三神（Ancient Trinity）

> **设计参考**：克苏鲁神话（轻度·不过度恶心）+ 自身原创
> **关键原则**：三神**未必是传统意义的"善神"**——他们有自己的立场和考量·玩家需要自行判断

#### 三神概览

| | **天使神** (Angel God) | **龙神** (Dragon God) | **海神** (Sea God) |
|--|---------------------|---------------------|-------------------|

#### 三神做过的事（传说版 vs 真相版）

| 事件 | 宗教/传说版本 | 可能的真相（供剧情探索） |
|------|-------------|---------------------|
| **世界创造** | 三神携手创造世界 | 可能只是"改造"了已有世界·或发现了这个世界 |
| **零粒子创造** | "神赐予的礼物" | 三神预感到自身力量衰减·为后代创造的"替代能源" |
| **虚空种封印** | "三神合力封印了恶魔" | 确有其事·但封印正在减弱·三神已无力维护 |
| **三神离去** | "升入了更高的维度" | 可能是**耗尽了神粒子**·可能是在对抗中受损·可能是**死了** |
| **陨石撞击** | "神的惩罚/考验" | **绝非三神本意**——零粒子提前扩散的意外事故 |

#### 克苏鲁式元素（轻度使用）

| 元素 | 表现 | 使用程度 |
|------|------|---------|
| **不可名状的存在感** | 三神的真实形态超出人类认知极限 | 轻——更多是"敬畏"而非"疯狂" |
| **古老的时间尺度** | 三神的存在以万年为单位·人类历史对他们如一瞬间 | 中——营造"渺小感" |
| **知识的代价** | 了解过多关于三神/世界真相可能导致精神负担 | 轻——表现为噩梦·偏执·而非发疯 |
| ** ambiguity（模糊性）** | 三神的"善恶"立场故意不明确 | 重——这是核心设计·留给玩家判断 |
| ** cosmic indifference（宇宙冷漠）** | 三神可能不在乎人类的死活·他们在乎的是更大的东西 | 中——推动剧情的关键动力 |

**明确不适用的克苏鲁元素**：
- ❌ 过度的肉体恐怖/身体畸形描写
- ❌ "看到就疯"的设定
- ❌ 无意义的混沌/随机性
- ❌ 人类完全无足轻重的虚无主义

### 5.2 零粒子传播的真相

`
时间线（推测）:

万年前: 三神带领种族击败虚空种·预感未来危机
        → 自身神粒子浓度开始降低（原因不明）
        → 决定创造"零粒子"作为替代能源
        → 零粒子在三神的控制下缓慢凝聚（预计需要数千年）

计划: 零粒子成熟后逐步释放 → 世界平稳过渡到"魔法文明"
      → 类似西幻世界观·有序发展

现实: ★ 意外发生了 ★
      → 大约数百年前·一颗富含高浓度凝聚态零粒子的陨石撞击了这个世界
      → 零粒子被**突然·无控制地**传播到全球
      → "迟来"的魔法导致世界走上了不同的演化路径
      → 不是有序的西幻魔法世界·而是"科技+魔法+宗教"混杂的独特形态

三神的反应: ???
      - 他们是否知道这次意外？
      - 他们是否还在干预？
      - 还是他们已经彻底消失/无力干预？
      → **这些是古神线要探索的问题**
`

### 5.3 小雪的特殊性

- 小雪是目前已知唯一能使用**神粒子類**的人（详见91_LEGACY_MAGIC_SYSTEM.md第8节）
- 这意味着她的身体可能保留了某种**与三神相关的古老遗传特征**
- 她的"后期觉醒"可能不只是普通的R级觉醒——而是**触发了某种更深层的、与古神相关的东西**
- **古神线 = 小雪的个人故事 + 世界起源真相 + 三神的真正面目**

### 5.4 "弑神"结局（古神线专属）

> **仅存在于古神线（小雪线）的特殊结局。**
> **参考**: 博德之门3的道德判断系统——没有标准答案·由玩家自行评判。

**前提条件**：走到古神线深层剧情·发现三神仍以某种形式存在·且他们的存在对人类构成某种威胁（或至少是限制）。

**选择**：玩家可以选择是否与三神对抗。

| 选择 | 结果 | 道德定位 |
|------|------|---------|
| **弑神** | 击败/杀死/封印三神残余 | 人类获得完全自由·但也失去了保护·未来 uncertain |
| **不弑神** | 与三神达成某种妥协/共存 | 人类继续在三神的阴影下生存·但有庇护 |
| **第三条路** | 成为新的"神"的继承者 | 极端路线·代价巨大 |

**这个结局不做价值判断**——每个选择都有其合理性和代价。如同博德之门一样·让玩家自己决定什么是对的。

## 六、结局系统详解

### 6.1 路线锁定机制

`
游戏流程:
  共通路线（所有人）
    ↓
  分歧点（基于玩家选择·通常在第3~5章）
    ├── 选择倾向白羽 → 进入【主线】
    ├── 选择倾向朱音 → 进入【帝国线】
    ├── 选择倾向艾露娜 → 进入【宗教线】
    ├── 选择倾向真夜 → 进入【虚空线】
    └── 选择倾向小雪 → 进入【古神线】
         ↓
  该路线专属剧情展开
    ↓
  结局分歧（基于路线内的选择）
    ↓
  最终结局
`

### 6.2 各路线结局表

| 路线 | 好结局 | 一般结局 | 特殊结局 |
|------|-------|---------|---------|
| **主线（白羽）** | 与白羽一起回到地球（或选择留在这个世界） | 白羽存活但两人分离 | **真相全揭露**（唯一能解锁的完全真相结局） |
| **帝国线（朱音）** | 携手推翻暴政·建立新政权 | 部分改革成功·但根深蒂固的问题仍在 | 朱音成为新统治者（可能走向新的独裁？） |
| **宗教线（艾露娜）** | 揭露教皇厅真相·宗教改革成功 | 部分真相曝光·但教皇厅势力仍在 | 艾露娜成为新的宗教领袖（循环？） |
| **虚空线（真夜）** | 击败虚空领袖·封印主要裂缝 | 阻止了一次大规模入侵·但威胁仍在 | 真夜觉醒真正力量·成为虚空猎人 |
| **古神线（小雪）** | 与三神沟通·达成理解 | 发现部分真相·但无力改变 | **"弑神"**（见上文） |

### 6.3 未攻略角色的命运

- 未进入某角色路线时·该角色仍有"好"或"一般"结局
- 但**不会有该角色的深入个人剧情**
- 他们的故事停留在"朋友/同学"层面
- **只有攻略才能解锁完整的个人线**
