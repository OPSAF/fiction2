## 一、帝国本质

**东和帝国是一个独裁君主制国家。**

| 维度 | 描述 |
|------|------|
| **政体** | 绝对君主制——皇帝拥有立法、司法、军事的最终决定权 |
| **人权状况** | 极度受限。没有言论自由、集会自由、出版自由 |
| **民主力量** | 被系统性压制。进步人士被监视、逮捕、或"意外死亡" |
| **对外目标** | 称霸世界·巩固统治·消灭一切潜在威胁 |

### 三权分立的实权结构

帝国的权力并非由皇帝一人独占，而是形成了一个**事实上的权力三角**：

- **天宮军阀（军）**——掌控军事力量
- **九条财阀（财）**——掌控经济命脉
- **教皇厅（魔）**——掌控魔法资源与精英教育

三者互相牵制又互相利用。皇帝作为最高统治者拥有**最终否决权和军队指挥权**，但日常政务中必须平衡三方势力。这种结构使得帝国既非纯粹的君主专制，也非真正的分权制衡——而是一种**脆弱的寡头平衡**。

## 二、帝国内部五大派系

### 1. 天宮军阀（军事派·鹰派）

- **核心**：天宮家本家 + 军方高层
- **主张**：扩军·对抗虚空种·支持三国轴心·"先发制人"战略·对亚顿联盟持强硬态度
- **魔法特色**：矢量学派·擅长大规模破坏性魔法
- **代表人物**：帝国大将军（朱音表伯）
- **民意基础**：虚空种威胁 + 亚顿联盟崛起带来恐慌 → 军方借机扩大影响力

### 2. 九条财阀（经济派·鸽派）

- **核心**：九条家 + 帝国银行 + 主要产业资本家
- **主张**：自由贸易·稳定优先·对轴心和对外战争持保留态度（战争影响贸易）
- **魔法特色**：物质/生物学派·擅长资源加工和医疗·商业情报能力极强
- **代表人物**：九条家当主（兼任帝国银行总裁）
- **民意基础**：资产阶级和城市中产阶级

### 3. 教皇厅（魔法/宗教派·技术霸权）

- **核心**：教皇（大魔法师）+ 魔法研究学会 + 星導学園高层
- **主张**：维护魔法垄断·控制技术研究方向·压制过度民主化
- **魔法特色**：全能型·教皇本人据说精通全部七大学派
- **代表人物**：现任教皇（大魔法师）——具体身份神秘
- **权力基础**：对魔法资源的绝对控制权 + 对精英教育的垄断

### 4. 皇帝派（皇权派·统筹者）

- **核心**：皇室 + 忠诚的皇家魔法师团
- **主张**：维护皇权至上·平衡三大势力·确保帝国稳定
- **魔法特色**：皇室血统魔法·传承自古代的秘术
- **代表人物**：现任皇帝（拥有实权的最高统治者）
- **民意基础**：传统主义者·农村保守势力

### 5. 民主派（改革派·弱势）

- **核心**：下议院部分议员·知识分子·记者·工会
- **主张**：扩大民主·废除贵族特权·改善劳动者待遇·反对轴心扩张
- **困境**：在魔法血统垄断的现实面前·民主诉求缺乏力量支撑
- **部分激进分子暗中同情亚顿联盟的理念**（这是帝国的最高禁忌）
- **代表人物**：下议院中少数派
- **民意基础**：城市知识分子·部分工人

### 6. 星導学園（学术/研究派·相对中立但受教皇厅影响）

- 学園は帝国の魔法研究の中枢·正式には帝国の機関だが、政治的には相対的中立
- 理事会メンバーは各派閥から派遣 —— 内部にも派閥対立がある
- 最大影響力来自教皇厅 —— 学園長は教皇が任命する
- 零を"観察対象"として入学させたのは理事会内の天宮系メンバーの判断（軍閥が新しい"兵器"候補を探している）

## 三、七大黑暗面（详细展开）

**这个国家远非表面光鲜：**

| 黑暗现象 | 具体表现 |
|---------|---------|
| **① 资本主义剥削** | 九条财阀控制经济命脉·工人超时低薪·童工存在 |
| **② 贵族歧视** | 五大家族（尤其是天宮家）垄断资源和机会·平民上升通道极窄 |
| **③ 兽族/亚人种歧视** | 亚人种（兽族）被法律性和制度性歧视·居住限制·职业限制 |
| **④ 奴隶交易** | 表面上"劳动斡旋"实质是人口买卖·亚人种为主要受害者 |
| **⑤ 贫富差距** | 顶层0.01%掌握80%财富·底层60%人口生活在贫困线以下 |
| **⑥ 言论控制** | 所有出版物需审查·异见声音被消音·秘密警察无处不在 |
| **⑦ 天宮家的角色** | **旧贵族/军国主义代表**·主张扩军·对外强硬·内部镇压·是帝国黑暗面的核心受益者和维护者 |

### ③ 兽族/亚人种歧视 详解

明文化された差別法は少ないが、慣習法や行政規則で差別が制度化されている：

- 兽族の居住区域制限·職業制限·結婚制限（人間との結婚は認められない）
- 精灵·魔族は"高等种族"として差別対象外（ただし人間より下に見る者もいる）
- 離散诸族：ほぼ無保護·法的に"存在しない"扱い

### ④ 奴隶交易 / 劳动斡旋制度 详解

- **表向き**：奴隶制は約100年前の民主化運動時に"廃止"
- **実際**："労働斡旋"という名目で亚人种（主に兽族）の半强制労働が横行
  - 戦争捕虜·債務不履行者·"保護された"辺境出身者 —— 名目上は"契約労働"
  - 実際は人身売買に近い —— 特に工場地帯で深刻
  - 法律のグレーゾーン：契約書があれば"合法" · しかし契約は多くの場合強制的
- **サラのケース**：魔晶精炼工場で"契約労働" —— 実質的な奴隶·高額の"違約金"設定で逃げられない
- **社会的態度**：多くの帝国市民はこの制度を"仕方ない"と黙認·一部の改革派が廃止を訴えている

## 四、觉醒者的军事用途

### 明面目的 vs 实际目的

| 层面 | 内容 |
|------|------|
| **明面目的** | 对抗虚空种 —— 这是对外宣传的正当理由 |
| **实际目的** | 制造超级士兵·巩固军事优势 |

### 关键事实

- 觉醒者是**不可复制的战略资源** —— 各国都在争夺
- 星導学園本质上是帝国的**觉醒者兵工厂**
- 天宮家通过军方渠道控制大部分觉醒者的去向
- 学園は魔法戦士の育成機関としての側面も持つ
- 七大魔法学派の教育·研究拠点
- 卒業生の多くは帝国軍に入隊
- 特別試験は軍事的適性を測る目的もある（学生には知らされていない）
- "星導の間"には古代兵器に関する情報があると噂される

## 五、法律与制度

### 劳动斡旋制度（实质奴隶制）

详见上文「③④」章节。核心要点：
- 法律形式上为"契约劳动"，实质为强制劳动/人口买卖
- 主要受害者：亚人种（兽族）、战争俘虏、债务不履行者
- 工厂地带最为严重
- 高额"违约金"设置使受害者无法逃离

### 亚人种歧视法

详见上文「③」章节。核心要点：
- 居住区域限制、职业限制、婚姻限制（人与兽族结婚不被承认）
- 离散诸族几乎无法律保护，在法律上被视为"不存在"

### 征兵制

- 18歳以上の男子に2年間の兵役義務
- 学生は猶予·貴族は免除可能·高魔力资质者は特例

| 階級 | 人口比 | 特徴 |
|------|--------|------|
| 皇室·教皇厅 | <0.01% | 最高統治者·魔法力の頂点 |
| 两強貴族（天宮·九条） | ~0.5% | 軍事·経済の実権·魔法血統維持 |

| 労働者階級（工場·鉱山） | ~35% | 都市部·E-F級·労働条件は改良されつつある |

> 主角零作为F级（地球人·无共鸣能力）处于社会底层·但因其特殊身份受到关注。

- 養父の月収：約2,500リル（下級公務員）

### 朱音的成长环境与压力

- **童年**：小镇唯一汽车接送·孤独·努力型天才·**"天宮"之名不容失败**
- **高中后**：因姓氏受特别对待 —— 本人不喜欢
- **家族期待**：作为天宮分家子女，承载着家族的荣耀与责任
- **内心压力**：既要维持"天宮不应失败"的完美形象，又要面对自己内心的正义感

### 天宮家在帝国权力结构中的位置

- 帝国常備軍約30万人·天宮家が実質的に指揮
- 魔法騎士団各地に配置 —— 星ノ宮にも駐在（朱音の父が団長）
- 対虚空種専門部隊"虚討隊" —— 最精鋭·帝国大将军直属
- 与九条财阀构成帝国"双子星"：表面合作·暗中竞争
- **旧贵族/军国主义代表**：主张扩军·对外强硬·内部镇压
- 是帝国黑暗面的**核心受益者和维护者**

## 八、帝国线的核心冲突

### 朱音的内心冲突

- **朱音出身天宮分家** —— 她既是既得利益者，又能看到系统的黑暗面
- **家族忠诚 vs 正义感**
- **帝国公民 vs 世界公民**
- 如果走朱音线：主角将深度卷入帝国的政治斗争·可能面临**革命 or 改革**的选择

### 其他潜在冲突点

- 长期使用虚空种能量的人会逐渐"虚空化" —— 这是帝国軍方の最高機密
- 部分激进军官默许甚至鼓励士兵使用虚空种能量 —— 以换取短期战力提升
- 民主派部分激进分子暗中同情亚顿联盟的理念 —— 这是帝国的最高禁忌
