## 1. 总论：两套语言，一个世界

### 1.1 核心前提

这个世界的"魔法"，本质上是一套**尚未被完整理解的物理现象**。

- 对这个世界的人而言，它是"魔力"、"魔素"、"神秘力量"
- 对拥有地球知识的主角（零）而言，它是一套**可以被归类、预测、甚至逆向工程的粒子物理体系**

**关键区别**：零的终端数据库里**没有**关于这套体系的任何信息。终端只知道地球的标准模型物理学。这意味着：

1. 零必须通过观察、实验、阅读这个世界的书籍来"重新发现"这些规律
2. 终端可以将地球物理学的框架**映射**到这个世界的魔法现象上——但这是事后归纳，不是预先编程
3. 这种"映射"过程就是零的核心优势：用更先进的分析框架去理解一个落后时代的现象

> 终端与魔法的详细关系 → [`05_TERMINAL_SYSTEM.md`](05_TERMINAL_SYSTEM.md)

### 1.2 双语言描述系统

## 2. 零粒子類：粒子分类体系

### 2.1 定义

**零粒子類（Zero-Particle Category）** 不是一种粒子，而是一个**粒子家族/分类**，类似于现实世界的"强子（hadron）"或"轻子（lepton）"这样的分类概念——每个分类下包含多种不同的基本粒子。

这个世界的常规物质（木头、铁、石头、水、空气等）由**标准粒子**构成——与现实世界完全相同。但除此之外，还存在一类**额外的粒子**，它们同样遵循物理定律（或其推广形式），却表现出"超自然"的性质。这类粒子统称为**零粒子類**。

### 2.2 子粒子结构（类比标准模型）

零粒子類内部存在层次化的子粒子结构，可以粗略类比为：

```
现实标准模型                    零粒子類（推广）
─────────────────────────     ─────────────────────────
夸克（6味）        ──类比──→    「核素粒子」（Core Particles）— 3~5种已确认
  ↓ 组成                          ↓ 组成
强子（质子/中子等）  ──类比──→    「聚合粒子」（Aggregate Particles）
  ↓ 与                            ↓ 与
电子（轻子）        ──类比──→    「载能粒子」（Carrier Particles）
  ↓ 构成                          ↓ 构成
原子               ──类比──→    「魔原子」（Magic Atom）
  ↓ 组成                          ↓ 组成
分子               ──类比──→    「魔分子」（Magic Molecule）
  → 宏观物质                      → 魔法物质
```

#### 2.2.1 三大子粒子类别

**①「核素粒子」（Core Particles, CP）— 类比夸克**

- 世界名称：「素核」「魔力种子」「根源粒子」
- 性质：携带"魔法荷"的基本单元，无法单独稳定存在（类似夸克禁闭）
- 已确认种类：**3种**（世界分别称为「炎之素核」「水之素核」「风之素核」）
- 推测未确认：2~3种（世界有模糊记载但未被实验证实）
- 关键特性：
  - 不同组合方式决定后续所有性质的"底色"
  - 类比夸克的"味"——不同味的组合产生不同"色"（这里的"色"是比喻，指宏观可观测的效应类型）
  - **禁闭性**：从未被单独观测到，总是以聚合形态出现

**②「聚合粒子」（Aggregate Particles, AP）— 类比强子（质子/中子）**

- 世界名称：「魔核」「魔力核」「结晶核」
- 性质：由2~6个核素粒子聚合而成的复合粒子，具有稳定的"魔法荷数"
- 类比对应：
  - 类质子型AP — 带"正魔法荷"，主动向外释放/输出效应
  - 类中子型AP — 不带净魔法荷，但参与结构稳定化
- 存在形式：天然存在于特定矿物中，或通过高能过程人工合成
- **稳定性谱系**：某些组合极不稳定（类似放射性同位素），这成为爆炸性魔法道具的基础

**③「载能粒子」（Carrier Particles, CaP）— 类比电子/光子**

- 世界名称：「流素」「魔力流」「以太粒子」
- 性质：在聚合粒子周围形成"壳层"结构或自由传播的媒介粒子
- 双重角色：
  - **壳层模式**：围绕AP形成轨道状分布，决定魔原子的化学性质（类似电子决定原子化学性质）
  - **传播模式**：脱离束缚后以波粒二象性方式传播能量和信息（类似光子）
- **这是"隐形魔法"的主要载体**（详见第5节）

### 2.3 魔原子与魔分子

**「魔原子」（Magic Atom）** = 1个聚合粒子(AP) + N个载能粒子(CaP)壳层

- 不同AP的"魔法荷数" + 不同CaP壳层构型 → 不同的"魔元素"
- 魔元素的"化学性质"由CaP外壳层的排布决定（类比电子排布决定化学性质）
- 魔原子之间可以通过共享/转移CaP形成**「魔分子」**

**「魔分子」（Magic Molecule）** = 多个魔原子通过CaP键合形成的结构

- 魔分子的性质不是组成魔原子的简单加和——会出现**涌现性质**
- 这就是为什么"合成"魔法物质如此重要且危险：
  - 单个魔元素可能无害
  - 但组合成的魔分子可能具有剧烈的放热/吸热/相变性质
  - **类比**：氮气和氢气都相对安全，但合成氨的过程需要高温高压；硝酸甘油的各种原料单独存在时并不爆炸

### 2.4 粒子相互作用的四大力（推广）

现实世界的四种基本力在这个世界**全部存在且不变**（引力、电磁力、强核力、弱核力）。零粒子類的粒子**额外**参与一种（或一组）**第五种相互作用**：

| 力的类型 | 现实对应 | 零粒子類中的表现 | 世界称呼 |
|---------|---------|-----------------|---------|
| 强核力（推广） | 夸克禁闭 | 核素粒子禁闭 — 无法单独分离 | （未命名，未知） |
| 电磁力（推广） | 电磁相互作用 | AP-CaP电磁类吸引力 + **额外魔法荷作用** | 魔力吸引/排斥 |
| 弱力（推广） | β衰变等 | 某些AP的同位素衰变 → 释放巨大能量 | 魔衰变 / 魔辐射 |
| **第五力** | **无直接对应** | **零粒子類特有的长程相互作用** | **魔力共鸣 / 魔法链接** |

**第五力的特点**：
- 作用距离比强核力远得多，比电磁力略短或相当
- 强度随"魔法荷"乘积变化，但也受环境因素调制（温度、压力、周围标准物质的存在）
- 可以被特定几何形状的材料"引导"或"聚焦"（这就是魔杖/法器的工作原理之一）
- **这种力在地球物理学中不存在**——是这个宇宙特有的附加相互作用

## 3. 魔法物质与元素周期表

### 3.1 魔元素周期表的概念

正如现实世界有元素周期表（H, He, Li...），基于魔原子结构也可以排列出一个**「魔元素周期表」**。

**当前世界已知魔元素**：约 **18~22种**（不同地区记载略有出入）

**周期表的排列依据**：
- 横行（周期）：CaP壳层数（主量子数类比）
- 纵列（族）：最外层CaP数目 + AP类型组合
- 同族元素具有相似的"魔法化学反应性"

### 3.2 魔法物质的三大类别

#### 类别一：天然矿石（原始魔法物质）

从矿脉中直接开采出的含有聚合粒子(AP)的矿物。

| 世界名称 | 科学解释 | 特征 |
|---------|---------|------|
| 魔矿（一般） | 含低浓度AP的硅酸盐/氧化物混合物 | 最常见，用于低端道具 |
| 魔晶矿 | 高纯度AP晶体结构 | 稀有，高端道具核心材料 |
| 魔铁矿 | AP与铁元素晶格共生 | 兼具金属强度+微弱魔法响应 |

- 这些矿石需要经过**提炼**才能释放出有用的魔法性质
- 提炼过程类似冶金：破碎→选矿→熔炼/溶解→提纯→成型

#### 类别二：合成魔法物质（人造魔法材料）

通过化学反应/物理过程将天然魔元素**重新组合**为新的魔分子。这是魔法道具工业的核心。

**著名的合成案例**：

| 合成物质 | 原料（魔元素） | 效果 | 危险性 | 类比现实 |
|---------|--------------|------|--------|---------|
| **烈爆粉** | 炎系AP + 特定CaP结构 | 受冲击后剧烈放热爆炸 | ★★★★☆ | 硝酸甘油/TNT |
| **凝霜剂** | 水系AP + 冰系CaP构型 | 快速吸热→局部冻结 | ★★☆☆☆ | 干冰/液氮（增强版） |
| **雷击石** | 风系AP + 电离CaP | 储存电能→定向释放 | ★★★☆☆ | 电容器/电池 |
| **浮空膏** | 风系AP + 反重力CaP构型 | 涂覆后物体减重50%~80% | ★☆☆☆☆ | 氦气球（固体版） |
| **愈疗液** | 混合AP + 生物活性CaP | 加速细胞修复 | ★★☆☆☆ | 生长因子+抗生素 |
| **隐烟丸** | 特殊AP + 光吸收CaP | 燃烧后产生浓密遮蔽烟雾 | ★★☆☆☆ | 烟幕弹 |

**合成原则**：
1. **不是凭空创造**——必须从已有的魔元素出发
2. **配比精确**——类似化学计量比，错误配比可能导致失败或意外
3. **条件敏感**——温度、压力、搅拌速度、甚至光照都可能影响产物
4. **意外发现**——很多合成魔法物质是"偶然"发现的（正如现实中很多药物和材料的发现）

#### 类别三：生物萃取（来自生物体的魔法物质）

某些生物体内会富集或代谢产生特殊的魔分子。

| 来源 | 萃取物 | 用途 |
|------|--------|------|
| 龙血（如有龙类生物） | 高浓度活性AP溶液 | 高级愈疗药剂基底 |
| 曼德拉草根 | 神经活性CaP化合物 | 精神类药剂/麻醉剂 |
| 月光藻 | 光敏性CaP聚集体 | 照明/储存光能 |
| 精灵之泪（隐喻） | 极高纯度单一CaP | 珍稀催化剂 |

### 3.3 魔法物质的存储与稳定性

| 稳定性等级 | 描述 | 存储要求 | 示例 |
|-----------|------|---------|------|
| S级（极稳定） | 几乎不衰减，半衰期>100年 | 常温常压即可 | 魔晶矿原石 |
| A级（稳定） | 缓慢衰减，需密封 | 干燥阴凉处 | 大部分合成药剂 |
| B级（较稳定） | 明显衰减，需定期更换 | 密封+定期充能 | 魔杖核心 |
| C级（亚稳） | 外界刺激可能触发 | 特殊容器+缓冲材料 | 烈爆粉 |
| D级（不稳定） | 自发衰变/反应 | 严格控温+隔离 | 实验室新合成物 |

## 4. 魔法道具：工具范式

### 4.1 核心理念

**人类本身不会魔法。**

人类使用魔法的方式，和人类使用火药、电力、核能的方式一样——通过**工具**来驾驭那些自身无法直接操控的力量。

- 人类不会自发放电 → 发明了电池和发电机
- 人类不会自爆 → 发明了炸药和炸弹
- 人类不会飞 → 发明了飞机

在这个世界里，这些功能的实现**不需要巨型机械**——因为魔法物质的性质使得"小型化"成为可能：

| 地球技术方案 | 这个世界的魔法道具等效 | 差异 |
|------------|-------------------|------|
| 枪械+子弹 | **魔杖**（内置烈爆粉/雷击石，瞄准触发） | 无弹道计算，直射+第五力引导 |
| 炸弹/导弹 | **爆符/爆瓶**（投掷或安置后触发） | 更小威力，但便携性极高 |
| 飞机 | **浮空药水/羽翼药剂**（口服或涂抹） | 个人飞行，非载具 |
| 空调/制冷机 | **凝霜阵/冰晶石**（放置即降温） | 无需能源输入，一次性消耗品 |
| 电灯 | **光萤石/恒光晶**（自发光） | 无需电路，亮度固定 |
| 无线电通讯 | **传音玉/共鸣铃**（成对使用，距离限制） | 无电磁频谱概念，纯第五力共振 |
| 医疗设备 | **愈疗药剂/恢复泉**（外用或口服） | 直接作用于生物组织 |

### 4.2 魔杖：枪械的魔法等价物

**魔杖是这个世界最典型、最重要的个人魔法道具。**

```
┌─────────────────────────────┐
│         魔杖结构             │
├─────────────────────────────┤
│  杖身（绝缘/导向材料）       │ ← 木质/金属/骨质，引导第五力
│  ├─ 核心（魔法物质容器）      │ ← 内置合成粉末/小晶体
│  ├─ 触发机关（点火装置）      │ ← 机械开关/嵌入符文
│  └─ 瞄准辅助（聚焦结构）      │ ← 杖头几何形状决定效果范围
└─────────────────────────────┘
```

**工作原理**：
1. 使用者按压触发机关
2. 核心内的魔法物质被激活（机械/摩擦/微小魔力脉冲）
3. 第五力沿杖身被引导至杖头
4. 杖头的聚焦几何形状将效应"发射"出去
5. 效果沿直线（或扇形）传播至目标

**与枪械的关键差异**：
- 无后坐力（第五力不涉及动量守恒意义上的反冲）
- 无需换弹（核心可重复充能或内置足够多次用量）
- "弹药"多样性——同一根魔杖换装不同核心即可改变效果
- **精度依赖使用者技巧**——不是简单的"扣扳机就打中"，需要对第五力引导的手感训练

**技能层级**（魔杖使用熟练度）：

| 层级 | 描述 | 占使用者比例 |
|------|------|------------|
| 新手 | 能触发，命中率低，经常误射 | ~40% |
| 熟练 | 稳定命中，能切换核心，基本战术运用 | ~35% |
| 精通 | 快速连射，偏转射击，同时操控多根魔杖 | ~15% |
| 大师 | 无需实体魔杖（用任意物品代替），第五力直觉级掌控 | ~5% |

### 4.3 其他常见道具类型

| 道具类型 | 功能 | 典型场景 |
|---------|------|---------|
| **药水/药剂** | 服用后产生临时生理变化（飞行/力量/隐身） | 冒险/探索/战斗辅助 |
| **符咒/卷轴** | 一次性触发效果，便宜便携 | 应急/陷阱/仪式 |
| **阵法/法阵** | 地面/物品上刻画的几何图案，持续产生效应 | 防御/结界/大型设施 |
| **饰品/护身符** | 被动效果（防护/幸运/感知增强） | 日常佩戴 |
| **傀儡/构造体** | 注入魔法物质驱动的自动机械 | 劳动/战斗/侦察 |

## 5. 隐形魔法场

### 5.1 核心概念：看得见的是冰山一角

现实世界中，**光**是人类赖以生存却无法直接"看见"的东西——你看到的是物体反射的光，而不是光本身（除非你直视强光源）。光子是真实存在的粒子，它携带着能量和信息，但它不是传统意义上的"物质"。

同样的逻辑适用于这个世界的魔法：

**大部分魔法效应并不是由可见的"魔法物质"产生的，而是由看不见的「场」和「粒子流」介导的。**

### 5.2 载能粒子(CaP)的双重存在形式

回顾第2节：载能粒子(CaP)有两种存在模式：

**模式A：束缚态（可见/可检测）**
- 作为魔原子外壳层的一部分
- 存在于魔法物质（魔晶、药剂、魔杖核心）中
- 可以通过重量、光泽、触感等方式间接感知
- 这是普通人理解的"魔法物质"

**模式B：自由态（不可见但可检测痕迹）**
- 脱离魔原子束缚后的自由CaP
- 以波的形式传播（类似电磁波/光子）
- **肉眼不可见**，但会产生可观测的次生效应：
  - 温度异常（吸热/放热）
  - 静电干扰（附近正常电器异常——如果有的话）
  - 生物本能反应（动物躁动、皮肤起鸡皮疙瘩）
  - 特定材料发光（某些矿物暴露后会发出微弱荧光）
  - 气味变化（空气电离产生的臭氧类气味）

### 5.3 感知与检测

#### 人类的感知极限

普通人类**无法直接感知**自由CaP场。但对以下情况会有**模糊的不适感或异样感**：

| 感觉 | 可能对应的CaP活动 | 类比现实 |
|------|------------------|---------|
| "背后发凉" | 高浓度CaP流经体表 | 静电感应 |
| "空气粘稠" | CaP密度异常升高 | 高湿度感觉 |
| "莫名不安" | 低频CaP振荡影响神经系统 | 次声波效应 |
| "皮肤刺痛" | CaP电离周围空气 | 静电放电前兆 |
| "闻到铁锈味" | CaP导致空气电离 | 臭氧产生 |

> **就像氧气**：人类感受不到氧气本身的存在，但能清晰地感受到**缺氧**的痛苦。同理，普通人感受不到CaP场的存在，但在CaP场**突然消失或异常增强**时，会感觉到"不对劲"。

#### 专业检测手段

| 方法 | 原理 | 使用者 | 精度 |
|------|------|--------|------|
| **灵兽/魔兽直觉** | 动物对CaP场变化的先天敏感性 | 驯兽师/猎人 | 中（定性） |
| **感应水晶** | 特定CaP频率下产生光学变化 | 魔道具师/商人 | 中高（半定量） |
| **阵法探测仪** | 刻画了特定几何的检测板，CaP流过时产生可读图案 | 学者/军队 | 高（定量） |
| **生物检测法** | 使用特定植物/昆虫的反应来判断 | 炼金术士/草药师 | 低中（定性） |
| **高手直觉** | 经验丰富的魔法使用者体内的受体细胞对CaP的直接响应 | D级以上能力者 | **极高（但主观）** |

#### 主角（零）和白羽的特殊性

**他们完全无法感知CaP场。**

- 对零而言，即使站在一个正在被强力魔法作用的区域中心，他也**什么都感觉不到**
- 就像一个先天性痛觉缺失的人——别人喊"好烫"，他摸上去毫无感觉
- 白羽稍微好一点——终端可以通过传感器数据**间接推断**CaP活动的存在（温度、电磁异常等），但这不是"感知"，而是"仪器读数"

**这对他们意味着**：
- 无法预警即将到来的魔法攻击（除非看到视觉上的前兆）
- 无法判断某个区域是否"有魔法陷阱"
- 在团队中处于"信息盲区"——需要队友提供魔法层面的情报
- 但反过来，他们也**不受**某些CaP场的负面影响（精神干扰类魔法对他们无效？——待定，取决于具体设计）

## 6. 生物魔法：人体内的炼金术

### 6.1 核心机制：基因→细胞→酶→魔法代谢

**这不是玄幻的"修炼"，而是生化过程。**

一部分人类（大约30%~40%）的基因组中包含一段**特殊序列**（世界未知其存在），这段序列的表达导致：

```
特殊基因序列表达
    ↓
生成「魔导细胞」（Magic-Conductive Cells, MCCs）
    ↓
MCCs表面表达「魔素受体」（Magic-Receptor Proteins, MRPs）
    ↓
MRPs结合环境中的游离CaP或摄入的魔法物质
    ↓
细胞内酶系统将结合的魔法物质"代谢转化"
    ↓
释放出可控的能量/物质效应 → 表现为「个人魔法能力」
```

#### 世界语言的等价描述

| 科学步骤 | 世界语言描述 | 常见说法 |
|---------|------------|---------|
| 特殊基因表达 | "天赋""血脉""血统" | "他家里出了好几个魔法师" |
| MCCs数量多 | "魔感强""魔力充沛" | "这孩子很有天赋" |
| MRP亲和力高 | "适性好""亲和性强" | "他和水系魔法很合" |
| 酶效率高 | "控制力精妙""操纵自如" | "她的火焰魔法像艺术品" |
| 代谢产物排放 | "魔力耗尽""虚脱" | "用完魔法后要休息" |

### 6.2 个人魔法能力的类型学

根据个体MCCs的特异性差异（不同人的受体和酶略有不同），个人魔法能力分为若干**类型**：

| 类型代号 | 世界名称 | 典型效应 | 类比现实生化过程 |
|---------|---------|---------|---------------|
| α型（阿尔法） | 「燃者」 | 体内产热→体外喷火/升温 | 代谢加速+能量爆发 |
| β型（贝塔） | 「驭水者」 | 体液操控→降水/水流控制 | 渗透压调节+水分再分配 |
| γ型（伽马） | 「唤风者」 | 气流操控→风刃/飞行辅助 | 呼吸气流强化+气压感知 |
| δ型（德尔塔） | 「撼地者」 | 震动传导→岩石操控/防御 | 肌肉振动频率控制 |
| ε型（艾普西隆） | 「愈者」 | 细胞再生加速→治疗 | 生长激素超量分泌+干细胞活化 |
| ζ型（泽塔） | 「感者」 | CaP场超强感知→侦查/预知 | 感官过敏化+神经处理加速 |
| 混合型 | 「双属/多属」 | 同时具备两种以上类型特征 | 多种酶系统并存（罕见） |

**重要约束**：
- 类型主要由基因决定，**后天无法改变根本类型**（但可以训练提升同类型内的精细控制力）
- 混合型极为罕见（<1%的有魔法能力人口）
- 即使同一类型，个体差异也极大（如同样的"燃者"，有人只能点个火柴，有人能喷出熔岩流）

> 觉醒者等级体系与非觉醒者评级 → [`03_RANKING_AWAKENING.md`](03_RANKING_AWAKENING.md)

### 6.3 魔法的"呼吸"：日常代谢类比

**使用个人魔法能力，对这个世界上的人来说，就像呼吸一样自然——但又不一样。**

| 方面 | 呼吸（现实） | 魔法使用（这个世界） |
|------|------------|-------------------|
| 必要性 | 生存必需 | 非必需（没有也能活） |
| 频率 | 持续不间断 | 按需使用 |
| 无意识进行 | 是（autopilot） | 低强度可以，高强度需专注 |
| 耗尽后果 | 窒息→死亡 | 虚脱→昏迷（极少致命） |
| 可训练性 | 有限（肺活量） | 很大（控制精度+耐力） |
| 外人看来 | 正常生理现象 | **像魔法**（如果不懂原理的话） |

**关键叙事张力**：
- 对于有魔法能力的人来说，喷个小火苗就像打个喷嚏一样平常
- 但对于**没有**魔法能力的普通人（包括主角零），这看起来就是不可思议的"魔法"
- 而对于零来说——他知道这本质上是一种**生物化学反应**，只是这个世界的科学水平还没发展到能解释它

### 6.4 魔力耗尽与恢复

| 阶段 | 身体状态（科学视角） | 世界语言描述 |
|------|-------------------|------------|
| 充盈期 | MCCs中魔法物质储备充足，酶活性正常 | "魔力充沛""状态正好" |
| 消耗期 | 魔法物质被代谢转化，副产物积累 | "在使用魔法""在输出" |
| 疲劳期 | 储备降低，副产物开始抑制酶活性 | "魔力下降了""有点吃力" |
| 虚脱期 | 储备耗尽，副产物达到阈值，MCCs暂时停工 | "魔力枯竭""透支了" |
| 恢复期 | 通过休息+营养摄入，MCCs重新储备魔法物质 | "在回复魔力""需要休息" |

**恢复方式**：
- 自然恢复：休息+正常饮食（8~24小时完全恢复，取决于透支程度）
- 加速恢复：摄入富含CaP的食物/药剂（2~4小时）
- 强制恢复：外部注入高浓度魔法物质（危险，可能过量）

## 7. 共存原则

### 7.1 核心规则

**零粒子類及其构成的魔法体系，不取代任何现有的标准物理/化学体系。**

这个世界的：
- 木头还是木头（纤维素+木质素，标准碳氢氧分子）
- 铁还是铁（Fe原子，标准过渡金属）
- 水还是水（H₂O，标准氧化物）
- 氧气还是氧气（O₂，标准双原子分子）
- 二氧化碳还是二氧化碳（CO₂，标准化合物）
- 煤炭还是煤炭（碳+杂质，标准化石燃料）
- 石头还是石头（硅酸盐，标准矿物）

**这些东西的物理化学性质与现实世界100%一致。** 魔法不会让木头突然变轻，不会让铁变成金，不会让水自己燃烧。

### 7.2 魔法的定位：额外的可能性

魔法是在**标准物质之上叠加的一层额外的物理现象**：

| 场景 | 标准解决方案 | 魔法增强方案 | 关系 |
|------|------------|------------|------|
| 取暖 | 燃烧木材/煤炭 | 火系魔晶持续放热 | 魔法更方便但更贵 |
| 杀敌 | 刀剑/弓箭 | 魔杖+烈爆粉 | 魔法射程远但弹药有限 |
| 治疗 | 包扎/草药 | 愈疗药剂 | 魔法效果快但资源稀缺 |
| 建筑 | 石头/木材 | 浮空膏减轻重量 | 魔法辅助但不替代结构 |
| 照明 | 油灯/蜡烛 | 光萤石 | 魔法无需燃料但亮度固定 |
| 交通 | 马/马车 | 浮空药剂（个人）/ 魔法驱动载具 | 魔法更灵活但载量小 |

### 7.3 数量约束

**魔法物质并不丰富。**

- 含AP的矿脉稀少且分布不均（类似稀土矿）
- 高纯度魔晶更是凤毛麟角
- 合成魔法物质需要专业设备和知识（不是谁都能在家造炸药）
- 生物萃取来源有限（特定生物/植物本身就稀有）

**结果**：
- 普通人的一生中可能很少接触到真正的魔法物质
- 魔法道具是**奢侈品/专业工具**，不是日用品
- 军队和国家才会大规模装备魔法武器
- 大部分人的生活与没有魔法的世界**差别不大**

### 7.4 人类直接使用魔法的比例

| 人口比例 | 魔法能力水平 | 描述 |
|---------|------------|------|
| ~60% | **无** | 基因中没有特殊序列，完全不能使用个人魔法 |
| ~30% | E~D级 | 有基础能力，但很弱（点个小火、感知附近魔法波动） |
| ~8% | C~B级 | 有实用价值的能力（战斗/生产/医疗用途） |
| ~1.5% | A级 | 强大的能力者（一人可抵一个小队） |
| ~0.5% | **S级及以上** | 传说级（一国之力可能只有个位数） |

## 8. 神粒子類

### 8.1 定义

**神粒子類（Divine-Particle Category）** 是比零粒子類**更高阶**的粒子分类。

如果说零粒子類是这个世界"常规魔法"的载体，那么神粒子類就是**能够直接修改物理常数级别参数**的粒子。

### 8.2 与零粒子類的关系

| 维度 | 零粒子類 | 神粒子類 |
|------|---------|---------|
| 粒子质量 | 较轻（类轻子/介子量级） | 极重（类希格斯玻色子以上量级） |
| 存在形式 | 相对稳定，广泛存在 | 极不稳定，几乎无法天然保存 |
| 产生条件 | 天然矿脉/生物体内合成 | **极端条件**（超高能事件/宇宙射线/???） |
| 作用方式 | 第五力中介的粒子相互作用 | **直接修改量子场参数** |
| 人类可否使用 | 可以（通过道具或生物魔法） | **几乎不可能**（已知唯一使用者：小雪） |
| 世界认知程度 | 有系统的（虽然浅显的）研究 | **传说/神话层面** |

### 8.3 作用原理：量子场参数编辑

普通的零粒子類魔法，是通过粒子之间的**相互作用**来产生效应的（第五力→热/光/力等）。

但神粒子類的工作方式完全不同：

```
常规魔法（零粒子類）：
  粒子A ←第五力→ 粒子B → 产生宏观效应（热/光/力）
  类比：燃烧（化学反应→放热）

神粒子類（量子场编辑）：
  直接修改区域的量子场参数
  → 该区域内所有标准粒子的行为规则被临时改写
  → 产生"违反常识"的效应
  类比：修改游戏引擎的物理参数（重力系数、时间流速等）
```

> 终端系统采用与神粒子類相同的量子场参数编辑原理，详见 [`05_TERMINAL_SYSTEM.md`](05_TERMINAL_SYSTEM.md)

**具体例子**：

| 操作 | 常规魔法能做到的 | 神粒子/终端能做到的 |
|------|----------------|-------------------|
| 放热 | 是（燃者/烈爆粉） | 是（但方式不同——直接提高区域热运动能量） |
| 冷冻 | 是（凝霜剂/驭水者） | 是（直接降低区域分子热运动速率） |
| 引爆 | 是（烈爆粉/魔杖） | 是（直接修改区域结合能阈值→物质自行崩解） |
| 物质转换 | **否** | **理论上可以**（修改夸克味→元素嬗变，但能耗极高） |
| 时间相关效应 | **否** | **可能**（修改时间对称性参数，极度不稳定） |
| 空间相关效应 | **否** | **可能**（修改空间度规参数，极度不稳定） |
| 信息层面操作 | **否** | **是**（终端特有——读取/写入量子态信息） |

### 8.4 为什么神粒子類极其罕见

- 产生需要**宇宙尺度**的能量事件（超新星爆发? 大型对撞机级别的自然现象?）
- 即便产生也**几乎瞬间衰变**回零粒子類或标准粒子
- 能够"稳定"神粒子類状态的容器/方法尚未被发现
- **小雪是已知的唯一例外**——她的身体（或基因？或某种未知机制）能够自发产生并维持神粒子類的稳定状态

### 8.5 小雪的特殊性

- **唯一确认的神粒子類使用者**
- 她的能力不被这个世界现有的任何理论所解释
- 世界把她视为"被神选中者""奇迹之子"
- **实际上**：她的MCCs（第6节所述的魔导细胞）发生了某种罕见的突变/进化，使其能够不仅代谢零粒子類，还能在其细胞内部**短暂维持神粒子類的稳定状态**
- 这种能力的代价：每次使用都会对她造成巨大的生理负担（原因不明——可能是神粒子類的辐射效应？细胞损伤？）
