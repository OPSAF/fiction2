## 写作时的注意事项

### 已确定的"不变真理"
1. **零粒子類 ≠ 单一粒子**：是一个粒子分类家族（`02_MAGIC_PHYSICS.md` §2）
2. **终端 = 量子场参数编辑器**：与神粒子類原理相同但无零粒子信息（`05_TERMINAL_SYSTEM.md` §1）
3. **零 = F级/魔力0/不能感知魔法/不能公开使用终端**（`11_CHARACTERS.md` + `04` §11.4）
4. **白羽 = 同样F级/可以用终端以假乱真/不被当猪**（`04` §11.5）
5. **双轨制**：F~A是非觉醒者 · R~EX是觉醒者 · A级是非觉醒者天花板（`03` §9.2）
6. **觉醒 = MCCs质变**：不是量的积累而是质的飞跃（`03` §9.3.1）
7. **实战主要靠道具**：所以F级的零差距不是天壤之别（`03` §9.3.4）
8. **单攻略制**：只能攻略一人 · 白羽=主线 · 其他=支线（`06_STORYLINES.md`）

### 容易混淆的概念辨析

| 容易混淆 | 区别 |
|---------|------|
| **零粒子 vs 零粒子類** | 后者是v3.0后的正确术语·指整个粒子分类家族 |
| **神粒子 vs 神粒子類** | 同上 |
| **F~A（实力） vs 啓蒙~伝説（学历）** | 前者=战斗力 · 后者=文凭（`03` §9.5） |
| **觉醒者（R~EX） vs 非觉醒者（F~A）** | 质的差异·不是量的差异 |
| **常规魔法（道具） vs 个人魔法（原生）** | 前者靠工具·后者靠体内MCCs |
| **终端效应 vs 魔法** | 原理完全不同（TNT vs C4）·外表相似 |
| **魔道教（宗教） vs 魔法（物理现象）** | 前者是人为组织·后者是自然规律 |
| **虚空种（怪物） vs 古神（造物主?）** | 前者是敌人·后者的立场需玩家自行判断 |
