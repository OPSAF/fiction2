## 1

星期五。放学后。

一周课结束。教室里剩几个人，我把笔记合上，看窗外。中庭再往北，旧校舍后面，一小片林子在斜阳里碎成金色。

那片林子是我的"秘密基地"。

开学没几天就发现了。几乎没人经过，观测者效应降到最低，终端能稳到接近独自在家的水平。这一周每天放学都在那棵老树下坐到天黑——调技能模块，分析早晨捕获的共振信号，或只是开着日志来回翻。不是在做"练习"——更像一个习惯了独处的人终于找到了一片画布。

但今天感觉不太一样。

白羽的座位已经空了。比平时走得早。去哪——我大概猜得到。

过去看看。

——

到林子时太阳刚沉到树梢下。

林间有风，裹着苔藓和湿土的味道。地面落叶踩下去细碎作响。我走到自己常待的那棵老树——然后停住。

**有人。**

树干另一侧。离我固定位置几步远。她坐在隆起的树根上，背微弯，手平放膝上。深蓝长发从肩滑落，被夕光勾了一圈融金轮廓。眼睛闭着——不是睡着。是专注。

然后看到了——右手腕上浮着一团**蓝绿色光**。

终端。她的终端。

我屏住呼吸。出声？退回去？

——

**咔嚓。**

脚下的枯枝。我自己的脚。

她睁开眼。同一瞬间——手腕上的光消失。肌肉记忆级别的反应。手指在虚空一抹——终端藏起来了，像做过无数次。

四目相对。

白羽：「…………」

零：「…………」

风穿过树梢。沉默被拉长。我脑子转了几圈，没一个念头落到嘴上。

白羽先开口了。

「……神代くんも。ここ、使ってるの。」（神代同学也……用这个地方？）

声音很轻。没颤。

「ああ。……人目がなくて、静かだから。」（嗯。没人看。安静。）

「……うん。私も。」（我也是。）

又是沉默。但不硬了。因为彼此都明白对方为什么选了这里。不用解释。

「……さっきの。」（刚才那个。）我说。

「……うん。」（嗯。）

「あれ——いつから。」（那个——什么时候开始的。）

她指尖在膝上绞了绞。

「……わからない。気づいたら——持ってた。子供の頃から。多分——ずっと。」（不知道。注意的时候——已经有了。从小。大概——一直都有。）

「……同じだ。」（一样。）

她抬头看我。

**我的手腕上——蓝绿色光正浮在那。**

我是故意的。呼出终端，让它亮出来。十年养成的习惯是藏。现在——亮出来。

白羽轻轻吸了口气。嘴唇动了一下，没发出声。

「……君は。何番だ。」（你——几号。）

「……六番。Observer #6。」（六号。）

「俺は——七番だ。Observer #7。」（七号。）

她眨了眨眼。然后——嘴角浮起了一丁点弧度。我在她脸上看到的第一个类似笑的东西。

「……後輩。」（后辈。）

「…………へ？」（啥。）

「私の方が——先に入ったから。」（因为我先进来的。）

……逻辑没毛病。就像比我早入学一年的学姐。

「……観測者の先輩後輩って、なんか違う気がするけど。」（观测者分先辈后辈总觉得哪里不对。）

白羽歪头。

「……じゃあ。同級生、で。」（那——同级生。）

「同じクラスだしな。」（本来就是同班。）

「……うん。」（嗯。）

然后——很自然的，两个人都笑了。

——

树下坐了快一小时。她告诉我她的终端是**PowerShell型**。

「ちょっと——見せてみる？」（要不要我打开给你看？）

她犹豫了一下，伸出手。

界面展开的瞬间我就看出了差别。

```terminal-win
Windows PowerShell
Copyright (C) Observer System v1.0.6
PS C:\Users\Observer_6> Get-QuantumFieldStatus

Quantum Field Status Report
----------------------------
Anchor Stability : 98.7%
Observer Count   : 1 (self)
Token Pool       : 100/100
Background Jobs  : 4 running
Anomaly Filter   : ACTIVE

PS C:\Users\Observer_6> Get-ObserverProximity

Nearest Observer Signature:
  ID        : #7 (Bash-type)
  Distance  : 1.2m
  Resonance : 87.3% match
  Status    : CONSCIOUS / TERMINAL ACTIVE
```

完全不同的界面风格。我的——黑底绿字，光标闪烁。她的——深蓝背景，白字输出，每个模块都是 `Verb-Noun` 格式的 cmdlet。标题栏是亮蓝色的，跟我的纯黑窗口放一起像两台不同操作系统的电脑。

难怪她观测者效应比我弱。PowerShell 的结构化特性让意识锚点更稳。同一个教室三十个人——我连终端都呼不出，她还能维持八九成稳定度。

「……すごいな。」（挺厉害。）

「……そうかな。」（是吗。）她收起终端，「でも——神代くんの方が、低レイヤーの操作は得意そう。」（但感觉神代同学的更擅长底层操作。）

确实。Bash 型直接面对量子场裸接口，灵活性高；PowerShell 型在工程化和稳定性上有优势。就像——Linux 和 Windows。

「……似てるけど、違うんだね。」（很像，但不一样。）

「ああ。」（嗯。）

她安静下来，靠着树干看慢慢变暗的天。

「……私。『帰る場所』がどこかにある気がしてる。でも——ここが『帰る場所』じゃないってことしか、わからない。」（我总觉得……有一个"该回去的地方"。但我只知道——这里不是。）

沉默了一会儿。

「……俺もだ。何も覚えてない。ただ——」（一样。什么都不记得。只是——）

「……ただ？」（只是？）

「時々、夢を見る。知らない場所。知らない街。——そこで、俺を呼ぶ声がする。名前は——思い出せないけど。」（有时做梦。不认识的街道。有人在梦里叫我。名字——想不起来。）

白羽沉默了几秒。然后轻轻说：

「……私も。」（我也是。）

我看着她。她也看着我。

「同じ夢。多分——同じ場所。」（一样的梦。大概——同一个地方。）

谁都没有再开口。风把树叶沙沙声送到耳边，头顶天空从金色变深橘变灰蓝。一种说不出口的东西沉在空气里——不是沉重，更像一块终于落了地的石头。

**终于。有人共享这个秘密了。**

## 2

天全黑了。林子里只剩远处路灯被树缝切成碎屑。

「戻らないと。門限がある。」（得回去了。门限。）我站起来。

「……うん。」（嗯。）白羽起身拍裙子上的叶子。

「——神代くん。」（神代同学。）

「ん。」（嗯。）

「このことは——」她犹豫，「……二人だけの秘密。に、しておく？」（这件事——当作我们两个人的秘密，好吗？）

我想了想。点头。

「ああ。まだ——誰にも話さないほうがいい。俺たち以外に誰が関わってるのか——わかるまでは。」（嗯。搞清楚还有谁牵涉在里面之前——先不说。）

白羽轻轻点头。然后——伸出右手小指。

「……約束。」（说定了。）

一本正经的脸。这家伙有点奇怪。但我还是伸出小指勾上了。

「約束だ。」（说定了。）

她的手指比想的凉。不过这话就不说了。

——

穿过旧校舍往回走。脑子里还在转刚才的事。她的终端。PowerShell型。记忆也空白。以及——

**她在等一个跟她一样的人，等了很久。**

路灯昏暗的窄巷子走到头就是主路。但在那——

一个人影倚在墙边。个子不高——不是高等部制服。深蓝水手服。中等部的。双臂交叠，两色异瞳在暗处慢慢锁定我。

<span class="ix person" data-ix="maya">黒鉄真夜</span>。中等部三年。我的学妹。

「……ふうん。」（哼嗯。）

她嘴角一勾。

「あんた——やっぱり面白いね。」（你——果然挺有趣的。）

「……どういう意味だ。」（什么意思。）

她从墙上撑起身体，前踏一步。距离约两米。没更近——但那两米里开始弥散一种说不清的压迫感。

「この学校には——高等部にも中等部にも、いろんな奴がいるけど。あんただけは——何も感じない。」（这学校里高等部中等部都有各种人。但只有你——我什么都感觉不到。）

「……F級だからな。」（我是F级。）

「魔力の話じゃないよ。」摇头，「もっと——深いところ。気配っつーか。——まあいいや。あたしは真夜。よろしく。」（不是魔力。更深的——气息之类的。——算了。我叫真夜。多指教。）

说完人影一晃就融进暗处。像一只从没被人见过全貌的猫。

我站在原地，数了三秒。

**（"什么都感觉不到"——吗。）**

她大概比普通人敏感得多。魔族血统？还是本身特殊？不管哪种——她说"感觉不到任何东西"，是因为我的量子签名不属于这个世界。

而她能感知到这一点。

……这学校有意思的人还挺多。
