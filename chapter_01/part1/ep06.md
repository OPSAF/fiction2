## 1

第二周。

午休。路过中庭时看见一个穿中等部制服的女生坐在喷泉边石台上，手里转着一根树枝。

深紫短发，异色瞳——<span class="ix person" data-ix="maya">黒鉄真夜</span>。

高等部校区对中等部学生来说是禁区。不过她显然不在乎。

「……ここ、中等部の区域じゃないぞ。」（这里不是中等部区域。）

「べつにー。」（无所谓。）头都没抬，「高等部の方が面白いこと起きそうだから。」（高等部这边好像会发生更有趣的事。）

她嘴角弯着，像早就知道我会路过一样。

……这人到底什么情况。

---

第一节课。魔法基础理论。今天主题——"观测者效应与魔力测定的不确定性"。

教科书上讲的"观测者效应"是魔法层的测量误差，跟我终端受的那种不是一回事。但巧合的是——恰好是我最熟的主题之一。

鷹司老师：「——魔法測定における観測者効果とは、測定行為自体が測定対象の魔力場に干渉することで生じる誤差です。最小化する方法はあります。——誰か、わかる人は？」（魔法测定中的观测者效应，指测量行为自身干涉被测对象魔力场产生的误差。存在最小化方法——谁知道？）

教室安静。没人举手。

「——神代くん。」（神代同学。）

空气轻轻一震。F级被点了。我站起来。

「……測定環境の魔力濃度を下げることです。具体的には——測定室の防護壁にカウンターフィールドを多重展開し、外部魔力場との相互作用を遮断します。さらに——」（降低测量环境魔力浓度。具体——在测量室防护壁上多重展开反相位场，阻断外部魔力场相互作用。另外——）

继续。语言自己往外滑，顺畅得我自己都意外。"既视感"——教科书读了一遍，却像在这个领域做了十年。讲到后来差点把终端实验的数据也说出来——还好及时刹住。

「——それから、一番重要なのは**人数**です。測定室内の人数が増えれば増えるほど、各人の魔力場が相互干渉を起こしてノイズが増大します。理想的な測定環境は——一人です。」（此外最关键的因素是人数。测量室内人数越多，各人魔力场相互干涉就越强，噪声越大。理想的测量环境——独自一人。）

最后一句教科书上没有。那是我用终端做了上百次实验**自己发现的。**

鷹司老师微微睁眼。然后——温和点头。

「……素晴らしい。完璧な回答です。特に最後の指摘——"人数"の要素に言及した学生は君が初めてだ。座ってください。」（精彩。完美的回答。尤其是最后一点——提到"人数"因素的学生，你是第一个。请坐。）

教室里空气微变。F级展示的知识深度——把某些人的"常识"轻轻摇了一下。

我坐下。白羽投来一瞥，嘴唇动了一下，没笑出来。

她当然懂。我那句"理想的测量环境是独自一人"——指的是什么。

## 2

放学后。老地方，林子里。

白羽坐树根上，膝上摊着终端。我在旁边坐下，也打开自己的。两人沉默，两团蓝绿色光在渐暗林间轻晃。

「——シンクロテスト、やるか。」（同步测试，来试试。）

白羽点头。

```terminal
> sync.connect(target: "Observer_6", mode: "PAIR_TEST")

[CONNECTING...]
[HANDSHAKE: Observer_6 — ACCEPTED]
[PAIR TEST INITIATED]

Test 1: Independent operation  — BOTH TERMINALS NOMINAL
Test 2: Synchronized command   — RESONANCE BOOST DETECTED (+18% efficiency)
Test 3: Data exchange          — BANDWIDTH: 4.7x expected baseline

[CONCLUSION] Observer pair synergy is REAL and MEASURABLE.
When both terminals operate in proximity (<5m), efficiency
improves by 15-25% for BOTH users.
```

「……すごい。」（好厉害。）白羽轻吸一口气。

两台终端在近距离同步运转——**互相提升性能。** 这发现比我预想的重大。本质上是设计成双人协同运作的。

「これ——どういうことなんだろうね。」（这——到底意味着什么呢。）

「……わからない。でも——多分、俺たちは元々"チーム"だったんだと思う。どこかのタイミングで——離ればなれになっただけで。」（不知道。但可能我们本来就属于一个团队。只是从某个时间点开始——失散了。）

白羽低头沉默一会儿。

「——神代くん。」忽然抬头。

「ん。」

「いつか——帰るのかな。私たち——」（有一天——会回去吗。我们。）

这句话的重量——和我每晚关掉终端之后反复想的那个问题一模一样。

「……わからない。」老实说。「でも——帰る場所があるなら。帰りたいと思うなら。その時は——」（不知道。但如果真有能回去的地方。如果想回去——到那时候——）

顿住了。该说什么？"一起回去"？我有资格说这种话？

「……一緒に考えよう。」（一起想办法吧。）白羽说。

「ああ。」（嗯。）

就这些。但够了。

## 3

夜。回宿舍路上，绕校舍背面。

暗处有人倚墙。今天是她先开口。

「よお、F級。」（哟，F级。）

真夜。第二次今天。

「あんた——最近、楽しそうだな。」（你最近——好像挺开心的。）

「……そう見えるか。」（看得出来？）

「見える。顔つきが変わった。——なんか見つけたんだろ。あんた、この前まではずっと警戒してる目をしてた。今は——ちょっとだけ、ここが"自分の場所"になりつつあるって顔だ。」（看得出来。表情不一样了。——找到了什么吧。之前一直是警觉的眼神。现在——有点像"开始把这里当自己的地方了"。）

她果然——比一般人会看人。暗杀一族血统？还是单纯性格？可能都有一点。

「……お前はどうなんだ。」（你呢。）

「あたし？——あたしはただ、面白いことが起きるのを待ってるだけだよ。」（我？只是等有趣的事发生。）嘴角一勾，「この学校には——きっと、すごいことが起きる。あんたが来てから——特に。」（这学校——一定会发生很厉害的事。尤其是你来了之后。）

说完硬币入暗处一样无声消失。

**"尤其是你来了之后"——吗。**

她到底感知到了多少。

**这不坏。**
