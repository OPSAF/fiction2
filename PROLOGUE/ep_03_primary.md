## 1

下午自由时间。我去图书馆找参考资料。

星導学園的图书馆比小镇的大了不止一个数量级。三百万册藏书，穹顶高到能装下三层楼的阅览区。下午的斜阳从彩色玻璃窗里落下来，把学术期刊区的地板染成暗金色。

我的终端在家里——学校的公共终端需要排队。所以今天只能手动查。

走到物理期刊区。手指在排列整齐的脊背上滑过——《帝国魔導学会誌》《零粒子物理评论》《魔法场理论季刊》——停。抽出一本《零粒子物理评论》最新刊，翻到目录。

然后心跳漏了一拍。

**《关于波粒二象性在CaP场中的观测效应》—— Q.E.D.**

**这是我写的。** 匿名笔名。上学期的研究刊物。我在自己的终端上写这些东西纯粹出于兴趣——"既视感"让那些公式看起来"本来就该是这样"。

正准备把杂志放回去——旁边有人。

银色的长发。翡翠色眼睛。精灵族的交换生——艾露娜·温德尔。她手里拿着**同一本期刊的另外一册**，正翻到同一篇论文。

她察觉到了身后的视线，转过头。

「啊，抱歉，挡住你了。」侧身让路，微笑——标准的温柔治愈系笑容。

我脑子一片空白，随口说了一句本不该说的话：

「那篇论文……你觉得怎么样？」

艾露娜的眼睛亮了一下。

「你也看过？作者的分析方法非常独特！尤其是第三节的参数推导——从CaP场的底层波动方程直接导出观测干涉项，我从来没见过这种思路——简直像是——」

她停住了。

盯着我看了两秒。

「……没什么。很高兴遇到同好。」

微笑着离开。

**我当时不知道的是**：艾露娜已经在怀疑我就是Q.E.D.了。因为我问"你觉得怎么样"时的语气，不像一个普通读者——更像一个作者在等待反馈。

## 2

从图书馆出来，天色已晚。

我走回男生宿舍的路上——感觉有人在看我。

转身。没有人。

再转身。

黑铁真夜靠在路灯柱上，双手抱胸，嘴角带着一丝玩味的笑。深紫短发，异色瞳——左深红，右金黄。极小的角在发间隐约可见。她穿着中等部的深蓝水手服——不是高等部的学生。

「F级。你是那个F级？」

「……是。有事吗？」

她走近一步。上下打量我——视线从我的脸滑到手腕上的旧型号终端，再到完全没有魔法波动的身体。

「有意思。这个学校里，你是第一个让我'感觉不到任何东西'的人。」

「……谢谢？」

真夜轻笑：「别误会。我不是在夸你。」

凑近，压低声音：

「我只是觉得……你这种人，应该活不长吧。」

说完转身走了。脚步声消失在夜色里。

我站在原地。

「……这是什么？威胁？警告？还是——打招呼？」

很久之后我才知道，真夜的"高攻低防"性格模式在那时候就已经完整呈现了——她主动发起攻击（语言上的挑衅），但实际上是在试探我的反应。如果我表现出恐惧或愤怒，她就失去兴趣了。但我的回答是"……谢谢？"——这种完全不在预期内的回应，反而让她记住了我。

## 3

回到宿舍。室友橘还没回来——大概还在食堂。

我用公共通讯终端给家里打电话。

响了两声就接了——声音有点喘，大概刚跑过来。

「もしもし——お兄ちゃん？！」（喂——哥哥？！）

小雪。

「ただいま。」（我回来了。）不，还没回去——但对她这样说比较自然。

「学校怎么样？有没有交到朋友？吃饭了吗？F级的事——没事吧？」

问题连珠炮一样。我靠在通讯终端旁边的墙上，忍不住笑了一下。

「都挺好的。朋友嘛——算是有了吧。F级的事不用担心，我又不是第一天知道。」

「嗯。……哥哥。」

「嗯？」

「……没什么。就是想听听你的声音。」

短暂的沉默。

然后挂断之前——

「哥哥。不管你在那里是什么级——你在我这里是SSR。」

我笑了：「你又偷看我玩的抽卡游戏了。」

「才没有！」

嘟——挂断。

我在通讯终端前站了一会儿。小雪这家伙——她不说"我想你了"或"我好担心你"。她用"SSR"这个只有我们俩懂的梗，说"你对我很重要"。既传达了感情，又留了退路。这大概是她保护自己——也保护我——的方式。

## 4

室友睡了。橘的呼吸声均匀地填满了房间。

我拿出自己的终端——一台从出生起就陪伴着我的、与意识绑定的存在。物理载体是养父从废品市场淘来的旧型号终端机，被我改装过无数次。

开机。熟悉的蓝绿色界面。Bash光标一闪一闪。

```terminal
Terminal v1.0.7
Observer #7 | Status: ACTIVE
Local Time: 星暦318年 4月1日 22:14:08
Token Remaining: 87/100
Terminal Stability: 96.8%

> system.resume background_tasks
[OK] 3 background tasks running.
```

这是他十年来每天都在做的事：调试技能模块、优化参数、测试新想法。今天打算把 `topo.preserve_v3` 的冷却时间再缩短几秒——上次模拟时30秒的维持时间差了一点。

习惯性地打开编程环境，开始写一些东西。手指自己动了起来——他自己也不知道为什么要写。

屏幕上的代码行数增加。某种算法。某种数学模型。

写到一半的时候——零突然停下来。

因为他发现自己**根本看不懂自己在写什么。**

那些符号、那些公式、那些逻辑结构——它们从指尖流淌出来，但大脑跟不上。就像……某个更深层的记忆在替他操作这双手。

屏幕闪烁了一下。

一行金色文字浮现在代码上方，持续了不到0.5秒：

`PHANTASM_PROTOCOL_v4.2 // AUTH_REQUIRED`

然后消失。

零揉了揉眼睛。

「……熬夜太久了。」

但他没有关机。而是打开了系统日志界面——翻到最底部，一直往下拉。日志的最早一条记录的时间戳——

```
[LOG START — Star Calendar 302, Month 3, Day 17, 03:42:01]
```

他的出生时间。

关机。躺下。

天顶在黑暗中模糊成灰。远处走廊有脚步声。

星導学園高等部。入学第一天。结束。

**已知：白羽栞也有终端。**
**未知：除此之外的一切。**
**不安：刚才那段代码——是谁写的。**
